　　第361章
<br/>　　若论内政，山寨没人比得过和珅，但说杀人，和珅不如程大雷，史上也不曾听说过和大人有什么了不得的战绩。
<br/>　　所以这次事情由程大雷亲自挂帅，找到那伙人，干掉那伙人。
<br/>　　将琴川外围的流民全部迁到琴川关附近，各村之间组成联防队，一发现情况，立刻敲锣鸣鼓。另外对新进迁入的流民，来往的商队严加盘查，恨不得连他们祖宗十八代都要问出来。
<br/>　　关鱼手下的士兵，每一百人分为一个小队，分区域，分时间巡逻搜捕，就算挖地三尺也要把这伙来历不明的恶人搜查出来。
<br/>　　这些命令，都是程大雷走出停尸房后下达的，并且是立刻执行，不能有任何耽搁。
<br/>　　程大雷隐隐有种感觉，这伙人既然来了，就不会轻易离开。如果不立刻行动起来，当还会有有下次恶性事件。
<br/>　　而程大雷自己，则和刘悲，樊梨花一起来到事发现场，或者称作第一凶杀现场。
<br/>　　这里是刚修的房子，夯土成墙，上面盖着瓦片稻草，三户人家连在一起，屋子里也没多少摆设。
<br/>　　程大雷刚走进去时，就嗅到一股淡淡的血腥味，就在昨夜，三户人家一十三口惨遭毒手。
<br/>　　其实被杀的都是平民，杀死他们也无须多少战斗力。令程大雷赶到恐怖的是他们手段的残忍，以双手撕开肺腑，捏碎头颅，最后又残忍的虐尸，这种事，不像是正常人能干得出来的。
<br/>　　难道来了一伙变态。
<br/>　　来的路上，程大雷就一直在琢磨会是谁动的手：戎族，凉州词派过来捣乱的，还是从其他地方闯到到此地的恶贼？
<br/>　　毫无线索，所以程大雷才想要来到案发现场，看看有没有什么线索。
<br/>　　凡走过必留下痕迹，此刻的程大雷恨不得变身福尔摩斯·雷，连一根头发丝都不能放过。
<br/>　　可惜他没有这方面的修养，眼睛瞪的再大，也是一无所获。同时，山寨也没有这方面的人才。
<br/>　　程大雷突然心中一动，是呐，山寨没有这方面人才，自己可以召唤出来一个么，如今的点将台已经可以定向召唤，自己召唤个捕快侦探之类的职业出来，例如包拯、狄仁杰之类的人，还不分分钟知道凶手是谁。
<br/>　　想到便付诸行动，可程大雷刚打开系统，便又选择了退出。
<br/>　　原因无他，他现在的恐惧值所剩无几，大概只能召唤一个顶级人物。可战斗职业也有很多种，怎么就能保证一定召唤到捕快呢。
<br/>　　只有一次机会，却也不能冲动。
<br/>　　眼前的情况，直令程大雷束手无策起来。他其实也知道，琴川八百里，想藏下一伙人太容易了。即便自己发动所有兵力搜寻，却也未必能抓住对方。
<br/>　　如果让对方杀了人，再大摇大摆的离开，程大雷无论如何也咽不下这口气。
<br/>　　“程当家在这里么？”
<br/>　　门外突然响起一个陌生的声音，程大雷走出屋子，看到徐神机陪着凉州钟伟虎出现在门外。
<br/>　　“程当家，我可找到你了。”钟伟虎擦了擦额头上的汗珠，显得很是兴奋。
<br/>　　“怎样，是过来自首的，还是专门来嘲笑我的？”程大雷脸色十分不好，本来就怀疑是凉州派的人干的。钟伟虎来得这么及时，说不得就是他。
<br/>　　钟伟虎怔了怔：“程当家这话从何说起？”
<br/>　　“哼，你来做什么？”程大雷。
<br/>　　钟伟虎靠前几步，道：“程当家，凉州发生了一件大事，我奉宋大人之命，来请程当家帮助。”
<br/>　　“要我帮助？”程大雷愣了愣。琴川和凉州城是保持着表面上的和平，但程大雷敢打赌，宋伯康做梦都想弄死自己，难道两家的关系，已经好到可以互相帮忙的份上了？
<br/>　　“什么事？”
<br/>　　“最近凉州来了一伙贼人，手段残忍至极，在凉州城，浩甲城都造了杀业，动辄屠人满门，现在被他们所杀的人，已超过上百人。”钟伟虎叹口气：“这件事怕程当家还不知道。”
<br/>　　“已经知道啦。”程大雷指了指门里：“你来找我做什么？”
<br/>　　“难道他们已经来了琴川！”钟伟虎一惊。
<br/>　　他收拾脸上表情：“整个凉州都在追查这伙人，派出最精锐的捕快，可不仅一无所获，我们派出的捕快，还接连遭到屠杀。整个凉州可以说已经是束手无策，所以宋大人才想到程当家，毕竟程当家是……可能程当家有些经验，会帮助我们抓到这伙恶人。”
<br/>　　意思是我也是恶人，大家恶人了解恶人，容易摸着对方的思路。
<br/>　　程大雷撇撇嘴：“是否想去琴川关里坐回，我们详谈这件事。”
<br/>　　“啊，这个……就不必了，程当家，这是关于这件事的所有资料，大家都等着程当家抓住这伙贼人，为凉州除掉一害。”
<br/>　　说着，钟伟虎丢下资料，马不停蹄的跑远了。或许，琴川关在他心目中已经留下阴影。
<br/>　　除掉一害？程大雷冷笑一声，怕只怕在大家心中，自己也是凉州一害。所以让两方狗咬狗，不管谁除掉谁都是好事。
<br/>　　程大雷抱着这些资料回到琴川关，窝在书房里翻开，这全部是最近的死亡记录，没有兵器，双手撕裂内脏，看手法也知道是一个人干的。
<br/>　　当程大雷阅读这些时，感受到的是一阵阵寒意，对方不为复仇，不会抢夺钱财，甚至不是为了女色……单纯就是为了杀人而杀人，手段之血腥可怖，令人发指。
<br/>　　可真想要通过这些资料，知道对方的来历，判断对方去向，甚至抓住对方，对程大雷来说，几乎是不可能的事情。
<br/>　　毕竟自己不是专业人士，而专业的事最好交给专业的人来做。
<br/>　　难道真的要拼脸召唤一次么，可万一召唤不到想要的人怎么办。
<br/>　　正这个时候，苏樱端着茶进来，道：“你也忙了一天了，先喝杯茶歇会儿。”
<br/>　　“怎么歇啊，一点头绪都没有……”程大雷忽然一怔，下意识站起来，在苏樱面前摊开双臂：“看我看我，你能不能看出点什么？”
<br/>　　“你……”苏樱吓了一跳：“……没洗脸。”
<br/>
