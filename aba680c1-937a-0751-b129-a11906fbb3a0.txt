　　第599章
<br/>　　青亮的刀光在空中挥过，发出铛地一声响，然后刀从石草的喉咙割过，血便咕咚咕咚涌了出来。
<br/>　　在问出想要的情报后，程大雷送石草走了。做这种事，他越来越熟练了，石草被他折磨得不成人形，程大雷身上却一滴血都没有。
<br/>　　这石草是北蛮部的人，这次是带着金问道的命令，前往戎族王城求援。
<br/>　　野原火想要吞掉北蛮部的心已经路人皆知，金问道也不能不做准备。他这次是想要得到呼延力的援助，集齐两家之力，将野原火扼杀在襁褓中。
<br/>　　当然，如此大事，金问道也不可能只派出一人。他一共派出三批人，程大雷劫到的石草只是其中之一。所以，程大雷对这件事并没有造成任何影响。
<br/>　　这个消息对野原火来说，可以说十分重要，但对程大雷来说几乎没有任何用处。不过，从石草口中，程大雷也得到了关于燕不归的消息。
<br/>　　他果然是戎族王的人，隶属于白狼部，也是整个戎族屈指可数的高手。
<br/>　　混了这么多日子，程大雷终于得到些有用的消息。按照时间推算，燕不归此刻应该已回了戎族王城。
<br/>　　程大雷目光眺望向西北方向，冲小白狼道：“我怕是要去一趟戎族王城。”
<br/>　　小白狼一惊，道：“你可要想清楚，王城是戎族的中心，必然高手如云。你我都不了解那里的情况，贸然前往，怕会十分危险。”
<br/>　　“倒也不是完全不了解了，我曾经就去过一趟。”
<br/>　　程大雷当初夜探戎族王城，还顺便将戎族曾经的王呼延跋杀了。无论怎么想，上次也要比这次更加危险。及早把燕不归除掉，程大雷也可去个心结，早些离开这蛮荒之地，自己就可以心无挂碍的回蛤蟆城。
<br/>　　却也不知道，自己离开这么多日子，大本营的情况怎样了。
<br/>　　程大雷主意已定，小白狼也没有再说什么，二人确认戎族王城的方向后，便向目的地行去。
<br/>　　一路无话，到黄昏时分，二人来到了戎族的圣湖。这湖有个名字，唤作女儿湖。据传说，草原上这数百个戎族部落，起源地便是这女儿湖。
<br/>　　程大雷之前曾经来过这里一次，也在草原上闹出过一番风波，不过过去这么多年时间，当初的事怕已没多少人记得。
<br/>　　“我与你一起去？”小白狼问道。
<br/>　　程大雷摇摇头：“你留在这里等我，如果顺利的话，天亮之前我能回来。”
<br/>　　小白狼也知道，凭自己的本事也没办法随着程大雷夜探王城。她能做的，也就只有在女儿湖边等程大雷归来。
<br/>　　程大雷换了夜行衣，将匹夫剑背在身上，步行赶往目的地。之前已经走过一趟，一切倒也是轻车熟路。
<br/>　　所谓的王城并没有太大规模，黄土夯墙，想要找一处城墙豁子比找到片完整的城墙容易。程大雷到的时候，天色已经黑下来，他几乎没费任何力气便进了城。
<br/>　　要去的地方，自然是戎族的王宫。今天这次行动，也不求一定可以杀了燕不归，能碰上燕不归，顺便杀了更好。如果碰不上，也趁机打探一下戎族权力中心的情报。
<br/>　　程大雷背着匹夫剑，与黑暗融为一体，很快他便到达了戎族的王宫。
<br/>　　身子趴在屋顶上，目光看过去一片黑暗。戎族的王宫和帝国不一样，虽然也很大气，但一切都很简陋，到了晚上连盏油灯都恨不得不点。
<br/>　　程大雷看过去，只有寥寥几间屋舍亮着灯，程大雷搜寻脑海中的记忆，他很快想起主殿在何处。曾经的戎族王死了，如今那里躺着新一代的王。
<br/>　　程大雷和呼延力照过面，心里琢磨着，要不干脆自己再把呼延力杀了。不得不说，程大雷很有潜质成为一名不错的刺客。身无彩凤双飞翼可以令他神不知鬼不觉的潜入，而阿飞快剑又是天下独一无二的刺杀剑术。
<br/>　　当然，这个念头只是想一想而已，他这次又不是奔着呼延力来的。不过，想要在此地守株待兔碰上燕不归，几乎也没有什么希望。
<br/>　　程大雷寻了一间亮着灯的屋子，身子轻飘飘的贴了过去，将瓦片掀开一条缝，眯起眼睛往里看。
<br/>　　只见一个老者坐在屋子中，缓缓翻着一本书。程大雷顺便查看了对方的信息，心中是咯噔一下。此人名叫白无双，乃是一位绝世高手。
<br/>　　他立刻屏住呼吸，尽量不发出任何声响。这要是被发现了，在此地动起手来，程大雷可没把握在戎族高手的围攻下脱身。
<br/>　　除了骑术外，戎族也是很擅长射箭的。
<br/>　　那老者看了一会儿书，忽然抬起头，冷冷的声音道：“既然来了，为何不现身？”
<br/>　　程大雷一激灵，当时就想转身逃走，他可不敢在此地与一个戎族高手动手。
<br/>　　就是这一愣神的光景，老者面前出现一人，程大雷眼睛瞬间睁大，正是他千思万想的燕不归。
<br/>　　自己果然猜得不错，跳崖没那么容易死的，燕不归这不就平平安安回了草原，看上去还身强体壮的。
<br/>　　燕不归在白无双面前跪倒，口中道：“师父。”
<br/>　　“回来啦。”白无双缓缓道：“回来便好。”
<br/>　　“徒儿回来了，但小师妹他却永远留在了帝国。”
<br/>　　“这件事我已知道了，我还听说你杀了几个人，替贞儿报了仇。”白无双道。
<br/>　　听到这话，燕不归脸上浮现一抹奇怪的快意，他道：“还有几个人没死。”
<br/>　　白无双点点头，道：“柳褐岩可以交给你，不过，即便杀了他们，贞儿也不可能死而复生。”
<br/>　　“是呐，贞儿再不可能活过来了。”燕不归长叹一口气，随之咬紧牙道：“所以，那些人更不能活着。”
<br/>　　“比如？”
<br/>　　“比如夫子，张六，柳褐岩……他们都该死。”燕不归咬牙切齿：“还有程大雷，总有一日，我要亲手杀了他。”
<br/>　　白无双奇怪的看着他：“那你还在等什么？”
<br/>　　“是。”
<br/>　　燕不归突然拔出剑，一剑冲屋顶扎过来。
<br/>　　程大雷一激灵，嘴里立刻骂出声来。
<br/>　　“我去你大爷的，怎么把我扯了进来。”
<br/>(本章完)
