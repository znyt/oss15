　　在听到从通讯频道中传来的声音时，罗昊脑海中首先蹦出来的一个词就是“国家力量”！
<br/>　　这就是强大的国家力量所带来的威慑力，不需要过多的头口抗议，不需要过多的交涉，只需要一个动作，就能让来犯者认怂后退！
<br/>　　如果当年有这样的军事力量，何至于发生南海上空的撞机事件，何至于让美国方面做出2.3456789万美元侮辱性赔偿？
<br/>　　而当数架苏-30战机护送着A340客机降落在某空军基地的时候，轰炸机编队大摇大摆的进入霓虹国海上空，逛了一圈后，又再次离开。
<br/>　　对于这样的举动，霓虹国媒体无比愤怒，对中方这种做法表示了强烈的谴责。而西方国家媒体则是对此感到无比意外，甚至有军事专家揣测中方这种做法的用意，一时间，网络上各种各样的说法都有。
<br/>　　国内主流媒体也进行了报道，一些著名的军事网站和论坛也转发了霓虹国方面所拍摄到的轰炸机编队照片，国内民众对这种扬眉吐气的做法大呼过瘾。
<br/>　　当然，也有一些大V公知跑出来谴责洗地，只是他们的谴责声比起全民的打呼过瘾声实在太弱小太弱小的，小到刚发出声音，就被淹没了，连个水花都没冒出来。
<br/>　　如所有人意料那般，霓虹国方面也对这种“危险行为”进行了抗议，但是中方外交部却并没有对此立刻做出回应，颇有一种“你抗议是你的事情，回不回应是我的事情，想抗议请继续”的味道。
<br/>　　而外交部的反应令得网上的讨论变得更加激烈起来，各种评论层出不穷，其中还有人贴出了当年大使馆被炸后，大学生和民众高举着横幅，拿着墨水瓶在美利坚驻华大使馆外抗议的照片，顿时引起了那一代人最不堪的一段回忆。
<br/>　　照片下方的评论区中，参与过当年在美国大使馆外抗议的人留言。
<br/>　　“我们国家要是有今日之实力，当年何至于忍气吞声。”
<br/>　　“如果有现在的军事力量，就算是借美国佬十个胆子，他也不敢把炸弹往我们大使馆扔。”
<br/>　　“今天他要再敢吡叨，分分钟给他炸回去！”
<br/>　　“今天也让你们体会一下抗议无效是种什么样的憋屈！”
<br/>　　“抗议？对不起，您的抗议在运输途中已遗失，请重新抗议。”
<br/>　　“抗议无效，休庭！”
<br/>　　“……”
<br/>　　这件事情的发酵和热议程度直到第二周国防部召开例行记者会都没有降温，甚至随着例行记者会的召开，大家对事情更加关注起来，因为所有人都清楚，一定会有人对事情进行发问，而国防部的新闻发言人也必然会对这件事情进行回答。
<br/>　　当然，对于在例行新闻会现场发生了什么，作为证件事情直接参与者的罗昊和云熙最开始并不知情，他们也是在例行新闻会结束后，通过网上的报道，才了解到国防部发言人那句“我们以后多飞几次，你们习惯了就好”霸气回答的。
<br/>　　……
<br/>　　飞机平安落地后，一行人就搭乘一架赵天宇派来的直升机返回了龙隐部队，将两瓶从西伦科特交给早已在机场等待的研究室工作人员后，跟随赵天宇返回了龙穴。
<br/>　　孙远、张子涛、沈飞三个人的遗体被运回国后就火化了，葬礼是在罗昊他们回来之后才举行的，这也是罗昊提出的要求。
<br/>　　除了还处于昏迷的徐旭东外，其他人全都出席了三个人的葬礼，就连赵国邦也在百忙之中抽空来了。
<br/>　　孙远和沈飞以前都是他手下的兵，他有必要来参加他们的葬礼，送他们最后一程。
<br/>　　张子航坐在轮椅中，看着前方自己弟弟的照片，眼眶不禁发红，这是他唯一的亲人。
<br/>　　孙远的姐姐，孙松岩也回国了。一身陆军常服，肩膀上挂着中校军衔的孙松岩站在队伍，沉默不语。作为军人，马革裹尸是最好的归宿！
<br/>　　相比于孙远和张子涛只有一个人的家属，沈飞家里来的人就比较多了。
<br/>　　沈飞也是军人家庭，哥哥、姐姐、堂哥、堂姐、表哥、表姐，叔叔伯伯都是部队里的，他们身上穿得常服囊括海陆空三军，军衔先是从尉级军官到校级军官不等，其中军衔最高的是一个陆军少将，这是沈飞的舅舅。
<br/>　　沈飞舅舅手里推着一把轮椅，轮椅中坐着一个头发苍白的老人，这是沈飞的外公，原西南军区副司令。虽然从位置上退下来很多年，但是身上的气场还在。
<br/>　　赵国邦走到轮椅前，对着沈飞外公敬了个礼，说道：“吴老，您的外孙是个好兵，他很不错。”
<br/>　　“我想去看看他。”沈飞外公苍老的眼睛中透着晶莹。
<br/>　　赵国邦从沈飞舅舅手中接过轮椅，亲自推着轮椅走到那张摆放着沈飞骨灰盒的桌子前，将沈飞的骨灰盒拿起放在吴老手中。
<br/>　　吴老用他如枯枝一般干枯粗糙的手，轻轻抚摸着手中大理石骨灰盒，泫然泪下。
<br/>　　罗昊从口袋中拿出一张照片，放在正前方的桌子上。照片是西伦科特被击毙后的现场照片，西伦科特的尸体被詹姆斯带回了美国，罗昊能拿到的只有这样一张照片，这也算是对孙远、沈飞、张子涛三个人一个交代。
<br/>　　悼词是赵国邦上台讲的，没有冗长的长篇大论，简短的慷慨陈词，说明三个人出众的军旅生涯。他们很多任务和行动都是属于绝密，保密年限从几十年到无期限不等，所以赵国邦无法明说他们在龙隐部队做了些什么，甚至他们的牺牲原因都是被保密的。
<br/>　　沈飞家里面也只有吴老才知道，沈飞其他的亲属只知道他是在行动中阵亡的，具体原因并不知晓。
<br/>　　龙隐部队陵园中多了三块新刻出来的墓碑，跟先前牺牲的沈俊辉、陶然、田建东、姜伟、吴克翔在同一排。
<br/>　　陵园前的大理石碑上，也多了三个名字。孙远、沈飞、张子涛，在他们的名字下方刻着他们的出生年月和牺牲年月，除此以外再也没有其他多余的信息。
