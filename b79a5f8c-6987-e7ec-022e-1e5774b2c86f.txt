　　第611章
<br/>　　这一场大雨来得猝不及防，将程大雷和小白狼淋得湿透。二人被逼近了一个山洞中，外面鬼哭狼嚎一般，二人在山洞中点了一堆篝火，盘算着等雨停了，就继续赶路。
<br/>　　路程算下来，距离蛤蟆城也不是很远。大概再有三五天就能赶回去，着急赶路的话，还能更快。如今二人的位置，已经算是帝国境内。
<br/>　　荒山野岭，外面风一阵阵刮着，虽然是白天，但云层压得很低，天空暗得像晚上一般。
<br/>　　程大雷将袍子解了，放在火边烤。总算是他有出门多带几套衣服的习惯，才没落到无衣可换的境地。
<br/>　　枯枝在火堆中发出噼啪响声，二人围着篝火，借助飘忽的火焰驱逐身上的湿寒。
<br/>　　小白狼冲山洞外看了一眼，道：“这节气不该有这样大雨的，也不知道这样一场雨，什么时候才能停？”
<br/>　　“那谁说得好，世道不正常天气也正常不了。”程大雷一边烤着衣服一边道：“看这雨是一时半会停不了，咱们估计今天得在这里过夜，明天天亮再上路。”
<br/>　　小白狼想去看看两头座骑如何，它们放在避雨的悬崖下，免得雨下得太大失了惊。结果刚冒个头，小白狼就看到远处树影晃动，正有人向这边过来。
<br/>　　她立刻同程大雷说了，二人的心意是一样的：都不想与这帮人发生接触。谁知道对方是善是恶，莫名接触总会有许多不必要的麻烦。而这里荒山野岭，肯在这样天气赶路的，估计也好不到哪里。
<br/>　　而有时候越不想发生什么，偏偏会遇见什么。这一队人穿过密林，向前面张望了一眼，为首一人高声喊道：
<br/>　　“东家，前面有个山洞，我们快些过去避避雨。”
<br/>　　一帮人踩着水过来，每个人都很兴奋。他们半路上碰到这场大雨，顿时乌云盖顶，东西南北都分辨不清楚。
<br/>　　等发现这座山洞时，立刻迫不及待过来。一行八个人，都想赶紧找个干净地方歇歇脚。
<br/>　　“有人！”
<br/>　　他们刚闯进山洞，就是一激灵，看见程大雷和小白狼守着火堆坐在那里。立时，刚走进山洞的几人便将刀拔了出来。出门在外，身上都有兵器防身，关键是程大雷凶神恶煞的样子，怎么看都不像好人。
<br/>　　“你们是什么人？”当先一人问道。
<br/>　　程大雷摆摆手：“别害怕，别害怕，我们是好人。”
<br/>　　“哪个瓜娃子才害怕。”
<br/>　　他们看到小白狼时倒是松了口气，毕竟程大雷虽然不是好人，但带着女人出来劫道的山贼毕竟少见。
<br/>　　一人分开众人走到程大雷面前，未曾开口，向抱拳行了一礼。
<br/>　　“手下粗鄙，还请二位勿怪。在下东方童，乃是贩草药的客商，天雨失路，我和我的同伴能否在这里躲躲。”
<br/>　　这东方童说话倒也是客气，程大雷拱拱手：“我们也是避雨之人，既然遇上了，也是缘分，诸位请。”
<br/>　　东方童一群人进来后，山洞内立刻拥挤起来。他们一行八人，以东方童为首，据他自己所说，他们是来附近采购草药的京州药商，因为遇上了雨，才被堵到这里。
<br/>　　他手下有个汉子，三角眼，留着两撇短须，身材并不高大，却透着一股激灵劲儿。此人名叫刘凯，是东方童的得力助手。
<br/>　　其他人倒是没给程大雷留下什么深刻印象，他注意到其中有个女人，一副郁郁不乐的样子，从进来开始，没有开口说过话。
<br/>　　大家都围着火堆，有人烤衣服，也有人拿出随身携带的炒米、肉干、烧酒之类的东西进食。。
<br/>　　东方童脸上一直堆着和气的笑容，他将酒囊递过来，道：“好汉要不要喝些酒，暖暖身子？”
<br/>　　程大雷摆摆手婉拒了，他们说自己是商人，程大雷就当他们是药商了？程大雷现在也算是老江湖，这样的亏也并不是没有吃过，他岂能在一条沟里摔倒两次。
<br/>　　程大雷已经分别看过他们每个人的信息，从他们的职业上，倒是看不出有什么古怪。不过这种生活职业，分得也不是很清楚，这些人是乔装改扮的也不是不可能。不过这东方童，刘凯，还有那黑衣女子，都有不错的身手。
<br/>　　但这也说明不了什么，这世道不太平，出门在外，总要有些保命的本事。
<br/>　　一众人饮酒谈笑，东方童话里话外也打探着程大雷的消息。
<br/>　　程大雷谎称自己名家牛三斤，是去凉州投亲的旅人。看东方童的样子，是明显不信任程大雷，不过他也没有拆穿，仍旧陪着笑与程大雷聊天。
<br/>　　雨淅沥沥下起来不停，天已经不知不觉暗下来，外面哗哗的风雨声。东方童的手下差不多已经睡着了，他们走了一路，想来此刻也是疲惫不堪。不过仍有人醒着，很明显是提防程大雷二人。
<br/>　　程大雷也困得不行，他与小白狼轮流守夜。此刻百无聊赖的守在火堆旁，目光下意识落在那黑衣女子身上。
<br/>　　毕竟，在一帮汉子中间，还是这黑衣女子更赏心悦目些。
<br/>　　黑衣女子也没有睡着，她抬起头目光看着程大雷，目光叵测如云雾，当真有几分味道。
<br/>　　这帮人怎么看怎么透着古怪，程大雷也不知道他们是做什么的，如此，倒也激起了程大雷一些好奇心。
<br/>　　过了一刻钟时间，那女子和刘凯结伴走出了山洞。程大雷打了个哈欠，估计女人是小解去了，不过这种事也要男人陪着，架子可是够大的。
<br/>　　过了很长时间，二人一直不回来，东方童皱起眉头。这时候，刘凯突然跑进来，口中道：“东家，她跑了……”
<br/>　　“什么！”东方童一惊：“你怎么让她逃了！”
<br/>　　“我，我……”刘凯满头冷汗，想说什么，明显很害怕。
<br/>　　“快找。”
<br/>　　东方童和刚才的气质截然不同，目露凶光，猛地回头，目光看着程大雷。他的手，已经摁在刀柄上。
<br/>　　相比满口谎话，很明显杀掉程大雷是最简单的方法。
<br/>　　程大雷笑笑，手拍着匹夫剑的剑柄：“怎么会事，人丢了，用帮忙么？”
<br/>(本章完)
