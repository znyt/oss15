　　说干就干，将吞天鼎召出之后，叶尘又把草雉剑和十拳剑的残骸，从乾坤镯中取出。
<br/>　　“草雉剑之中蕴含一丝空间法则，而十拳剑之中则蕴含一丝时间法则，两者融合之后，不知道会发生什么？”
<br/>　　叶尘自言自语了一句，将两把残剑丢入鼎中，然后释放出三昧真火，开始煅烧起来。
<br/>　　一晃眼，
<br/>　　足足用了大半天的时间，
<br/>　　叶尘总算才勉强将两把圣器彻底融化，可是很快他却发现，两者根本无法完美融合！
<br/>　　“主人！这两把剑的材料属性不同，恐怕很难直接融合，不如将嗜血妖龙的龙骨，也也一并融入其中试试！”
<br/>　　奥维斯忽然又开口提醒道。
<br/>　　叶尘顿时觉得大有道理，
<br/>　　一般高级妖兽的精骨，的确是炼器过程中，所必需的的材料之一，具备催化的作用，而且那根嗜血妖龙精骨中，还有一道妖龙的残魂，刚好可以将其炼化成器灵，说不定能炼出一把仙器也未可知！
<br/>　　一念及此，叶尘当即便取出龙骨，也丢入吞天鼎之中。
<br/>　　滋滋滋~
<br/>　　随着三昧真火不断的炼烧，嗜血妖龙的龙骨，开始渐渐与那团灵铁融合。
<br/>　　……
<br/>　　一转眼，三天时间过去了，叶尘始终待在炼丹房之中。
<br/>　　这期间，楚飞烟和苏岚来过几次，苏蔓和萧若曦等人也都各来过一次。
<br/>　　可惜，叶尘在炼丹房周围布下了层层禁制，他们根本无法进入。
<br/>　　终于在第四天的时候，
<br/>　　轰！
<br/>　　炼丹房猛然巨震了一下，连带着整个宗门，都一阵震动。
<br/>　　这边的动静太大，很快便将楚飞烟等人吸引了过来。
<br/>　　不过，等人众人赶到的时候，炼丹房却又忽然安静了下来。
<br/>　　“这家伙已经待在里面好几天，他到底在干嘛？”
<br/>　　楚飞烟眉头紧蹙，隐隐感觉到，那炼丹房之中，似乎有着一股极度危险的气息！
<br/>　　一旁的萧若曦弱弱道：
<br/>　　“他之前说过，要给我炼制养魂丹……”
<br/>　　楚飞烟摇了摇头道：
<br/>　　“不可能！我虽然不知道养魂丹是什么，可是刚刚炼丹房中所发出的那道恐怖的气息，绝对不可能是丹药！”
<br/>　　楚飞烟如今好歹也是神境级别的高手，对危险的嗅觉，自然比之前强大了无数倍。
<br/>　　她声音才刚刚落下，
<br/>　　彭！
<br/>　　整个炼丹房仿佛要炸开了一般，再次发出一道巨响！
<br/>　　继而，
<br/>　　咔嚓嚓！
<br/>　　天空之上，竟然凭空出现一道恐怖的雷电，然后精准无比的劈落在炼丹房之上，
<br/>　　彭！
<br/>　　诺大的一个房间，瞬间便被那道突如其来的神雷，轰成了稀巴烂。
<br/>　　“这……”
<br/>　　所有人目瞪口呆。
<br/>　　等到烟雾散去，众人慌忙定睛一看，顿时更加惊愕的说不出话来。
<br/>　　只见，在那炼丹房上空，不知何时，出现了一把，周身散发着紫色金芒的长剑！
<br/>　　那紫色长剑，通体流光闪烁，而且犹如宝石般璀璨夺目，不过在那华美无比的外表之下，却隐藏这一道极度危险的气息，让人望而却步，可远观而不敢靠近。
<br/>　　“哈哈哈哈！”
<br/>　　就在众人都无比惊愕之时，忽然响起了叶尘的狂笑声。
<br/>　　“没想到我第一次炼器，居然就炼造出了一把准仙器！！”
<br/>　　叶尘大笑之后，伸手一招，那把无比华美的紫色长剑，已经自动飞入叶尘的掌中，继而发出一道清脆的剑鸣声，而且竟然给人一种愉悦欢快的感觉！
<br/>　　叶尘顿时更加欢喜，
<br/>　　此剑为半仙器，已经具备了不低的灵智，而且又是他亲手打造，他便等同于是这把剑的父母。
<br/>　　“好剑！好孩子！以后你就叫紫琼吧！”
<br/>　　“啨~~”
<br/>　　紫色仙剑似乎听懂了一般，顿时再次发出一道清脆的鸣叫声，剑身也跟着一阵乱颤。
<br/>　　叶尘心念一动，紫琼剑便瞬间化作一道紫光，没入叶尘的身体之中了。
<br/>　　他虽然恨不得，立即想要试一试紫琼剑的威力，可是眼下在宗门之中。
<br/>　　而且整个黑森林之前经过一场大战，已经四处废墟，众弟子都在卖力的修整，实在没有让他试手的地方。
<br/>　　收起紫琼剑后，叶尘脚下轻轻一迈，便来到众人跟前，回头望了一眼，已经成为稀巴烂的炼丹房，不由得摸了摸鼻子，有些尴尬，道：
<br/>　　“飞烟，实在不好意思，不小心把屋子给毁了……”
<br/>　　楚飞烟也是一脸的幽怨，
<br/>　　“算了！我回头让人再修就是了！”
<br/>　　说到这里，楚飞烟忽然想到了什么，当即话锋一转，道：
<br/>　　“对了！叶尘，那个……叶将军走了！”
<br/>　　叶尘微微一愣，旋即淡淡哦了一声，并没有多说什么。
<br/>　　楚飞烟又道：
<br/>　　“你就不好奇，他去了哪儿吗？”
<br/>　　叶尘淡淡道：
<br/>　　“他去哪儿，和我有什么关系？”
<br/>　　楚飞烟却再次开口道：
<br/>　　“最近米国在东海一带活动频繁，和华夏海军时有摩擦，两大国之间的战争，一触即发，叶将军已经在前天赶赴前线了！”
<br/>　　说完这话，楚飞烟两眼死死盯着叶尘，想要看看他是什么反应。
<br/>　　可惜，她失望了，
<br/>　　叶尘脸色平淡依旧，似乎对叶天歌的事情，一点也不关心。
<br/>　　楚飞烟微微叹了一口气，道：
<br/>　　“岚姨回饭店之前，让我转告你，她希望你就算是看在她的面子上，去东海走一趟，保证叶将军的安全！”
<br/>　　叶尘听到这话，顿时眉头一皱，
<br/>　　“岚姨她真的这么说？”
<br/>　　楚飞烟和苏蔓等人，都纷纷点头。
<br/>　　“好吧！”
<br/>　　叶尘也不由得暗叹了一声，
<br/>　　“反正我和米国之间的恩怨，也该清算一下了！”
<br/>　　说到这里，叶尘眼眸之中，露出一抹冰冷之色。
<br/>　　……
<br/>　　叶尘虽然答应了下来，不过却并没有立即出发，而是先在宗门周围，以当初在岛国得到的那颗镇元石为根据，布置下了一个护山大阵。
<br/>　　然后，又一口气布置几十个聚灵阵、生灵阵、化灵阵。
<br/>　　直到将楚飞烟搜集的灵石美玉，以及他手里的存货，全部用完，这才罢休。
<br/>　　有了这个护山大阵，便是人仙境界的强者，也无法轻易进入山门！
<br/>　　而且最后，叶尘又在黑森林周围，设下了几道空间禁制，普通人根本无法靠近。
<br/>　　做完这一切之后，叶尘又回了一趟苏氏饭店，和苏岚当面告辞之后，这才朝东海方向而去……
