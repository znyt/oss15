　　古老的混沌门户打开，源源不断的混沌气息从其中喷射而出。

<br/>　　就在楚风眠布置出原有的护山大阵上，一道新的阵法，笼罩在了上面。
<br/>　　这一道阵法，跟楚风眠布置出的护山大阵，相辅相成。

<br/>　　她继承的完整云羽天宫的传承，懂得许多数万年前，云羽天宫的阵法。

<br/>　　但是布置出一个小型的阵法，加持楚风眠所布置的护山大阵，并非是一件难事。

<br/>　　又是一道阵法缓缓升起，这三道阵法，融为一体，庞大的灵力，笼罩这整个剑道门。

<br/>　　楚风眠心中大喜道。

<br/>　　楚风眠可以源源不断的凝聚灵石，但是也不可能一直将时间浪费在这上面。
<br/>　　现在这护山大阵，用来保护剑道门已经是暂时足以了。

<br/>　　在其他星域，其他势力之中，也是一样。
<br/>　　楚风眠回到了九域之中，八劫古帝已经是见到了不少了，但是九劫古帝一个都没有见到，足以证明九劫古帝的稀少了。
<br/>　　等到楚风眠前往归墟天，在夺得几座灵泉，打入到剑道门地下之后，整个护山大阵的威力，还足以在提升不少。
<br/>　　“这就是我的势力！”
<br/>　　楚风眠目光扫视着下方，无数剑道门弟子，都是看到这护山大阵的变化，一个个惊呼起来。
<br/>　　以他们的实力虽然无法看出这护山大阵的全部玄妙，但是粗略一看，也足以明白，这比起曾经当年北邙学院的护山大阵，要不知道好上多少倍。
<br/>　　“这一方小世界，好好经营，未来未必会比起那圣龙秘境的圣地要差。”
<br/>　　楚风眠看着剑道门周围的一切，感叹道。
<br/>　　当初楚风眠的大师兄剑凌，在选择建立北邙学院的时候，也是花费了大量的力气。
<br/>　　这是一方彻底独立的小世界，如果封闭起来的话，只怕是让九劫古帝，也难以寻找得到。
<br/>　　遇到危机的时候，只要是封闭起来，足以度过劫难。
<br/>　　现在这剑道门，就是楚风眠的老巢了，日后哪怕是楚风眠回到圣龙秘境，彻底的撕破脸，他也可以带着青梦师姐，罗康等人，来到剑道门。

<br/>　　楚风眠看了一眼，平静的说道。

<br/>　　邓灵，弑剑君，洛河三人恭敬的说道。
<br/>　　楚风眠点了点头，一步上前，来到了大殿之下。
<br/>　　这大殿，就是云羽天宫，现在也是剑道门的主殿。
<br/>　　楚风眠踏入大殿，在大殿之外，无数剑道门弟子，见到楚风眠的来到，纷纷跪拜。
<br/>　　楚风眠一步一步的走入大殿。
