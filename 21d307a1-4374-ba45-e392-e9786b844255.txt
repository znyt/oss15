<br/>				<br/>　　    凤九坐在厅中喝着茶，任由着厅中的一些护卫和婢女打量着，正好到了吃饭的点了，而她也还没吃，此时肚子饿了，便对站在一旁的护卫道：“给我弄点吃的来，我肚子饿了。”<br/>　　<br/>　　    她说得自然，如同在自己家中一般，全然没有半点拘束以及紧张感，倒是那护卫听到她的话后愣了一下，才应了一声后退了出去。<br/>　　<br/>　　    喝着茶的她在等着，两杯茶都喝完了，也还没见那城主的身影，明显，对方是想晾着她在这里给她来记下马威。<br/>　　<br/>　　    然而，喝了两杯茶后的凤九却是在那些一些端上点心后，吃起了点心来，倒是城主未到，那少城主反倒先过来了。<br/>　　<br/>　　    一进大厅，见厅中只有那红衣男子一人，正吃着点心时，他黑沉着脸问：“你不是走了吗？又为什么要回来？”<br/>　　<br/>　　    凤九睨了他一眼：“谁说是我要回来的？是你父亲让人请我过来的。”说着，唇角一勾，露出一抹邪邪的笑容来：“你现在可是我的奴隶，怎么？奴隶见到主人不用行礼？”<br/>　　<br/>　　    听到这话，那凌天羽脸色一变，他不由的倒退了一步防备的盯着凤九：“这里可是我家！你不要太过份了！”<br/>　　<br/>　　    “过份了吗？我不觉得啊！”她轻笑着，拿起一个点心吃着，一边道：“过来，给我倒酒。”<br/>　　<br/>　　    几乎是凤九这话一出，那凌天羽整个人便不由自主的走了过去，他心中一浮起抗拒时，眉心处就仿佛千刀万虫在钻动着一般，难受得让人抓狂。<br/>　　<br/>　　    “愣着干什么？上酒来！”他沉着声音对着一旁的护卫即喝着。<br/>　　<br/>　　    那护卫一见，连忙让人上酒，送上酒来后，那凌天羽便亲自为凤九倒酒，如同一个侍候的小童一般站在一旁候着。<br/>　　<br/>　　    听到消息的城主大步朝大厅走来，当看到那厅中的一幕时，险些没气晕过去。他深吸了一口气才走上前，锐利的目光带着审视与打量的落在那一袭红衣举止洒脱从容的红衣男子身上。<br/>　　<br/>　　    就是这人赢了他儿子？<br/>　　<br/>　　    目光带着神识的从她的身上扫过，见这人竟是仙帝的实力，而且一身气息内敛，很不简单，这么看来，他儿子会输他倒也是情有所原。<br/>　　<br/>　　    在他打量凤九时，凤九也在打量着他，淡淡的一瞥，见对方身上上位者气势强大，而且进了这里也并没收起身上的气息，显然是为了震摄于她，只是，她是凤九，不是别人。在对方强大的上位者气势之下，她依旧如初，面上神色连一点变化也没有。<br/>　　<br/>　　    “这位便是凤公子了？”城主迈步走了进去，同时询问着。<br/>　　<br/>　　    “正是。”凤九行了一礼，道：“不知城主请我过来，所为何事呢？”<br/>　　<br/>　　    哼！这是明知故问了！<br/>　　<br/>　　    一旁的凌天羽在心下冷哼一声，心下怨言甚大，偏偏一副敢怒不敢言的模样。谁让他输给了凤九呢？哪怕再不愿，此时也只能站在这里为他倒酒了。<br/>　　<br/>　　    城主听到凤九的话时，也是怔了一下，继而笑道：“是这样的，我希望凤公子可以解开与我儿的奴隶束缚。”<br/>
