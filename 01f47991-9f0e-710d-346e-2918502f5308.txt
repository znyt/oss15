　　第468章
<br/>　　一场追逐战在草原上展开，由夜半开始，进行到黎明尚未结束。
<br/>　　薛丁山小胜获得的胜利，早因这场追逐战而烟消云散。过程中他发现一个事实，程大雷逃得很快，明明刚才还在眼前，转眼就消失了踪影。再往前，程大雷就在不远处等着他们。
<br/>　　一次又一次，帝国军在追逐过程中筋疲力尽。随之，军中已有怨言腾起，薛丁山斩了一个，才压下去士兵的不满。
<br/>　　薛丁山太渴望这场胜利了。无论是恶名还是威名，反正程大雷已经扬名天下。帝国不知有多少人想踩着程大雷的肩膀往上爬。此刻程大雷虎落平阳，还不正是痛打落水狗的最好机会。
<br/>　　有一个念头，薛丁山和程大雷想的一样：今天不杀，怕以后就没有杀死他的机会了。
<br/>　　而与之相反，程大雷的自信心却渐渐回来了。仗着神行加成，薛丁山想要追上自己，几乎可以说是痴心妄想，而己方全部是骑兵，长途行军依旧能保存体力。但敌人……怕现在累得双腿都在打摆子吧。
<br/>　　“杀！”
<br/>　　程大雷突然转身，杀了一个回马枪，二百骑兵如一杆长矛将帝国军捅穿，程大雷并没有恋战，收割一批人头后边撤出战斗，将敌人甩在身后。
<br/>　　薛丁山压抑着愤怒，纵然心头怒火燃烧，但他表情依旧很平静。对方兵种占有绝对优势，想要追上敌人几乎是不可能的事情。然而，就这样任程大雷逃脱，也绝不是他愿意看到的。
<br/>　　“撤退。”薛丁山下令。
<br/>　　手下人则大呼幸运，心头松了口气，想着可以早点回营，还能赶上吃午饭。
<br/>　　看到这一幕，薛丁山的心情简直糟糕到极点。靠着这些人，怎么上阵打仗。
<br/>　　令帝国军没有想到的是，当薛丁山下令逃跑时，程大雷反而缀了上来。像草原上追逐猎物的黄狼，只等猎物筋疲力尽时，才会扑上来，用牙齿咬断猎物的喉咙。
<br/>　　薛丁山却似乎不知道情形有多糟糕，带着士兵有条不紊的回撤，当程大雷攻上来时，便布置防御。一群疲兵自然不可对程大雷形成有效的抵抗，但薛丁山却尽力将伤亡将到最小。
<br/>　　过程中，帝国军一直在减员，有一千人死在义字军的马蹄下。
<br/>　　在中午，程大雷又一次发动进攻，程大雷不介意用这种手段，一直将帝国军耗死，然后，在一斧切掉薛丁山的脑袋。
<br/>　　但这一次进攻，发生了意外。正在程大雷杀得痛快，想着要不要顺势砍了薛丁山时，从后方突然杀出一哨人马。人数虽不多，大约只在一千左右，却是精锐的骑兵，由陆承领队。
<br/>　　“杀！”
<br/>　　“走啦，走啦！”
<br/>　　程大雷挥舞着斧头，撤走厮杀得痛快的义字军，向远处逃去。
<br/>　　“薛将军，薛将军……”关键时刻，陆承带着援军赶到。
<br/>　　“没什么，我已将程大雷耗在此地，我们此刻杀过去，正是敌方虚弱时。”薛丁山。
<br/>　　“是，崔相已说了，这一千骑全部由薛将军指挥。”陆承。
<br/>　　“好了，我去追敌，这些人就交给陆军师了。”
<br/>　　忍着耗损兵员，也要把程大雷拖在此地，薛丁山心里其实一直压着一股火。这次有了精锐力量补充，他立刻带上骑兵队伍，去追击程大雷。
<br/>　　然后，他发现一个糟糕的事实，即便以骑兵追击，仍然赶不上程大雷的速度。感觉程大雷的队伍像是长着翅膀一般，不紧不慢，可你偏偏就是赶不上。
<br/>　　程大雷带着人跑得像兔子一般，谁能想到薛丁山这么恨，拼着三分之一的伤亡，就是为了拖住自己。
<br/>　　慈不掌兵，义不行贾，老实说，这薛丁山比自己更适合带兵呀。
<br/>　　当然，因为神行的作用，薛丁山想要追上自己也不容易。不过，今日杀不掉薛丁山，程大雷总是有些惋惜。
<br/>　　他骑着黑牛逃跑，偶尔停下，冲追得气喘吁吁的薛丁山比个鬼脸。
<br/>　　正在此时，从程大雷对面杀出来一捎人马，约在千人左右，为首一员大将，冲着程大雷大喝。
<br/>　　“程大雷，你哪里逃！”
<br/>　　程大雷急忙转向，没想到此地还有伏兵，赶紧带着士兵往东走。
<br/>　　东方一座草坡上杀出一哨人马，截断程大雷的去路。
<br/>　　“程大雷，你哪里走！”
<br/>　　程大雷又一次改变方向，往东北去，东北树林中杀出一队人马，带队是个用棍的大将。
<br/>　　东北、东南、西北、西南……四面八方，程大雷被团团围住。每支队伍的人数并不多，多的千余人，少的数百人，却是占据了程大雷能逃走的任何一方位。
<br/>　　他们口中高声喝着，层层推进，缩短着对程大雷的包围圈。
<br/>　　这不完全是相府的队伍，包括将军府，扬州军以及其他诸侯。
<br/>　　杀死林少羽后，对帝国朝廷来说，正义教的事情已了，不过就是剩下程大雷这个小尾巴而已。在这几天，各路诸侯都派出人手追杀程大雷。所以这支草原上行动的不仅仅有薛丁山一支，陆承遭遇程大雷的事情已经传开，各家的队伍都往这里赶，终于在此刻汇合，将程大雷围了个水泄不通。
<br/>　　如今，程大雷颇有种九头虫遭遇天兵天将追捕的感觉，上天无路，入地无门，不管你走到哪里，都会探出个脑袋，大喝一声：“哪里走！”
<br/>　　程大雷的路的确被堵死了，七撞八撞撞不开这重围，最后程大雷被堵到一道大河前。回头望去，帝国军层层推进，距离越来越短，短到能看清楚每个人脸上的表情。
<br/>　　身前是滔滔不休的大河，身后是一层又一层的帝国军。现在程大雷能体会林少羽走投无路的感觉，但这一次，却没有杀虎口那样的天险供程大雷周转。
<br/>　　一员银戟小将拍马向前，口中喝道：“程当家，某乃京州薛丁山，可愿在我戟下送死？”
<br/>　　程大雷恨得牙痒痒，他还是第一次认真打量薛丁山，委实说，模样长得真俊呐。现在他恨得牙痒痒，一恨薛丁山这小白脸，二就是很系统……
<br/>　　如果被自己召唤出来的人弄死，当真是没处说理去。
<br/>(本章完)
