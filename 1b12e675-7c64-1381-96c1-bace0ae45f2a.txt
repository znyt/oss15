　　第153章
<br/>　　差不多时间，来自幽州城的物资到了青牛山。
<br/>　　马车共运来五百双军靴，随车还有李善言的一封信。心中内容简略来说：幽州王已经开恩，你也不要给脸不要脸，麻溜快的出兵。
<br/>　　程大雷又给李善言写了封回信，心中内容大概是：军靴已经收到了，山寨上下都很感激幽州。大家都明白，这军靴是幽州王给的面子，大家踩的那里是鞋啊，根本就是幽州王的脸。
<br/>　　无论如何，总是从幽州王那里搜刮点东西，山寨有五百人换上新靴。
<br/>　　这样，大家干起活来就更方便。
<br/>　　随着时间流逝，蛤蟆寨的建设已快竣工，程大雷心里盘算着日子，说不得自己能在开春之前，成功把山寨升到三级。
<br/>　　程大雷琢磨着，自己能否趁着这段时间，再从杨龙停那里刮点东西。反正也不嫌少，得点是点。
<br/>　　而这一天还没过去，程大雷就收到一个消息。
<br/>　　杨龙停兵发伏虎山，趁夜攻城，以二百老弱佯攻，将熊老大诱出山寨。而自己则趁着山寨空虚，一股脑闯了进去，再以山寨的老弱妇孺为盾牌，强攻熊老大。
<br/>　　熊老大的老婆孩子都死在这场战斗中，手下喽啰伤亡无数，熊老大由几个亲信护着，闯出重围，现在下落不明。
<br/>　　一夜之内破山，从战斗开始到结束，不过两个时辰。而杨龙停总共动用的兵力，不过一千。
<br/>　　将为兵之魂，这一千人在罗铁手中只能做丧家犬，但在杨龙停指挥下，却杀敌如杀狗。
<br/>　　这次，幽州王在所有人面前露了露牙齿，熊老大是个猛人，但杨龙停却比他更加猛。莫看他年逾古稀，一口老牙还硬得很，什么都吞得下。
<br/>　　整个幽州道的绿林匪人在听说这件事后，吓得体若筛糠。
<br/>　　包括程大雷。
<br/>　　本以为将山贼重建好后，就能撑得过这次事情，但根据这件事判断……估计悬。
<br/>　　“都愣着干嘛，赶紧干活，把墙再加高三尺。”程大雷冲周围人。
<br/>　　“秦蛮！”
<br/>　　“怎么了？”
<br/>　　“快骑马，把我的信追回来。”程大雷跺脚大喊。
<br/>　　本来还琢磨着从幽州王那里刮点东西，自己信里还半开玩笑的点了他几句。可谁知道他这么猛，他打伏虎山用一千人，打蛤蟆寨估计也用不了多少人。
<br/>　　马比马车快，在半路上秦蛮追上幽州城的马车，将程大雷的信讨了回来，又递上一封程大雷新写的信。
<br/>　　这封信上只有四个字：立时出兵。
<br/>　　……
<br/>　　伏虎山虽破，但余烬未净，熊老大也还没有死。这次出兵，就是搜捕逃进山林的熊老大。
<br/>　　幽州王传下令来，凡是归顺的绿林匪人，都要参与这件事，谁若抓到熊老大，不管死活，幽州王统统有赏。
<br/>　　伏虎山一战让大家明白，杨龙停宝刀未老。对于他传下的军令，无人再敢违抗。几乎整个幽州道的匪人这几天都翻山越岭，寻找熊老大的踪影。
<br/>　　程大雷也点了五六十号兵，显得自己时刻跟随幽州王的脚步。
<br/>　　这一日午后，几人在山林中择了一地进食。
<br/>　　“大当家，俺今年都六十三了，能不能下次就别让俺出来了。”一个老头道。
<br/>　　“老张，不是我说你，你要有老骥伏枥志在千里的劲头，你看看人家杨龙停，实在不行，你就把你的刀让小黑拿着。”
<br/>　　小黑是个才十三岁的少年：“张爷爷，我能替你拿，我力气很大的。”
<br/>　　秦蛮无奈的看着眼前这一幕，程大雷的确选了五六十人，可都是山寨的老弱，老得半截身子都快埋进土里，小的没有一个过十五。
<br/>　　“大当家，那熊老大可凶得很，这几天，我听说又有追捕他的人被他杀了。咱们这些人要是碰上熊老大，估计不够他们打。”秦蛮不乏担忧道。
<br/>　　“什么话，咱们哪有那么倒霉，偏偏能遇到他。”程大雷咬了口干面包：“晃悠完今天咱们就回去，这干粮越吃越难吃了。”
<br/>　　说着，程大雷冲秦蛮道：“秦蛮，你是猎户出身，就不能打点东西，给咱们改善改善伙食。”
<br/>　　秦蛮憨憨一笑：“这冰天雪地的哪有猎……”
<br/>　　秦蛮脸上的笑容忽地怔住，双目直勾勾的看着前面：“熊……”
<br/>　　程大雷顺秦蛮的目光看去，果然见一头黑熊从大树后绕出来。
<br/>　　“这头熊怎么看得有些眼熟？”秦蛮。
<br/>　　“熊不都是一个熊样！”程大雷口水都快流出来了：“听说熊掌很补的……秦蛮，看你的了。”
<br/>　　秦蛮提着自己的大枪站起来，顺着树干悄悄趴在大树上。程大雷率领他的老弱之兵，静悄悄的不发出任何声音。
<br/>　　只见秦蛮顺着树干趴到黑熊头顶，双手握枪，猛地从树上跳下来，枪尖向黑熊头顶狠狠扎去。
<br/>　　……
<br/>　　熊老大最近很不痛快，妻儿老小被杨龙停赶尽杀绝。悬赏他的赏银越来越高，不知有多少人正漫山遍野追捕自己。曾经笑傲一方的绿林贼枭，如今像条丧家犬一样惶惶逃命。
<br/>　　更悲催的是，自己的坐骑丢了。曾经熊老大全仗着这头黑熊提升格调，熊老大珍爱它甚于珍爱自己的斧头。
<br/>　　可逃命时慌不择路，与黑熊走散了。如今熊老大一边要躲避追兵，一边还要寻找自己的熊。
<br/>　　那畜生凶得狠，若在哪里闹出人命，岂不就泄漏了我的行踪。
<br/>　　转过一个山坡，熊老大忽然有种不好的预感，向前进了几步，看到一堆人正围着火堆大快朵颐。
<br/>　　“秦蛮不是我夸你，你这身手真的没得挑，一枪就把这头黑熊撂倒了。”
<br/>　　“这熊好像很虚弱，不然也不能这么轻松。”
<br/>　　“好啦，好啦，不要谦虚。大家来尝尝这熊掌，打猎我不行，烧烤我可是一绝！”
<br/>　　忽然意识到什么，程大雷猛地抬头，看到隔着几棵大树，熊老大拎着双斧，双目冒火的看着自己。
<br/>　　“我的熊！”
<br/>　　“我不知道，是真不知道。”程大雷尴尬的看着手中的熊肉：“要不，你也来点儿。”
<br/>　　如果程大雷知道这黑熊是熊老大的坐骑，他是绝对不舍得杀的，骑头熊该有多威风。
<br/>　　“狗贼，去死吧。
