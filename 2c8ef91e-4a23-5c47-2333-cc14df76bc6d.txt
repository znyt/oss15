        起初，冷夫人所说的那件事情，童筱颖转身就给抛之脑后。
<br/>
<br/>        殊不知，她的心里其实一直有种空空的感觉。
<br/>
<br/>        “我们去吃海鲜吧！我听说，临海城最出名的就是海鲜大餐！”童筱颖笑着说。
<br/>
<br/>        傅夜沉走在童筱颖的身侧，应道：“好。”
<br/>
<br/>        他现在对她可谓是百依百顺，只要她开心，让他做什么事都好。
<br/>
<br/>        失去她的这半年里，磨掉了他一身的冷傲，哪怕现在只是远远地看着她笑，他都已经心满意足。
<br/>
<br/>        沙滩上的夜宵摊，摆在了一排木屋的露台上，带着咸味的海风迎面扑来，潮湿中透着一丝凉爽。
<br/>
<br/>        来给他们点单的女服务员，一双水灵的眼睛，直勾勾地落在傅夜沉的身上。
<br/>
<br/>        傅夜沉一袭白衬衫，黑色的休闲裤，衣襟顶头松开两颗扣子，领口微微敞开，露出的锁骨，很是性感。
<br/>
<br/>        且他与生俱来的尊贵和优雅的气质，与这周遭的环境显得有些格格不入。
<br/>
<br/>        女服务员看傅夜沉看得有些出神。
<br/>
<br/>        而傅夜沉则手拿着点单平板，很认真地在看菜单。
<br/>
<br/>        童筱颖瞄了一眼女服务员，又看了看傅夜沉，见他完全没注意到身旁多了个小迷妹，忍俊不禁道：“大哥，妈今天跟我说，要我帮你介绍个女朋友，只要人好心善，其他条件都无所谓。”
<br/>
<br/>        女服员一听这话，顿时一阵欣喜。
<br/>
<br/>        原来这个男人还没交女朋友啊！
<br/>
<br/>        “宝贝，你是想给我开‘后宫’吗？。”傅夜沉一本正经地继续看菜单，头也不抬地回应着童筱颖的话。
<br/>
<br/>        女服员刚喜上眉梢的表情，瞬间阴沉。
<br/>
<br/>        童筱颖尴尬地瘪了瘪嘴。
<br/>
<br/>        “我点好了，就这些。”傅夜沉将手中的平板递给女服务员，抬眸意味深长地看着坐在圆桌对面的童筱颖。
<br/>
<br/>        女服务员接过平板后，低着头，悻悻地去了后厨。
<br/>
<br/>        是她童筱颖提议要来这种小夜宵摊上来吃东西的，不然，以他傅夜沉的身份，必定是要去临海城里的高档餐厅。
<br/>
<br/>        不过，今晚这里的夜宵摊倒是被他傅夜沉一个人全包了。
<br/>
<br/>        所以，露台上一排排的空座，让他两显得格外突兀。
<br/>
<br/>        他好久没有这样明目张胆地在外抛头露面了。
<br/>
<br/>        自从他救了季雅珊后，他便过起了“地下城”的生活。
<br/>
<br/>        他若是想躲，就没有人能找得到他。
<br/>
<br/>        傅夜沉给自己造了一座“地下城”，那天他带童筱颖去的地方，还只不过是他整个“地下城”的一颗芝麻。
<br/>
<br/>        只是，现在令他担忧的是，夏阳枭会不会言而无信。
<br/>
<br/>        海鲜大餐上桌后，童筱颖的嘴巴就一直没停过。
<br/>
<br/>        她比以前瘦了许多，肤色有些病态的白，倒是这胃口大增。
<br/>
<br/>        傅夜沉吃得比较斯文，还不停地关问她：“慢点吃，别噎着了。”
<br/>
<br/>        饱餐一顿后，他又陪着她在沿海的公路上散步。
<br/>
<br/>        这条路很宽，又因为靠近海边郊区，所以，一到晚上，几乎没有往来的车辆。
<br/>
<br/>        童筱颖一边吹着海风，一边面对着傅夜沉倒退着走，举着手机给他录像。
<br/>
<br/>        这长得帅的男人，就是很上镜，录出来的视频，就像个男明星拍的微电影。
<br/>
<br/>        她走着走着，突然停了下来，目光落向了傅夜沉身后，远处的那一座高高的崖岸。
<br/>
<br/>        傅夜沉见童筱颖脸上的笑容渐敛，下意识地顺着她的目光回过头去。
<br/>
<br/>        当他看到那座高大的海上石崖时，心口上莫名其妙地涌上一股闷气。
<br/>
<br/>        半年前，那是筱颖为了他选择自我了结的地方。
<br/>
<br/>        她当时，一定很绝望……
<br/>
<br/>        “筱颖，对不起。”傅夜沉忽然低沉道。
<br/>
<br/>        童筱颖蓦然回神，只是微微一笑：“你为什么要跟我说对不起？”
<br/>
<br/>        “因为，我没能好好保护你”傅夜沉深深地自责起来。
<br/>
<br/>        童筱颖挪步来到傅夜沉的跟前，踮起脚尖，抬手抚平这个男人紧蹙的眉心，欣慰地笑着说：“其实，是我应该谢谢你才对。谢谢你找到我，谢谢你收留我。”
<br/>
<br/>        如果不是这个男人，她真不知道自己离开实验室后，该何去何从。
<br/>
<br/>        就在此时，不远处传来机动车的鸣笛声。
<br/>
<br/>        “突突突”地很是刺耳。
<br/>
<br/>        下一秒，一个戴着黑色头盔的男人，骑着摩托车像是失控了般飞速向他们撞过来。
<br/>
<br/>        傅夜沉眼疾手快地搂住童筱颖往旁边躲，但是摩托车的前轮还是撞到了他的腿上，童筱颖手中的手机也因为惯性而从手里甩了出去。
<br/>
<br/>        而那个骑摩托车的男人也摔倒在了地上，摩托车沿着地面擦出了火花星子，滑出了一段距离后，突然“嘭”地一声，便爆炸了。
