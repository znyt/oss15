了”<br/>　　<br/>　　    王惊蛰淡淡的说道：“你敢去自然可以去，但我不得不事先跟你说一嘴，湖底古城的状况，我们现在能了解的连根毛都算不上，谁也不知道再探古城会发生什么状况，就别说你了，哪怕是我也有可能去了就回不来了”<br/>　　<br/>　　    李菲儿不可置信的问道：“那你又为什么敢去？”<br/>　　<br/>　　    王惊蛰沉默的摇了摇头没有解释，他自然不会告诉李菲儿，我本来也不过几年好活，所以我一直努力的挣扎着，为自己寻求一切可以越过那道坎的机会。<br/>　　<br/>　　    也许，那个机会就在这座湖底也说不定呢。<br/>　　<br/>　　    两人在湖边聊着的时候，两辆从机场开过来的警车途径到湖边，车里的人看见岸边的王惊蛰后就让司机停了下来，随即林文赫和王令歌一起走了下来。<br/>　　<br/>　　    “咣当”<br/>　　<br/>　　    “咣当”<br/>　　<br/>　　    两声关车门的动静响起，王惊蛰回头看见车旁站着的人，起身拍了拍手就迎了过去。<br/>　　<br/>　　    “欢迎来到美丽的抚仙湖……”王惊蛰张开胳膊，朝着他俩抱了上来。<br/>　　<br/>　　    王令歌叹了口气，说道：“我一堆事呢，哥们你十万火急的把我召唤来，到底是为啥啊？”<br/>　　<br/>　　    “一会再说！”<br/>　　<br/>　　    王惊蛰朝着林文赫眨了眨眼睛，说道：“你看我牛逼不，没过几个月就给你找了个减刑的机会”<br/>　　<br/>　　    林文赫摇头说道：“我不看”
