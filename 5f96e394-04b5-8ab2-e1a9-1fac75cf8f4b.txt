　　轰…轰…轰…
<br/>　　八龙焚火炉上，此刻的抖动，爆发着一声声剧烈轰鸣。
<br/>　　萧逸始终感知着炉内情况。
<br/>　　随着第二道封印的解开，火焰空间增加一倍。
<br/>　　自然的，涌入的金光力量，亦愈发多，而此时，更是已达到填满的地步。
<br/>　　金光力量愈发多，以愈发强猛，继续开始冲击着第三道封印。
<br/>　　而八龙焚火炉上的第三道封印，显然亦非寻常封印。
<br/>　　故金光力量的冲击，让得此刻整个火炉内轰鸣不断。
<br/>　　萧逸感知着，脸上期待之色，愈发浓郁。
<br/>　　自从以前在东域解开了第一道封印后，之后的封印，萧逸一直无可奈何。
<br/>　　哪怕是以他现今实力，也奈何不了第二道封印。
<br/>　　可这里的金光力量，却帮他冲破着封印。
<br/>　　这对萧逸而言，毫无疑问是一次惊喜，也是一次极大的收获。
<br/>　　刚才那只兽爪，应该便是第二道封印空间里的东西。
<br/>　　第二道封印被解开，第二片空间折叠而出，与第一道空间融为一体。
<br/>　　自然的，那只兽爪刚才便忽然出现。
<br/>　　第二道封印空间内，有着这只莫名而强大的兽爪。
<br/>　　那么第三道封印空间内呢？
<br/>　　萧逸愈发期待着。
<br/>　　此时，源源不断的金光力量涌入八龙焚火炉内，冲击着第三道封印。
<br/>　　看情况，想必也要一段时间。
<br/>　　当然，能否冲破也是一个问题。
<br/>　　毕竟里头封印萧逸一直无可奈何，可想而知其封印之强。
<br/>　　便看这里的金光力量有多强了。
<br/>　　萧逸暂时收回感知，打量着自己的左手。
<br/>　　此时，左手已成兽爪。
<br/>　　黝黑的手臂，鳞片点点，仿佛蕴含无坚不摧之力。
<br/>　　鳞甲之间，点点火红光芒，看起来更像是一个个即将爆发的火山蕴含其中，爆发力惊人。
<br/>　　不过，这是什么妖兽的爪子？
<br/>　　以萧逸的眼力，竟也认不出这属于什么妖兽。
<br/>　　但毫无疑问，这必是一头实力极其恐怖的妖兽。
<br/>　　“炎龙吗？”萧逸自语一声。
<br/>　　这是他一瞬间产生的念头。
<br/>　　毕竟，这是八龙焚火炉内封印之物。
<br/>　　而据剑姬前辈所说，里头封印的，乃是八道龙炎吐息。
<br/>　　可现在，里头封印的，显然不是龙炎吐息，而是强大的妖兽之手。
<br/>　　“怎么可能呢。”萧逸摇头一笑，瞬间否定了这个念头。
<br/>　　先不说炎龙乃是传说中的神兽，是否存在还是两说。
<br/>　　即便真的存在，又有什么东西能将它击杀？甚至将其封印于此？
<br/>　　炎龙大陆，以炎龙为名，可想而知即便炎龙真的在上古时期存在，也是处于大陆当之无愧最巅峰的存在。
<br/>　　萧逸摇了摇头，没有再乡下去。
<br/>　　这只手，属于什么妖兽，他无法确定。
<br/>　　或许，之后去猎妖殿总殿一趟，查阅一些上古典籍，能查到是什么妖兽也不一定。
<br/>　　现在，就不去纠结太多了。
<br/>　　恰在此时。
<br/>　　嘭…
<br/>　　半空中，一阵惊天爆响。
<br/>　　一股火焰，以极其狂猛的速度溢散而出。
<br/>　　这是第三道封印被冲破的迹象。
<br/>　　萧逸早有准备，一剑劈出。
<br/>　　30道星光长河护于身前。
<br/>　　不过，这一次，30道星光长河竟是瞬间便在火焰的溢散爆发下溃散。
<br/>　　嘭…一声爆响，萧逸直接被这股火焰轰飞百米。
<br/>　　待得火焰溢散消失，萧逸已被轰得灰头土脸，满身狼狈。
<br/>　　“好强的火焰溢散。”萧逸吃了一惊，一个闪身回到原地。
<br/>　　萧逸看向八龙焚火炉，放出感知。
<br/>　　果然，第三道封印，已然被冲破。
<br/>　　那片火焰空间的大小，再次增加。
<br/>　　这一次，萧逸特地注意了起来，果然，火焰空间内，再次凭空出现了一兽手。
<br/>　　“出。”萧逸低喝一声。
<br/>　　一团赤红色光芒，从八龙焚火炉内激射而出，聚集于萧逸右手掌中。
<br/>　　又是一阵剧烈的炽热感，萧逸的右手，渐渐被黝黑鳞甲所包裹，不多时，化作了兽手。
<br/>　　第二道封印解开后，空间内出现的，是这头不知名妖兽的左手。
<br/>　　而今，第三道封印出现的，是右手。
<br/>　　萧逸看了眼自己的双手。
<br/>　　这双已化作妖兽身躯的手，鳞甲密布，看起来瘆人而狰狞。
<br/>　　但那散发着的点点赤红光芒，又让这双狰狞的手，看起来充满了爆炸性的威力。
<br/>　　萧逸握了握拳头，一拳轰出。
<br/>　　轰…空间瞬间炸裂。
<br/>　　一股气流，陡然从中爆发，直冲千米。
<br/>　　气流中，带着惊人的炙热气息，沿途所过，地面悉数化作焦土。
<br/>　　“厉害。”萧逸赞叹一声。
<br/>　　目光，再次看回八龙焚火炉。
<br/>　　此时，八龙焚火炉内，仍旧极速地融入着金光力量。
<br/>　　萧逸看了眼巨大坑洞之内。
<br/>　　原本，整个巨大坑洞内，金光氤氲。
<br/>　　那浓郁而浑厚的金光力量，充满了这个深不见底的巨大坑洞。
<br/>　　可现在，就萧逸肉眼可见的深度范围内，金光已然悉数消散。
<br/>　　准确来说，是金光已悉数涌入八龙焚火炉内。
<br/>　　萧逸直视而下，在坑洞的极深之下，仍旧有着点点光芒。
<br/>　　证明着，下方仍旧有金光力量。
<br/>　　不过到底还有多少，就不得而知了。
<br/>　　毕竟这个坑洞有多深，萧逸也不知晓。
<br/>　　只知，肉眼难见的深度之下，仍旧有金光溢散而出。
<br/>　　两个时辰后。
<br/>　　八龙焚火炉上，再次传来轰鸣声，再次剧烈抖动起来。
<br/>　　“开始冲击第四道封印了。”萧逸脸上的喜色，浓郁到极点。
<br/>　　在他的感知中，那片更加巨大的火焰空间内，再次被金光力量所填满。
<br/>　　继续地涌入金光力量，便也在等同冲击着这片空间，冲击着折叠空间中的封印。
<br/>　　第四层空间内，有什么？
<br/>　　萧逸有些期待。
<br/>　　当然，他也迫切想知晓八龙焚火炉上，到底还有着什么秘密。
<br/>　　若能冲破八道封印，里头秘密必展露无遗。
<br/>　　半个时辰后。
<br/>　　八龙焚火炉上，一阵剧烈的火焰冲击而出。
<br/>　　这一次，几乎是火焰冲击的一瞬间，萧逸已脸色大变。
<br/>　　那炙热的气息，单单是传来的温度，已高得发指，甚至让得地面融化。
<br/>　　他手中冷焱剑，更是冒出阵阵热气。
<br/>　　“好强的火焰。”萧逸脸色大变，一瞬间便猜出单凭冷焱剑绝对挡不下这道火焰冲击。
<br/>　　“冰玄封界阵，起。”萧逸瞬间暴喝。
<br/>　　天机圣盘凭空出现于手中，早已布好的大阵顷刻发动。
<br/>　　轰…轰…轰…轰…
<br/>　　一连四道轰鸣，从八龙焚火炉中响现。
<br/>　　一连四道火焰冲击，从中爆发。
<br/>　　待得轰鸣消散，火焰冲击消失。
<br/>　　萧逸看了眼大阵之外，瞳孔一缩。
<br/>　　大阵之外，方圆数十里，已是一片灼热火海。
<br/>　　树木，瞬间在高温下蒸发虚无；地面，已然融化。
