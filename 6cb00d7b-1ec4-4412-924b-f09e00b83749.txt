　　“其实老夫一开始最看好的人，也是你这个小家伙。”
<br/>　　器灵老者看着楚风眠，笑着开口道。
<br/>　　“很好，果然你没有辜负老夫的看重，跟我走吧！”
<br/>　　器灵老者缓缓开口道，周围的空间，突然破碎从组，化为了一道巨大无比的隧道。
<br/>　　楚风眠一眼望去，都看不出这隧道的尽头，到底是有着什么。
<br/>　　“好可怕的力量！”
<br/>　　这器灵老者出手的一刻，楚风眠似乎都无法看出他的实力，达到了达到了哪一步。
<br/>　　比起什么碎命之境的强者，还要强大的多。
<br/>　　尤其是这器灵老者，还仅仅是一个器灵，而并非武者，器灵，在印象之中，便是足以辅佐灵器之用。
<br/>　　楚风眠还从未想象过，一尊器灵，居然会强大到如此地步。
<br/>　　“哼，这算什么，在真正强大的器灵面前，这老头还算不得什么。”
<br/>　　好像是听到了楚风眠的话，在脑海中，星玄不屑的声音突然响起。
<br/>　　“若是我能够恢复全力，一个手指头，就可以捏死这老头。”
<br/>　　这星玄的话，狂傲之际，但是楚风眠听着，其中的语气中，蕴含这是极端的自信，是从心底之中，都没有将这老者，放在眼中。
<br/>　　“一个手指头捏死这器灵老者？”
<br/>　　楚风眠听到星玄的话，都有些不信。
<br/>　　这器灵老者的实力到底有多强大，楚风眠不清楚，不过绝对是比起大部分超越生死的老古董，都要强大。
<br/>　　以现在楚风眠的灵识，都无法看穿这老者的真正实力，足以看得出来他的强大。
<br/>　　可在星玄看来，这老者却还是极为弱小，甚至都不被他放在眼中。
<br/>　　“怎么，不信吗？哼，若是让我恢复了全力，这小小九域，又算得了什么。”
<br/>　　星玄冷傲的开口道。
<br/>　　“既然你如此强大，那你怎么会沦落到了今日地步？”
<br/>　　楚风眠听到星玄的话，不由的开口道。
<br/>　　听到楚风眠这句话，星玄似乎是一时间没有了脾气，稍微沉默了一会，才开口道。
<br/>　　“我所经历过的，是你无法想象的，等你真正站在九域巅峰的时候，也许就真的有资格，知道当年的事了。”
<br/>　　说完，星玄便是任由楚风眠如何询问，都不在开口了。
<br/>　　“九域之外，另有乾坤。”
<br/>　　楚风眠如今大约也猜得出来，星玄的来历，绝不是在九域之中，而是在九域之外，在无尽域外星空之中。
<br/>　　域外星空，还有着其他武者，其他势力，这一切都是远古一来的传闻，还没有人真正证明过。
<br/>　　可是星玄曾经教给楚风眠的战龙决，可绝不可能是九域功法，整个九域任何宗门，任何强者，都无法创造出如此妖孽的功法。
<br/>　　以现在楚风眠的实力，居然就连其中第一层都没有练就完，足以看得出战龙决的妖孽。
<br/>　　这种功法的来历，只有可能是一点，便是来自于九域之外。
<br/>　　而且星玄剑这个名字，也在九域历史之中，从未见到过。
<br/>　　“战龙决，星玄剑，不知道到底是什么人物，才能创造出这种存在。”
<br/>　　楚风眠低声感叹了一声，便是同时抓着寒月漓的手，一同步入到了隧道之中。
<br/>　　在这器灵老者踏入隧道的时刻，一道声音，便是传遍整个云羽天宫。
<br/>　　“云羽天宫传承，已经开启，云羽天宫即将关闭，所有人尽快离去！不离去者，只有死！”
<br/>　　这一声威严的声音，顿时响彻在整个云羽天宫之中。
<br/>　　“云羽天宫的传承，已经开启了？”
<br/>　　“是谁？这么快就得到了五万枚羽化晶？”
<br/>　　“是楚风眠？还是荒圣君？”
<br/>　　“不知道，不过这云羽天宫真的要关闭了，我们快点离开，不然被困在其中就完了！”
<br/>　　一些武者听到这声音，抬起头纷纷交谈着，不过一些武者已经是注意到了。
<br/>　　这云羽天宫，已经是开始关闭。
<br/>　　在不离开，就会被彻底困在其中了。
<br/>　　一处宫殿前，荒圣君正站在原地，在他的面前，密密麻麻摆放着无数羽化晶，如同一座山峰一样。
<br/>　　“四万九千九百九十七枚。”
<br/>　　当数完最后一枚羽化晶的一刻，荒圣君的脸色，阴沉到了极点。
<br/>　　而就在这个时候，器灵老者的声音，突然从天响起。
<br/>　　“碰！”
<br/>　　荒圣君咬牙切齿，猛然一拍，在他面前的羽化晶，纷纷化为了粉末。
<br/>　　“该死！该死！该死的楚风眠！我一定要杀了你！”
<br/>　　荒圣君疯狂的咆哮道，咬牙切齿，他的眼神中拥有的，只有无尽的愤怒，跟怒火。
<br/>　　“师兄，我们快点离开吧，这云羽天宫的大门已经关闭了。”
<br/>　　一名荒古门弟子，急忙走了过来，也顾不得荒圣君的怒火，急忙开口道。
<br/>　　“哼！楚风眠，我一定要杀了你！”
<br/>　　荒圣君眼神中，蕴含无尽杀意，心有不甘了看了一眼这云羽天宫，身躯一动，离开了这云羽天宫。
<br/>　　在场的武者，纷纷离开这云羽天宫，终于在五分钟之后，这云羽天宫的大门，猛然关闭，再也没有任何武者，可以在进入其中。
<br/>　　另一边。
<br/>　　楚风眠跟寒月漓，踏入到了这空间隧道的背后，却是一处极为庞大的宫殿，这一处庞大的宫殿，位于九天之巅。
<br/>　　比起那之前的一百座宫殿，还要高的多。
<br/>　　在这宫殿大门之上，有着四个大字，云羽神殿。
<br/>　　这四个大字，似乎是曾经被一位绝代强者，刻画在上面的，经历无数岁月的流逝，这四个字依然清晰，其中的笔锋，都是一眼可见。
<br/>　　楚风眠这才注意到，刻有这四座打字的牌匾，居然是以极品天玄晶打造的。
<br/>　　极品天玄晶，号称是时间最为坚硬的材料，就算是在强大的武者，都无法撼动其一分一毫。
<br/>　　可是这四个大字，却是刻画在其上，这刻写云羽神殿四个字的人，他的实力，绝对是远远超出楚风眠的想象。
<br/>　　“这里，就是我云羽天宫的禁地，云羽神殿。”
<br/>　　器灵老者，看着这云羽神殿，眼神中也是露出了几分怀念道。
<br/>　　“七万年没有人踏足过这里了，不错，这一次也算是为我云羽天宫选了一个好苗子，你们两个，进来吧。”
