　　“一元复始，万象更新，洪荒之内，吾为玉清元始！”
<br/>　　一股滔天威压透体而出，迎向苍穹，像是要与天比高，与地比宽。只是瞬间，威压尽皆散去。元始一步踏出，再次回到昆仑山谷，三清所在之地，原地只留下一片破败。
<br/>　　看了一眼太清老子与上清通天，元始发现他们经过元始这些年灵气风暴的滋养，也已经开智慧了，要不了多久，就可以交流了。
<br/>　　不再管他们，元始开始闭关巩固修为，他还要创出今后的修炼功法，不能一直都如此直接吸收天地灵气，也不知转化。
<br/>　　……
<br/>　　百年时间匆匆而过，元始终于将自己刚刚突破的修为巩固了，同时，对于雷霆一道的领悟，经过此次渡劫，上升了许多。
<br/>　　雷霆，毁灭的象征，但毁灭之中又蕴含着浓浓的生机。雷霆之道，刚强狂暴，用来对敌再好不过了，特别是对付阴魂之类的东西，那是属性上的完全克制。
<br/>　　如今修为已巩固，元始也考虑自己今后的打算了。首先想到的，自然是他今后修炼的功法了。功法作为一个人道的体现，十分重要，一部好的功法，发挥到作用也无限，对于一个修道者来说，一步完全适合自己的功法，能够使自己的战力发挥到百分之两百。反之，一部与自己完全不契合的功法，会影响到自己实力的发挥。
<br/>　　各人有各人不同的道，功法也尽不相同。但总的来说，功法按性质可分为三个种类。一为修道功法，二为炼体功法，三位元神功法。三者作用各不相同。从名字上就可以看出，修道功法为修道者修炼法力道行的功法，为道的体现，炼体功法主要是修炼道体的功法，但洪荒之中，有些人却不甚在意自己的道体。元神功法自然是修炼元神的功法，元神功法却十分难得。
<br/>　　“话说，我如今已有元神功法《神元九转》，这已经足够我修炼到混元大罗金仙了，倒是修道功法及炼体功法需仔细打算？我如今这具身体虽比大多数人的要强大，但比起混沌魔神真身来说，还是差太多了。好歹我以前的肉身也快要达到半步不朽，而如今这具身体却连混元都达不到，实在是太弱了。”元始如是想到。
<br/>　　对于元始来说，强大的肉身修为是必不可少的，想想盘古，靠着不朽境的肉身，就可以碾压众多魔神。他自己的肉身也很强大，而如今的肉身却十分孱弱，他自然是有些不习惯。他也不会像现在洪荒众人，为了追求大道，只修练元神，反而不注重肉身修炼。
<br/>　　对于元始来说，人体一阴一阳，若元神为阴，则肉身为阳。世间如一个苦海，人在海中，修行就是为了度尽苦海，登临彼岸，再行超脱。
<br/>　　人在海中，肉身可为船只，元神为船里的人，船载着人，一直驶向彼岸。没有坚固的船只，又怎能达到彼岸？
<br/>　　对于元始来说，肉身与元神修为同样重要，所以，他想要创的功法，就要炼体功法与修道功法同创。
<br/>　　“到底要创怎样的道体功法呢？”元始思索道，这关系到自己以后的道路，由不得自己不费心。
<br/>　　“要说道体，单一属性灵体不再考虑之中。道体之中，就属盘古的混沌真体最为强大，与混沌真体同级别的有玄黄不灭体。在这两者之下的有阴阳圣体，五行圣体、苍天霸体等，不过，那些不再我的考虑之中。
<br/>　　我元始是什么人，一生不弱于人，不做就不做，要做就做到最好。故而，唯有混沌真体与玄黄不灭体在我考虑之中。”
<br/>　　“混沌真体，要在混沌之中才能很好的修炼，虽然威力巨大，但，如今我身处洪荒之中，混沌之气基本没有，到哪儿修炼去，难道要我再回混沌？不，好不容易在洪荒弄到一个身份，再回混沌那不是我的真意。再说，在洪荒修炼混沌真身，天道也不会允许的。
<br/>　　那么，我的选择也就只剩下一个了，那就是与混沌真体同级别的玄黄不灭体！”
<br/>　　“玄黄不灭体想要修炼大成，就必须要玄黄之根。玄黄之根，为玄黄母气，乃天地初开时所产生的万物母气。玄为天精，黄为地髓，玄黄是天地的精髓。
<br/>　　无名，天地之始，是为道。有名，万物之母，是为天地。玄黄是天地之精，是万物的母气，是锤炼一切有形之质的圣物。玄黄二气可是与混沌之气齐名的存在。玄黄之根作为玄黄的母气，更加珍贵，几乎消失殆尽。”
<br/>　　“不过，要是别人对此还真可能没有办法，但对于我来说，也不是没有办法。至少我是知道，太清老子的天地玄黄玲珑宝塔就是玄黄之根、玄黄之气与天地功德所结而化的。
<br/>　　还有，我在开天之初抢到的鸿蒙量天尺中，至少也有一道玄黄之根。再说，玄黄不灭体想要大成，才需要玄黄之根，之前只需要玄黄之气与其他就行了。此等事情，等真正到了那时再说。”
<br/>　　“至于修炼道法的功法？那就有些麻烦了。我之道，为元始之道。囊括万道，融进万物。想要创出此等功法，还真是不易啊。啊，不管了，先闭关再说。”
<br/>　　想好方向之后，元始再次闭关，开始了创法之路。
<br/>　　……
<br/>　　洪荒之中，时间最是不值钱，又是数万年时间匆匆而过，经过多年闭关，元始再次出关，他想要创的功法，也终于创出来了。
<br/>　　“哎，时间过得真快，眨眼间又是数万个春秋。不过，此番闭关，我的功法还真让我给创出来了。”元始心情不错，因为他闭关的目的达到了。
<br/>　　不错，经过数万年的时间，元始终于将自己今后所要修炼的功法创造出来。元始创功法的同时，也将自己的大道梳理了一遍，使道行更深一步。
<br/>　　元始此次闭关，共创出两步功法，一部为炼体功法，一部为修道功法。
<br/>　　炼体功法名为《玄黄不灭》，功法共分九重，一重仙元炼玉身，二重五行开道宫，三重四象化四极，四重龙脉融龙骨，五重星辰入神窍，六重阴阳筑神魂，七重虚空成虚境，八重混沌衍洞天，九重玄黄成不灭。
