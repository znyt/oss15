　　第645章
<br/>　　程大雷有些无语，一个正常的家族会安排什么密语么？你说你华家是个安分家族有人信么？
<br/>　　华正则房中就有笔墨纸砚，程大雷亲自为他磨墨，华正则想了想，在纸上奋笔疾书。程大雷在旁边瞧着，想要揣摩一下华家的密语，结果什么都没看出来。
<br/>　　一个传承数百年的家族，总是有些旁人看不穿的东西，程大雷现在隐隐觉得，在这次灾祸中，华家怕是没多少损伤。
<br/>　　写罢之后，华正则又看了一遍，确认无误后用信札封了，双手递给程大雷。
<br/>　　“程当家，劳驾了。”
<br/>　　程大雷拍拍他的肩膀，道：“华大夫放心，此事包在我身上，必定尽我之力。”
<br/>　　华正则也并非铁石心肠，相反他还是个很感性的人。程大雷亲自为他铺纸磨墨，足见程大雷的热心。
<br/>　　程大雷拿着信，想着交给谁去办这件事。他倒是很想再走一趟江湖，可朔方城刚刚有些起色，他实在脱不开身。而关鱼、赵子龙等人虽然有帅才，但并没有多少闯荡江湖的经验。
<br/>　　正思索间，刘发财从朔方城回来了，刚回来便找到程大雷。
<br/>　　“大当家，华大夫没事吧？”
<br/>　　“你怎么回来了，我不是嘱托你好好在朔方城歇息么？”
<br/>　　“那个……我实在放心不下，便回来看看，只是大当家的坐骑太快，我一路追也没有追上。”
<br/>　　程大雷心里寻思着，这刘发财倒是有闯荡江湖的经验，不过他本身的武艺实在太弱，不足以在江湖上立足。他忽地想起一人，道：“刘发财，我有件事吩咐与你，你可愿意去做？”
<br/>　　“大当家吩咐便是。”
<br/>　　程大雷亮出手中的信，道：“我要你走一趟东海，寻找华氏一族，以此信为证。”
<br/>　　“属下立刻就出发。”
<br/>　　“嗳，欲速则不达，你一来一去已经累得够呛，立刻出发非得死在路上不可。”程大雷道：“你先歇息一夜，明早出发，我再帮你寻一帮手，能保全你的性命。”
<br/>　　刘发财尴尬的点点头，程大雷这话明显是看不起的他武艺，不过刘发财也承认这一点，如果他的武艺真说得过去，也就不需要在江湖上招摇撞骗了。
<br/>　　程大雷差人将黑衣尼姑唤来，这黑衣老尼武艺自不必说，真如果动手，程大雷都不敢说稳胜过她。而她又对程大雷言听计从，程大雷让她往南，她绝对不往北，哪怕撞破南墙也不会回头。
<br/>　　程大雷吩咐一番，黑衣老尼点头答应，过程中竟没有说一句话。刘发财打量自己此行的搭档，心道：这老尼姑行么？
<br/>　　按照程大雷的吩咐，他先休息一夜养足精神，次日天明带了干粮兵器，与黑衣老尼各骑一匹快马，前往东海。
<br/>　　程大雷是脱不开身，不然还真想走这一遭。他现在是寄希望在刘发财身上，希望他能寻到华氏一族，带到蛤蟆城。二来，也是希望刘发财弄明白东海的具体情形。
<br/>　　帝国太大了，信息又落后，再快的快马又能有多快。东海虽然乱了，但传到这里已经是十几天后，现在东海是什么情况，程大雷完全不知道。
<br/>　　说不得，刘顶天已经被人灭了，也或许，他又灭了别人。
<br/>　　……
<br/>　　凉州城，王府。
<br/>　　宋伯康窝在自己书房内，心情有些不痛快，不痛快的原因是刚刚得到的消息。
<br/>　　“那混江龙确实是被程大雷灭了，据说，他从黑风寨拉走了几十船的粮食。混江龙十几年积累的家业，全部被他抢了。”陆洛垂着头道。
<br/>　　“那黑风寨也有些兵马，岂是好对付的，如果好对付我早就动手了。”宋伯康道：“他想要动黑风寨，我怎么一点风声都没得到？”
<br/>　　“属下打探到的消息，那程大雷是带人夜袭，打了黑风寨一个措手不及。黑风寨的山贼死的死逃得逃，至于程大雷，似乎没费什么力气便拿下了黑风寨。”
<br/>　　宋伯康长出一口气，任谁也能听出他胸中的不痛快。可不痛快又能如何，天下诸多事，又有几件是他能够算定的。
<br/>　　这次听了陆洛的计策，用朔方城一座空城分化程大雷的兵力，自己收拾掉百里胜后，再腾出手对付程大雷。
<br/>　　然而，事情的发展总是出乎预料。自己没能消灭百里胜，他竟然带着一波残兵败将突围逃了。而程大雷的实力也并没有被自己分化，他劫了黑风寨也解决了粮食危机。
<br/>　　现在的凉州从之前的三足鼎立变成现在的两家对峙，自己还是不敢向程大雷动手。
<br/>　　“那……程大雷现在在忙什么？”宋伯康问。
<br/>　　“还是在朔方城，目前似乎在修城。”陆洛答道，他现在心情也有些忐忑，自己献的计策似乎也没有起到什么作用，他生怕宋伯康会责罚自己。
<br/>　　宋伯康无可奈何的摇摇头，他却是想要收拾掉程大雷，可实在没有这个实力。也就只有看着程大雷一点点发展，不过，蛤蟆城与朔方城中间隔着金城，自己随时可以切断他们的粮道。而程大雷想要将朔方城发展起来，也并不是一件容易的事情。
<br/>　　如此，也就只有任他去吧。
<br/>　　宋伯康还有一件更关心的事情，他问道：“百里胜到了哪里？”
<br/>　　陆洛沉默片刻，道：“一路向东，往长安城去了，戴白将军传来的消息，现在还没有截住他们？”
<br/>　　宋伯康手敲着桌面：“百里……呐……百里……”
<br/>　　百里胜不仅代表百里胜自己，身后还有一个百里家族，那也是数百年传承的大世家。当然，宋伯康背后的宋家，陆洛背后的陆家，戴白身后的戴……都是世家豪族，他们之间的斗争合作联系也并非一天两天的事情。
<br/>　　宋伯康敢打百里胜，自然是崔相已经解决掉一些朝堂上的事情。然而有些暗地里的事情，只适合在暗地里进行，可这些事情是见不得光的。
<br/>　　如果百里胜真的到了长安城，把桌子下面的事情摆到桌子上面，那就连宋伯康也不知道会发生什么事情了。
<br/>(本章完)
