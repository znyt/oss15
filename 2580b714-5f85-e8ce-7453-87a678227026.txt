　　第467章
<br/>　　弓开若满月，箭走似流星，这一刹那星月交晖，一枚羽箭直冲徐神机背心而去。
<br/>　　徐神机犹若梦中人，浑然不觉背后的杀机，口中高喊：“大当家我回来啦，我终于回来啦。”
<br/>　　程大雷眼睛眯起，心脏砰地一下跳动，察觉到了扑面而来的杀机。
<br/>　　“徐神机，下马！”
<br/>　　两地相距太远，程大雷想救也来不及，一边骑着黑牛冲了出去，一边口中高喊。
<br/>　　徐神机尚不知怎么会事，幸运的是他和程大雷长期相处，二人已有难得的默契。在听到程大雷的声音后，几乎没有任何考虑，直接从马上跳了下来。
<br/>　　几乎就是一刹那，羽箭从他原先身处的位置飞过，而徐神机在地上滚了两下，摔得头破血流。
<br/>　　程大雷此时已到他身边，俯身将他从地上拎了起来，又回到大本营中。
<br/>　　这一下只在电光火石之间，虽然说是有惊无险，却是吓出程大雷一身冷汗。
<br/>　　不久前，刚看着林少羽在自己面前死去，这个坎儿程大雷现在还没过去。今日若在目睹徐神机死在自己眼前，程大雷当真不知该如何是好。
<br/>　　将徐神机丢在地上，回身望着帝国军，双方相距五百步，其实看不真切，但程大雷隐隐觉得，哪里有一双眼睛也正望着之间。
<br/>　　五百步的距离，杀意弥漫。
<br/>　　程大雷本想摸鱼占点便宜，便继续往青牛山逃命，但现在，他已不准备逃了。
<br/>　　今日一定要杀了薛丁山。
<br/>　　若是不杀，日后此子必成心腹大患。
<br/>　　两军对峙，谁都没有着急发动进攻，但却都在悄悄调动。
<br/>　　“布口袋阵，向前推进一百步，原地掘陷马坑。”
<br/>　　薛丁山下令，三千兵丁一起行动，以手中兵器就地掘坑，限制程大雷的推进。
<br/>　　陆承也没有想到，徐神机会临阵反水，枉他对徐神机如此推崇，现在颇有些恼羞成怒的意思。
<br/>　　“薛将，下令进攻吧，莫让贼人逃了。”陆承。
<br/>　　“我有种感觉，他不会走的。”薛丁山重复一遍，更加确定这种感觉：“陆军师，我还有一件事交待你去办。”
<br/>　　“薛将请讲。”
<br/>　　“你带人速速返回大营，领大军回来，多备强弓劲弩，今日一定要在此地击杀此贼。”
<br/>　　两地遥隔，由黑夜阻挡视线，程大雷不知对方发生了什么。他现在正面临全军休整，养精蓄锐后，准备全力攻击。
<br/>　　“军师，你是怎么落到他们手里的？”
<br/>　　“嘿，别提了。”徐神机用手捂着被撞破的喉咙：“那天兵荒马乱的，我被几个不开眼的小兵抓住了。嘿，他们也不想想我是谁，崔相见了我是纳头就败，三日一小宴，五日一大宴……”
<br/>　　“好啦，你别说了。”程大雷摆摆手，打断了徐神机的话，现在他觉得和徐神机聊这个，几乎是浪费时间。
<br/>　　“单兄弟。”程大雷远眺敌阵。
<br/>　　“末将在……”
<br/>　　“我们不宜在此地久留，要冲杀一阵，你带部队佯攻敌方右翼，遇到抵抗便转主阵，这次，我们直接要薛丁山的脑袋。”
<br/>　　“是！”
<br/>　　一阵山呼海啸，三百义字军发起冲锋，单雄信挥舞着大槊领阵，程大雷掩在其后。
<br/>　　“全体准备！”薛丁山立刻开始调兵遣将：“骑兵在前，弓兵在后，步兵保护弓兵，与我杀。”
<br/>　　两军对撞，先是骑兵的厮杀，之后单雄信带人扑进敌人右翼。飞箭从敌阵中射出，众人用武器挥砍，与敌军接触。中途突然变相，扑向了薛丁山。
<br/>　　程大雷鬼鬼祟祟，这次没有背斧，拎着匹夫剑去刺杀薛丁山。
<br/>　　万军从中取上将首级，程大雷要干的就是这件事。出于对危险的警觉，程大雷第一时间就察觉到这薛丁山不是一个易于之辈，现在他还是顶级，如果等他升到绝世，自己还能制得住他。趁着现在本事还强过他，直接要了他性命，一了百了。
<br/>　　噗通噗通！
<br/>　　本来奔驰的骑兵队伍，突然被崎岖不平的陷马坑绊倒，后队收阵不急，撞在前队身上。
<br/>　　薛丁山一挥手中银戟，领着手下冲锋。
<br/>　　“与我杀，杀贼有赏！”
<br/>　　程大雷大惊失色，却是没料到薛丁山在短暂时间便下令掘出陷马坑，这等执行力，当真是令人不寒而栗。
<br/>　　义字军一开始损兵折将，他们的确配得上『勇』这个字，即便掉落马下，也没有放弃进攻，挥舞着武器厮杀，没了武器的，就用拳头砸，用牙齿咬。
<br/>　　薛丁山此刻已与单雄信撞在一起，你来我往，倒是不分胜负。
<br/>　　程大雷一剑逼退薛丁山，继而进剑，要割他的喉咙。帝国军的兵丁围上来，程大雷一剑点七灯，送他们上了西天。
<br/>　　但身边帝国兵越围越多，程大雷倒是无虑，但义字军却有全军覆没的危险。
<br/>　　“走啦！”
<br/>　　程大雷只能选择撤退，领着义字军在敌阵中冲杀一阵，突出包围圈。
<br/>　　退出五百步外，程大雷清点手下的伤亡，三百义字军，这次冲杀死了百十人，伤亡超过三分之一。
<br/>　　自从出世以来，程大雷还是第一次遭遇这样的溃败。固然有义字军骑术不精，周转不开的原因，但自己的莽撞大意却是主要原因。
<br/>　　的确是大意小看了薛丁山，这一次交锋，算是程大雷落败。
<br/>　　“大当家……”徐神机。
<br/>　　“走！”程大雷收剑还鞘：“先离开此地。”
<br/>　　程大雷纵有以一当千的武勇，手下士兵却不能以一当十。目前兵力完全不占优势，只能选择撤退。
<br/>　　今日杀不了，只能日后再杀了。
<br/>　　撤退前，程大雷恶狠狠的回头看了一眼。
<br/>　　薛丁山此刻也十分懊恼，自己的安排没有任何问题，以做到能力范围内的最好。问题是这三千帝国军太过不中用。
<br/>　　如果他们行动更快些，迅速布好陷马坑，杀敌时再勇敢些，就绝不会让程大雷离开此地。
<br/>　　当然，这也是没有办法的事，帝国军失去勇气，已经有几十年了。薛丁山心里思量着，若自己日后本事再大些，亲手打造一支精兵，必能无往而不胜。
<br/>　　当然，这是以后的事情了，现在要做的是……
<br/>　　“追！”
<br/>(本章完)
