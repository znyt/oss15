　　今日，乃是检验萧逸能否胜任少家主之位的重要日子，所有族人早早便已经等候在家族比武场。
<br/>　　以往，历代萧家的家主，受九大长老辅助，所有族人拥护，乃是紫云城最顶尖的人物，更是称霸一方的武者。
<br/>　　家主的重要性，可想而知。
<br/>　　少家主作为日后的接班人，其重要性不言而喻。
<br/>　　长老席上，头一次出现了萧逸的身影，坐于三长老之下。
<br/>　　“萧逸，请上比武台。”一位后天执事忽然高声道。
<br/>　　这是上次那位裁判。
<br/>　　萧逸脚步一动，跳上了比武台。
<br/>　　“萧逸上场了。”所有人眼前一亮，目光中带着希冀与期待。
<br/>　　他们都很想看看，家主唯一的儿子，到底能否胜任少家主之位。
<br/>　　他们更想看看，萧逸是否能再度大放异彩，如同半个月前大败家族的其他出色子弟。
<br/>　　萧逸上台后，看了眼长老席上的长老们。让他惊讶的是，今天竟然九大长老全来了，包括一直极少露面的二长老和大长老。
<br/>　　“萧逸，你是否愿意接受任何子弟的挑战？”
<br/>　　裁判的声音，将萧逸的目光拉回到比武台上。
<br/>　　萧逸淡淡地点了点头，神色自若，风度翩翩，浑身散发着傲然与自信的气息，格外不凡。
<br/>　　“好。”裁判再次高声问道，“有哪位子弟要上台挑战？”
<br/>　　如今还在家族中的子弟，就只有那些年幼的小屁孩，还有十多天前进入紫云洞修炼的子弟，其他子弟，早就被派出去历练了。
<br/>　　那些小屁孩自不必多说，他们只是单纯来看热闹的。
<br/>　　所以裁判的目光仅仅在凡境六重的子弟身上扫视着。
<br/>　　平日里高傲的子弟们，纷纷目光躲闪，没有一人应答。
<br/>　　从半月前家族比武上，萧子木轻松败于萧逸之手时，他们就知道，自己绝不可能是萧逸的对手。
<br/>　　“我来。”
<br/>　　忽然，一道清冷的声音响起。
<br/>　　声音从长老席上传来，正是五长老下方的萧若寒。
<br/>　　萧若寒也是脚步一动，跳上了比武台。
<br/>　　其实所有族人都知道，今天，根本就是萧逸与萧若寒的一战。
<br/>　　当然了，他们更看好萧若寒。毕竟，萧若寒从小到大都是萧家最耀眼的天才，而且还是黄阶武魂拥有者，日后最低的成就都是成为先天境武者。
<br/>　　萧家年轻一代第一人，非他莫属。
<br/>　　既然有人上台挑战，代表着比武即将开始，裁判嘱咐道，“仍旧是那句话，分出高下即可，不可伤人性命。”
<br/>　　萧逸和萧若寒同时点了点头。
<br/>　　“比武，开始！”裁判说罢，退到了一边。
<br/>　　偌大的比武台，成为了萧逸与萧若寒的对战之地。
<br/>　　“萧逸，你先出手吧。否则，我一出手，你将连一秒的时间都支撑不住。”萧若寒故作大度地说道。
<br/>　　在其它族人眼里，萧若寒是个修炼天才，更是个温文儒雅，大度成熟的子弟。
<br/>　　萧逸淡淡一笑，道，“既然是你挑战我，便由你先出手吧，免得说我不给你出手的机会。”
<br/>　　战斗尚未开始，空气中已经弥漫着一股硝烟。两人狂妄的话语，无不表示出自身对对方的轻蔑。
<br/>　　最终还是萧若寒急于对付萧逸，率先出手。
<br/>　　“萧逸，既然你自己狂妄自大，那就别怪我手下不留情了。对付你，我连使用武魂都不需要。”
<br/>　　萧若寒大喝一声，瞬间朝萧逸攻去。
<br/>　　萧若寒快速接近萧逸，并一掌打出。
<br/>　　“哼，狂妄自大的是你！”萧逸冷哼一声，同样一掌打出。
<br/>　　长老们个个都目光严肃，紧紧盯着战斗。
<br/>　　而其他族人们，则早已沸腾。
<br/>　　“萧若寒率先出手了，听说他从紫云洞出来时，已是凡境九重，论实力，比一般的成年族人还要强。”
<br/>　　“萧逸在做什么，不躲不闪的，他打算硬扛这一掌吗？”
<br/>　　“萧逸会被这一掌轰下比武台吧？”
<br/>　　族人们看向萧若寒的目光是惊讶以及满意，看向萧逸的目光则是疑惑与担心。
<br/>　　“轰”
<br/>　　当萧逸与萧若寒两掌对轰时，气氛却骤然停格，唯有比武台上一声大响。
<br/>　　萧逸正面接住了萧若寒的一掌，且纹丝不动，反倒是萧若寒被震退了几步。
<br/>　　“怎么可能？”萧若寒眼中尽是惊讶。
<br/>　　“没什么不可能的。”萧逸冷声说着，随即身上一阵气势爆发。
<br/>　　“萧若寒，别以为只有你是凡境九重。”
<br/>　　萧逸大喝一声，随即掌化拳头，一拳将萧若寒震退十数步。
<br/>　　“凡境九重？”长老席上，三长老吃了一惊。
<br/>　　在他旁边，还有一位老人，目光中透漏着浓浓的讶异之色，竟是二长老。
<br/>　　“这小子，我明明记得他进紫云洞前不过是凡境五重，如今竟已是凡境九重，岂不是说他短短十天内连跳了四重？”二长老看向萧逸的目光顿时变了，眼中尽是火热。
<br/>　　周围的族人们也同样想到了这个问题，看向萧逸的目光已经不能用惊讶去形容，比震惊还震惊。
<br/>　　“我没记错的话，家族比武时，萧逸才凡境五重吧？”
<br/>　　“对，没错。”
<br/>　　“短短十天，连跳四重，天啊，萧逸什么时候变得那么厉害了。”
<br/>　　“这样算起来，他的天赋岂不是比萧若寒还要强得多？”
<br/>　　“十天突破四重，萧家还从未有人有过这个成就吧？”
<br/>　　族人们脸上的惊讶之色溢于言表，甚至有些不敢相信眼中所见。
<br/>　　听着族人们的赞美和惊讶，五长老与萧若寒的脸色完全黑了起来。
<br/>　　“小废物竟然突破到凡境九重了？”五长老暗恨一声，“该死，早知道之前就该不惜一切代价将他抹杀掉。”
<br/>　　比武台上，萧若寒也面露杀意，“凡境九重又如何？凭我的黄阶武魂，足以碾压你。”
<br/>　　想罢，萧若寒再次发动了攻击，一条火云蟒凭空出现，带着铺天盖地的火焰，气势十足。
<br/>　　“火云蟒，去。”萧若寒大喝一声。
<br/>　　庞大而浑身被火焰覆盖的大蟒蛇搅动而出，那狰狞的火舌意图将萧逸卷住并吞噬。
<br/>　　“黄阶武魂，果然不凡。”萧逸眼中微眯，表情变得严肃。
<br/>　　使用武魂力量的萧若寒，战力比刚才强了数倍都不止。
<br/>　　当然，萧逸仍旧信心十足。
<br/>　　“哼，米粒之珠，也敢放光华？”萧逸冷哼一声，双手化爪。
<br/>　　“吼”，一阵震天虎啸响起，萧逸的身影变了，浑身冒火，如同一只火焰猛虎。
<br/>　　“出现了，萧逸那些古怪的招式又出现了？”族人们惊呼一声。
<br/>　　在他们眼中，场上的战斗，变为了猛虎与火蛇间的较量，更为激烈，也更为凶险。
<br/>　　到底是火蛇绞杀猛虎，抑或猛虎将火蛇咬断并踩于脚下，没人能妄下定论。
<br/>(本章完)
