　　叶少阳没跟他打过，但知道这货不好对付，也只有硬着头皮上了。
<br/>　　“叶天师，贫道早想一战！”
<br/>　　秋明子战意很浓，判官笔凌空横扫，卷起一股罡气，朝叶少阳迎面打过来，叶少阳以剑抵挡，两人各退半步，秋明子心中一惊，道：“你而今是上仙了？”
<br/>　　叶少阳比他更吃惊，这就一交手，立刻试探出自己的境界，这家伙也是厉害啊。没时间跟他闲扯，上前强攻。
<br/>　　秋明子知道他急着去动庆云金灯，也知道靠自己擒不住叶少阳，干脆招招防守，不让他有机会冲过去。
<br/>　　叶少阳暗骂他老奸巨猾，出手更加狠辣，但始终无法从他身边闯过去，转头看自己小伙伴们，也都被架住了，慈心师太等人虽然菜一点，但退守到门口，几个人贴在一起，形成防线，就像打篮球里的联防一样，集体对抗，一时间还能撑得住。
<br/>　　在他们身后，通玄道人还在被点着“天灯”，但火焰已经越来越小了，庆云金灯里的火苗却是越来越明亮，缓缓升起，将整个庙堂照得通亮。
<br/>　　时间肯定不多了。
<br/>　　叶少阳急在心里，面色一沉，冲秋明子冷冷说道：“前辈，我跟你无仇，但我今天对这庆云金灯志在必得，你若再阻拦，我便不客气了。”
<br/>　　这虽然是一般人打架时候常说的话，叶少阳此刻说出，表达的却是两个字：拼命！
<br/>　　秋明子当然也明白他的意思，嘿嘿一笑：“叶少阳有是本事只管使出来，老道奉陪便是，但若想从我这里闯过去，却是万万不可。”
<br/>　　叶少阳摸出七张暗金神符，一股脑贴在七星龙泉剑上，再度朝秋明子扑了上去，秋明子也不敢大意，当下摸出一只铜铃，一边躲避叶少阳额攻势，一边摇动铜铃。
<br/>　　他的动作跟个调酒师似的，将铜铃摇得极有节奏，他面前的土地上，平地升起了一股旋风，将尘土都卷起来，形成了一个人的形状。
<br/>　　傀儡术！
<br/>　　这是众阁派的看家本领，叶少阳看到这场面，也知道秋明子是动真格的了，无奈有傀儡拦着，秋明子一个劲往后躲，轻轻摇铃，那尘土组成的傀儡便往死了进攻，而且力道极大，叶少阳跟它硬拼了几下，发现自己几乎拼不过，无奈只好从七星龙泉剑上撕下一张焚天符，又斗了几个回合，瞅了个机会贴在了傀儡后背上，念动咒语：“太上三清，四方大帝，神威通天，焚天灭地，急急如律令！”
<br/>　　一大团火焰，从傀儡身上腾起，整个人迅速成了一团火球，左右摇晃着，跟人着火之后的反应差不多。
<br/>　　叶少阳趁机朝秋明子冲过去。
<br/>　　众阁派最厉害的就是傀儡术，秋明子虽然自信修为上不输给叶少阳，但硬打的话手段上吃亏，于是且战且退，跟捉迷藏似的，绕着傀儡旋转起来，叶少阳一时半会也抓不住他，心中恼怒，这种不太要脸的路子，一想只有自己好意思干，没想到居然有一天被人抢了先，当下破口大骂起来。
<br/>　　“追他做什么，去抢庆云金灯！”林三生也入场了，但跟美华他们一样被架住了。
<br/>　　对啊！叶少阳一拍脑袋，自己目的是庆云金灯，秋明子怂了，不管他就是，追他干啥，当下放下秋明子，朝庙堂奔过去。
<br/>　　突然，一股异动从脑后传来，伴随着林三生的一句“小心”，叶少阳本能地一个错步，闪出几米之外，回身望去，顿时惊得目瞪口袋：
<br/>　　那个傀儡身上的火灭了，居然又冲上来，叶少阳只好应敌，几个回合下来，心中大惊：这傀儡非但没事，力道好像比之前更大了，而且胳膊腿都更加结识，就像铜墙铁壁似的，法术攻击打在他身上，一点反应没有。
<br/>　　这是啥子情况？中了自己焚天灭地符，非但没事，反而更厉害了？天下哪有这种事情？
<br/>　　“哈哈……”秋明子继续摇铃，口中发出老母鸡一般的奸笑。
<br/>　　他这笑声反而给了叶少阳提示，猛然想到什么，五行，特么的，五行属性！
<br/>　　这个劳什子傀儡是尘土所生，五行属土，自己用火去烤它……结果把人烤成瓷器了，更加厉害了。
<br/>　　五行属土，木克土，那就用木系法术来对付他？
<br/>　　叶少阳搜肠刮肚，寻找自己掌握的木系法术，一扫眼看到包子，大叫了他一声，包子正跟瓜瓜一起对付慈心师太等人，立刻跑到叶少阳身边，叶少阳一边抵挡傀儡进攻，一边简单把情况说了，包子一听，道：“这个好办！”
<br/>　　当下现出真身通灵佛子花，趁着叶少阳抵挡傀儡，枝叶加速生长，爬到傀儡升上去，如同触手一般，很快爬满了他全身，傀儡行动受限，不管秋明子怎么摇铃，动作上慢了许多，但全身关节都被枝叶困住，也无可奈何。
<br/>　　“美华大姐帮我！”
<br/>　　包子大声叫来美华，美华聪明，看了一眼就知道怎么回事了，从旗袍的兜里摸出一只海螺，轻轻一吹，有碧绿的水像喷泉似的冲射在了傀儡身上。
<br/>　　“好！”
<br/>　　包子用力操纵那些困住傀儡的枝叶，用力扭动了一下，然后快速地收了回去，傀儡还在傻站着，浑身湿漉漉的，几秒钟后，咔嚓一声，脑袋掉了，接着整个身子都塌了下去，化成一堆碎泥。
<br/>　　他身上的灵气，被包子之前吸干，又被水这么一冲，本就是土质的东西，不跨才怪。
<br/>　　在这泥人傀儡倒下的瞬间，秋明子握着铜铃的那只胳膊也颤抖了一下，叶少阳早就看见，不等他再次作法，已经手持七星龙泉剑冲了过去。
<br/>　　秋明子举起左手，结印想要硬拼，叶少阳却凌空打出一张灵符，在他面前爆开，等火星散去，叶少阳已经冲到了慈心师太一干人面前。
<br/>　　慈心师太，逍遥飞，还有青城山、武夷山的两个老大，叶少阳见过也勉强认识，还有几个面熟叫不上名字，但知道都是一些大门派的首脑，看到叶少阳靠近，一起都要腾出手来对付。
