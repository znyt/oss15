　　著名的戏剧《梁山伯与祝英台》，梁山伯要死的时候，他娘去求医，得到的结果是想要救梁山伯，非得用蛟龙头上的角，蠓虫头上的浆。虽然是用来形容无药可医，但这里说的蠓虫，其实就是尸蠓。
<br/>　　这种小而致命的东西，是某种特殊情况下，在尸体内长成，然后潜伏着，当时还是第一形态，一旦暴露在空气里，立刻就会破蛹而出，成为会飞的尸蠓，比蚊子的嗅觉还要好的多，见人就叮……
<br/>　　“尸蠓，卧槽，过来了！”叶少阳刚回过神来，喊了这一嗓子，就看到一群黑压压的尸蠓飞了过来，大伙一看，也都吓的不行，纷纷往后躲。毕竟这玩意不是僵尸鬼怪之类的，用法术可以对付，这玩意虽然单个一只不算什么，但是成群结队的，令人防不胜防，就算法力再深厚，在没有准备的情况下也不能把这些蚊虫怎么办。
<br/>　　“用火烧！符火！”毛小方突然叫道。
<br/>　　叶少阳突然回过神来，他虽然也是第一次对付尸蠓，不过蚊虫之类的都怕火，这些是尸蠓，体内有尸气，用符火肯定能克制，当下立刻打出了几道地火符，凌空飞过去。
<br/>　　这些蚊虫可比人跑得快，灵符打出去的时候，第一波已经赶到，好在符火也燃烧起来，呼啦一下，将一大堆尸蜢烧成了焦灰。但总有漏网之鱼，从别的方向冲过来。
<br/>　　好在这时候他们已经进入墓道，墓道两三米宽，只有一个出口，这些蚊虫不能从四面八方围剿，只能正面进攻。
<br/>　　有了叶少阳开头，身后几人也都各自祭出符火，不过灵符燃烧得很快，需要不断作法，不然难以为继。
<br/>　　这时候梧桐站出来，拿出五宝金莲，吹出金莲业火，不断焚烧着蚊虫。
<br/>　　“还是你这个爽啊！”
<br/>　　这些尸蠓没什么智商，一股股地飞来送死，飞蛾扑火，叶少阳看着它们一批批地惨死，心中十分地痛快，冲梧桐笑道：“你这五宝金莲是好用，不过用它们来杀尸蠓，就好像高射炮打蚊子，大材小用了。”
<br/>　　“什么炮？”梧桐皱起眉头。
<br/>　　叶少阳这才想起自己一不留神又说了自己那个时代的东西，只好呃呃了两声代过，突然脖子上一疼，用力拍了一下，对着火光一看，是一只尸蜢的尸体。
<br/>　　“不好，有飞过来的，你们注意点别被叮了！”叶少阳一边说一边脱下了外衣，在梧桐身上四下拍打起来，避免尸蠓落在她身上。
<br/>　　“照顾好你自己就行了！”梧桐说道。
<br/>　　“我不怕，我没事。”
<br/>　　这样坚持了有一会，尸蠓总算是杀干净了。梧桐又坚持了一会，自己法力也消耗了不少，这才放下了手里的五宝金莲，大口喘着气。
<br/>　　叶少阳也松了一口气，回头说道：“你们都还好吧？”
<br/>　　顿时间怔住了。
<br/>　　身后妙心、毛小方等人，七歪八扭地倒了一地，连陈晓宇的那只猴王都倒了。
<br/>　　这是……尸蠓咬的？
<br/>　　叶少阳站在最前面，之前一直光顾着注意梧桐和面前的尸蠓，根本没注意身后，这一看顿时震惊了。
<br/>　　先冲上去查看毛小方，伸手一摸，脖子后面有个圆鼓鼓的东西，把他拉到发光的石头前面看去，这才发现后脖颈上趴着几个圆鼓鼓的尸蜢，这些小虫子吸了人血之后，身体立刻大了十几倍，圆鼓鼓的，脑袋都已经扎进了皮肤里面，看上去十分恶心。
<br/>　　果然，是不小心中了这些尸蠓的招。
<br/>　　叶少阳伸手要拉尸蜢，被梧桐挡住，说道：“你这样一拉，脑袋容易留在皮肤里，伤口很难长好的，有针吗，挑出来。”
<br/>　　“有有。”叶少阳一拍脑门，从包里拿出十八神针，为毛小方清理了脖子上的几个尸蠓，挑出来之后捏死了。
<br/>　　这个时代的人穿的都多，长袖长裤的，基本上暴露在外头的只有头脸和双手，双手一直在活动，不可能被叮咬，叶少阳检查了脖子和头脸，确定没了，然后再为别人清理。
<br/>　　这些人的修为，至少都是准天师，对于吸入的或者品尝到的尸毒之类的，都比一般人有更强的抵抗，但是尸蠓叮人，是把尸毒直接注射在血液里，这个就比较坑了，中了尸毒之后，就算是地仙也得暂时昏迷。
<br/>　　不过他们体内有强大的罡气，会抵抗尸毒的入侵，逐渐溶解，人会没事，这种昏迷也只是暂时的，就好像很多人休克一样，不管他自己都会恢复。
<br/>　　因此，叶少阳在检查了他们的身体之后，都比较放心。
<br/>　　“没想到他们都中招了，我来化一碗糯米水给他们喝。”叶少阳说着就打开背包，取出瓷碗，开始操作。糯米是僵尸最大的克星，糯米水自然能解尸毒，让他们快一点醒来。
<br/>　　梧桐也不紧张，蹲在一边看着他，突然开口说道：“为什么你没事，你没被叮？”
<br/>　　叶少阳被她这么一提醒，一抹头脸，发现耳朵上多了一个“耳垂”，自然是自己不小心让尸蠓给叮了，赶紧让梧桐帮忙给自己用针挑出来。
<br/>　　梧桐这下更好奇了，再度提出了刚才的问题，“你明明被叮了，为什么没事？”
<br/>　　“这……”叶少阳想找一个借口，梧桐却盯着他说道：“我希望你能说实话，至少在我面前。”
<br/>　　“好吧，其实，我是先天灵体……”
<br/>　　梧桐看着他，沉默了几秒钟，失声叫起来：“先天灵体！！”
<br/>　　一向冷静克制的她，内心也是按捺不住地震动起来。
<br/>　　先天灵体，在法术界只是一个传说……传闻几百年才会出一个这样的人。梧桐不信，但是让他如此震惊的，还有另一个原因，难以启齿……
<br/>　　梧桐看着他，突然摇了摇头，“不可能，你怎么会是先天灵体！”
<br/>　　叶少阳摊了摊手，“我没有骗你，你让我说实话的，我是先天灵体，百邪不侵，别说是几只尸蠓，就算被尸王咬一口也没什么事。”
<br/>　　梧桐凝视他，喃喃道：“可是，你是先天灵体，为什么只有天师的修为？”
