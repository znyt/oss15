　　心神停留在无字玉璧上，古拙难懂的金色文字，逐渐产生变化，陡然，别人眼中古拙难懂的文字，居然变得通俗易懂起来。
<br/>　　夫天地造化，盖谓混沌之时，蒙昧未分，日月含其辉，天地混其体，廓然既变，清浊乃陈。
<br/>　　天地所以能长且久者，以其不自生，故能长久。
<br/>　　然天地万物，皆有其相，众生沉迷，惑於我相，人相，众生相，寿者相，以为众相故，心生三毒三惧三恐怖，不可久矣。
<br/>　　天象无刑，道褒无名，是故说无我，无人，无众生，无寿者，即达光明，持一正道，内体自性，天地以本为心者也。
<br/>　　故动息地中，乃天地之心见他。
<br/>　　故无实无虚也。
<br/>　　故天地任自然，无为无造也。
<br/>　　故物不具存，则不足以备哉！

<br/>　　...
<br/>　　整整一天的时间，慕清霄的心神都在无字玉璧上。
<br/>　　陆雪琪则早就结束修炼状态，坐在一旁，静静的等待，无字玉璧上，金色古拙的文字，她连一个都看不懂，更别提参悟了。
<br/>　　就在这时，慕清霄身躯一颤，一股无形的波动，在空气中荡起一阵涟漪，随后向四周席卷而去。
<br/>　　待再一次睁开眼眸，慕清霄仿佛历经沧桑，星辰一样的眼眸消失，取而代之的是无尽的沧桑。
<br/>　　“元婴五重中期！”
<br/>　　口中吐出一口浊气，慕清霄身上凌厉的气息全部收敛起来，再次回归以往的平淡，看上去就像是个没有修为的普通青年。
<br/>　　“参悟结束了吗？”
<br/>　　转过头来，迎上来的就是温柔的目光，慕清霄替她捋了捋眼前的青丝，道：“嗯，你参悟的如何？”
<br/>　　陆雪琪微微摇头，道：“无字玉璧来历绝对不普通，上面的金色字体，我一个都看不懂，也无法参悟其中的奥妙，让你失望了。”
<br/>　　“无碍，或许是修为不足，无法参悟其中的奥妙罢了，我修为略有提升，需要稳固下，我们就在天音寺暂住几日，聚时就离开。”
<br/>　　“嗯，都听你的。”
<br/>　　......
<br/>　　......
<br/>　　与往常的日子一样，悠扬的晨钟又一次敲响，回荡在须弥山脉之间，在薄雾山风里回荡着，穿过了无数光阴岁月，还将如此，在未来的日子里日复一日的回荡下去。
<br/>　　站在清晨的晨光中，慕清霄负手而立，侧耳倾听。
<br/>　　此时的他，容貌依旧神骏，但看上去与以往却有些不同，从他的气度神态上，可能发现，比之往昔，多了一分稳重。
<br/>　　陆雪琪一袭白衣，手握天琊神剑，美的惊心动魄，默默的站在他的身后，温柔的视线只为他一人。
<br/>　　片刻后，法相走了过来，双手合十，口宣佛号，道：“阿弥陀佛，听说慕师兄要离开天音寺，恩师暂时无法离开，小僧特地来恭送两位......”
<br/>　　来到山脚下，慕清霄看向法相，道：“替我向普泓主持问好。”
<br/>　　“法相记住了，慕师兄，有缘再见。”
<br/>　　慕清霄回头看了一眼天音寺，在这里住了半个月的时间，他的修为已经稳固在元婴五重中期，的确是一处静修的好地方。
<br/>　　至于有缘再见，慕清霄只好报以微笑，往后恐怕都不会见面了，微微点头道：“行了，就送到这里吧。”
<br/>　　......
<br/>　　......
<br/>　　死亡沼泽位于神州浩土西南水泽之地，方圆几达八千里，连绵不绝，自古人烟罕见。
<br/>　　而其中又分作两层地界，一是外泽，属死泽外围，占去死泽十之七八土地，其中巨树参天，无底泥坑密布，毒虫极多。
<br/>　　内泽比起外则更加凶险，其间终年被剧毒沼气围绕，阴灵怨鬼不计其数，凄风苦雨连绵不绝，误入者十有八九无觅其踪，不知生死。
<br/>　　此时，距离离开天音寺，已经过去一个月的时间。
<br/>　　慕清霄牵着陆雪琪，站在死亡沼泽的最边缘，剑眉皱在一起，道：“西方大沼泽的环境，居然如此恶劣，雪琪，不如你在外围，等我办完事在来寻你？”
<br/>　　陆雪琪微微摇头，道：“当初不是说过，无论何种处境，我们都要一起面对，所以任凭这死亡沼泽凶险无比，我也要与你一起进退。”
<br/>　　闻言，慕清霄知道，想要阻拦陆雪琪是不可能了，微笑道：“好吧，那我们就进去吧。”
<br/>　　说罢，慕清霄就搂着陆雪琪，化作一道剑光，迅速向沼泽内部疾射而去。
<br/>　　天地宝库，传为天帝修真之所，藏有太初神物，有神鬼不测之为，历来为修真之士苦心寻觅，非但如此，便是洪荒巨兽，也多起贪婪之心，蠢蠢欲动。
<br/>　　只是宝库凌然居于亿年神木之上，悬浮与九霄之外，其间禁制重重，更有九天神鸟守护，稍有不慎便万劫难复。
<br/>　　他自然知道这些都是传言，天书虽含有天地至理，但他却能够参悟，若真是太初神灵所有，绝对不可能出现在此处，传闻毕竟是传闻。
<br/>　　谁也没有抵达过天帝宝库，传闻越来越多，版本也越来越多，谁也不知道是真是假，慕清霄并不知道传闻中的神木在何处，但天帝宝库绝对是在这死亡沼泽中，唯有一步步寻找。
<br/>　　在死亡沼泽中，地毯式的寻找了两天，期间斩杀毒虫，抹灭阴灵冤鬼无数，也就是特殊形势存在的精神力量，这种特殊的精神力量能够侵入人的心神，心神不坚定者，有走火入魔的风险。
<br/>　　当然，对慕清霄来说，这些精神力量与他双眼中存在的精神力量相比，简直是小巫见大巫，完全不可同日而语。
<br/>　　越是深入死亡沼泽，其内部的环境就越是恶劣，外围还好，到了内部，天空也被浓浓的雾气所笼罩，彼此涌动，身处其中时间一长，便会使人感到压抑。
<br/>　　这种压抑就像是慢性毒药，配合死亡沼泽中的重重危险，随时可能要了闯入者的性命。
<br/>　　越是深入，雾霾就是深厚，肉眼所能看到的终究有限，慕清霄也不得不用神识开始仔细搜索每一块角落。
<br/>　　也只有他能够如此便利，别的修士，唯有抵达金丹后，方能稍微开启一丝神识，但他的神识笼罩范围，随着修为提升，早已达到万米。
<br/>　　所以，搜索对他来说，轻而易举。
