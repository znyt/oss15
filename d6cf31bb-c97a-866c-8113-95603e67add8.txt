<br/>				    尘烟铺天而起阻挡了众人的视线，他们只感觉到一股强大的威压自那窟窿处往外荡开，强大气流呼的一声传出，让那周围百米之内，无论是人还是物尽数飞出。<br/>　　<br/>　　    半空之中，凤九清眸中难掩震惊的看着下面的一幕，没想到墨泽竟有这个能力一拳将那魔主从半空中狠狠击落。<br/>　　<br/>　　    只见，那尘烟渐渐散去之时，那窟窿处的一幕也随着出现在众人的视线里。<br/>　　<br/>　　    那一袭黑袍，一脸胡子的大汉一拳抵在那被击倒，后背着地躺在窟窿里面的魔主胸口处，那魔主此时脸色苍白，一口鲜血从那口中溢出，他仿佛起不来身一般，躺在那里闷哼着，倒抽着气。<br/>　　<br/>　　    而那一袭黑袍的胡子大汉通身戾气，抵在魔主胸口处的那一拳的拳头处，还有着雄厚的灵力气流卷动着，他的目光冷冽而森寒，嗜血杀意在深邃的眼瞳中浮现，没给魔主喘气的时间，他再次挥拳击落。<br/>　　<br/>　　    这一拳快而狠的从魔主的右下巴处往左挥去，一拳击出的同时，那魔主一口鲜血也随着喷，而混夹在那一口鲜血中的，还有被轩辕墨泽打落的牙齿。<br/>　　<br/>　　    然，当他准备掐断他的喉咙取魔主的性命时，那魔主身上猛的迸射出一股强大的威压，见他身体气息起了变化，轩辕墨泽当即松开手迅速退离。<br/>　　<br/>　　    也就在他退离的那一刻，一道红色闪光咻的一声从他刚才所在的地方袭过。看见这一幕，深邃的目光眯了起来，手中光芒一闪，轩辕剑便出现在他的手掌之中。<br/>　　<br/>　　    魔主一手捂着五脏六腑皆疼的胸口站了起来，他另一手握着一把泛着血红色的利剑，目光阴森地盯着那轩辕墨泽，下一刻，身影一闪便以着光速掠上前去。<br/>　　<br/>　　    血色长剑上的剑罡之气如同一道血柱一般的袭出，当那看到轩辕墨泽以上古神剑轩辕剑抵挡住他那一击时，他的速度没有停留，再度战了起来。<br/>　　<br/>　　    两抹黑色的身影，不分上下的实力，一来一往的在半空中交战着，强者的气息弥漫而出，强大的威压比起之前更盛……<br/>　　<br/>　　    也许是之前冷不防的被轩辕墨泽一拳击中胸口，伤了五脏六腑。此时，魔主在与轩辕墨泽交手数十招后，速度便渐渐的慢了下来，就连攻击也弱了几分。<br/>　　<br/>　　    意识到自己身体内伤的加重，以及眼前这胡子大汉强硬的实力，知道一时半分无法将之灭杀的魔主在挡住轩辕剑狠狠劈来后剑后，整个身体猛的往后退去，瞬间退出几百米的距离后才在半空中停下来，盯着轩辕墨泽以及凤九。<br/>　　<br/>　　    “本魔不会就这样算的！凤九，你们给本魔等着！本魔一定还会再来的！”声音一出之时，身影往天空之处掠去，随着身影的远去，那声音也渐渐的弱了下来。<br/>　　<br/>　　    看见魔主撤离，剩下的那些负了伤的魔修也在看到他离开之后，迅速地跟着撤退。<br/>　　<br/>　　    见那些魔修如同潮水般退去，底下的众人只觉不可思议……<br/>
