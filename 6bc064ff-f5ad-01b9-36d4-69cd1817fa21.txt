<br/>　　“吾为紫霄，见过道友。”那人面色和善的对着沉浸在突破感悟之中的道祖微微一笑。

<br/>　　“恭喜道友突破准圣。”
<br/>　　元始看着道祖，恭喜道。同时，他心中却多少有些复杂，道祖就是道祖，在没有人教导之下，依然是以斩尸之法突破准圣，在另一个洪荒之中，斩尸之法正是道祖所创，虽然不是同一个人，但命运轨迹依然很相似，这真是一件奇怪的事。
<br/>　　同时，他也在猜测，也许，这道祖便是洪荒世界鸿钧道祖的一个“他我”之身吧。无数个相似的“他我”，究竟谁是真身，谁是应身，恐怕谁也说不清。不过，最终终究是逃不脱被人融合的命运。
<br/>　　“同喜，想必道友距离突破也不远了吧。”道祖同样对着元始恭喜道。
<br/>　　元始微微一笑，也不回答，他这五百年时间里，一直都在参悟大道，同时也炼化黑暗本源，已经凝聚出道源雏形了，距离突破混元已经不远了。不过，终究是受到些黑暗本源的影响，他的大道是以黑暗为主，融合了一些其他道则星辰的独特暗黑大道。
<br/>　　黑暗大道并不是暗黑大道，黑暗大道是以黑为主，其他三千大道为辅的大道，以黑暗凝聚大道，具有一切道性特征，凝聚的黑系法则越多，大道便越强，而暗黑大道却是以暗为主，暗并不是黑，暗借指一切暗系大道，其中包括黑暗，两者十分相似，但却又不完全相同。三千魔神之中，就有黑暗魔神与原暗魔神两种，其实他们并不是一人。
<br/>　　突然，元始感受到一种强烈的死亡危机，他豁然抬头，看向大殿深处，因为他觉得自己浑身冰凉，如坠冰窟，好似被什么东西盯上了。
<br/>　　道祖同样有这样的感觉，转身看向大殿深处。
<br/>　　果然，于那殿宇深处，黑暗迷雾之中，有一双略显空洞的眼睛，缓缓睁开，凝视着两人，让两人浑身发凉。
<br/>　　这人果然没死！
<br/>　　那强大无比的气机完全锁定了两人，让他们无处可逃，元始有种感觉，若是此刻自己只要向外动一动，迎接他的便会是一道强大的攻击。虽然这人的状态好似出了问题，导致神志不清，但即便强如元始，也不敢去试探一下，他有预感，自己恐怕不是这人的对手。
<br/>　　道祖同样如此，两人对视一眼，都感觉到了对方心中的那份沉重。

<br/>　　“为你而来！”
<br/>　　元始冷漠警惕的看着黑暗深处那人，淡淡回答道。在他看来，对方今日是不可能放他们离去，甚至可能将他们吞噬，补充精气，既然早晚都是敌人，自然不需要多费口舌，而且他也不是一个喜欢多扯闲话之人。
