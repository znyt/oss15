　　“嗖！”
<br/>　　瞬息之间，楚风眠手中的剑锋落下。
<br/>　　刹那。
<br/>　　万道剑气，同时打出，周围的一切，仿佛都陷入到了无数剑气之中。
<br/>　　整个空间，都是化为了剑的世界。
<br/>　　“什么？”
<br/>　　众人看到这变化，简直是不可置信。
<br/>　　万道剑气。
<br/>　　就在刚刚，楚风眠打出的一剑中，居然是直接分化为万道剑气。
<br/>　　在这兵器之道的领悟上。
<br/>　　楚风眠的剑道，甚至是还要超越洪狱的三千虚幻。
<br/>　　“这怎么可能！”
<br/>　　“一剑万重，这是传说中的境界啊！北域中的无数剑道宗师，都没有踏出到这一步！”
<br/>　　“此人到底是什么来历！这么会有着如此可怕的剑道！”
<br/>　　围观武者中的一些剑修，看见楚风眠展现出的剑道，一个个都要发狂。
<br/>　　一剑万重，绝对是九域中无数剑修梦寐以求的境界，无数人都是嫉妒发狂。
<br/>　　“这！”
<br/>　　洪狱看到这万重剑气，打出的一刻，脸色也是骤变，但是一瞬之间，便是又化为了之前的自信。
<br/>　　“剑道高强又如何？力量不足，不过只是花拳绣腿罢了，真正的力量，才是王道，你就看着我的三千虚幻，毁了你的剑道！”
<br/>　　“破！”
<br/>　　洪狱的一身灵力，已经是全然展现出来，全部加持在了他的三千虚幻之上。
<br/>　　要以一力破万力。
<br/>　　以力取胜。
<br/>　　看到这一幕，楚风眠心中冷笑连连。
<br/>　　就算是对拼力量，楚风眠也不会怕。
<br/>　　“你以为跟我对拼力量，你就是我的对手了？从你敢挑衅我的第一刻起，你的失败，就是注定的了！”
<br/>　　楚风眠脸色平静到极致，心神一动，催动万道剑气，同时袭杀过去。
<br/>　　“轰隆！”
<br/>　　三千虚幻，万道剑气，在空中，顿时碰撞在了一起，一道道余波，都是恐怖至极，不下于在场许多武者的全力。
<br/>　　这碰撞激烈无比，但是在瞬息之间，便是分出了胜负。
<br/>　　只见从中还存在的，却是万道剑气。
<br/>　　洪狱的三千虚幻，其中的所有力量，都被全部吞噬，被这万道剑气，吞噬殆尽。
<br/>　　“不止是在武道领悟方面，就连力量，洪狱都败了？”
<br/>　　众人看到这一幕，都是一愣。
<br/>　　洪狱的实力，在场任何一人都清楚的。
<br/>　　在武道领悟上败了，众人就已经无比惊讶了。
<br/>　　但是在北域中，出现一些剑道天才，也并非是无法理解的事。
<br/>　　毕竟天才终究是天才，剑道领悟在高，没有实力，也没有任何意义。
<br/>　　但是能够打破洪狱的三千虚幻，这样已经不是在武道领悟上，超越了洪狱了。
<br/>　　而是在力量上，楚风眠甚至都是超越了洪狱。
<br/>　　这天盟的副盟主，居然如此的不堪一击？
<br/>　　一些武者的心中，都突然冒出了这个想法，一直以来近乎无敌的人物，居然败的如此惨，谁也无法相信。
<br/>　　只是大部分的人，却是将目光看向了楚风眠。
<br/>　　不是洪狱的实力弱，洪狱的实力，放在外面，依然无敌，而是楚风眠的实力太强了。
<br/>　　这等实力，已经是强大到足以碾压洪狱的地步。
<br/>　　才会让洪狱，败的如此之惨。
<br/>　　“绝不可能！我拥有无数奇遇，我怎么会败！”
<br/>　　洪狱疯狂的咆哮着，似乎还是无法接受眼前的这一幕。
<br/>　　他无论如何也不会想到，他居然会败在楚风眠的手中。
<br/>　　在洪狱看来，楚风眠只是一个小人物，甚至是就连让他认真资格都没有的小人物，怎么可能会想到，居然会败在楚风眠的手中，败的如此彻底。
<br/>　　没有任何一点点的，还手之力。
<br/>　　“我说过，你在我的面前，就是蝼蚁罢了，神力境中，绝无可能有人跟我抗衡。”
<br/>　　楚风眠看着洪狱，冷冷的开口道。
<br/>　　“既然你冒犯了我，那你也就跟那欧阳烈一样的下场吧。”
<br/>　　“给我跪下！”
<br/>　　轰！
<br/>　　楚风眠开口之际，在他的身上，便是显现出了极为恐怖的威压，如同一座高山，将楚风眠的身躯疯狂拔高，令楚风眠看起来，简直便是如同一尊庞然大物一般。
<br/>　　无尽龙威，从楚风眠的身上涌现出来，简直是如同一尊龙皇在世一般。

<br/>　　听着洪狱的话，楚风眠眼神中露出几分不屑道。
<br/>　　这洪狱，死到临头，还敢来威胁他。
<br/>　　“给我跪！”
<br/>　　楚风眠大手一挥，一股恐怖威压直接压迫在了洪狱的身上。
