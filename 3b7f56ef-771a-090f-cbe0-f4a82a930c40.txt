        恰巧，傅夜沉当时有想拓展娱乐圈里的业务，所以顺便提携了苏漫雪。
<br/>
<br/>        他给她安排了一个优秀的经纪公司去包装她，还介绍了不少的好资源去推销她。
<br/>
<br/>        苏漫雪是“C位”出道，又加之她有一张整过容的精致脸蛋，身材火辣，几乎是一夜爆红。
<br/>
<br/>        当时，傅夜沉还曾想过，如果是筱颖的真容出面，说不定筱颖一定比她苏漫雪会更红，毕竟，筱颖的歌声很动听。
<br/>
<br/>        二人一番寒暄过后，苏漫雪的目光便落在了李忆莎的身上：“这位小姐是……”
<br/>
<br/>        李忆莎抬头看着傅夜沉，不敢随便回答，实际上她也不知道自己该怎么做自我介绍。
<br/>
<br/>        今晚的电影节颁奖典礼以及现在的这场晚宴，是冷爷爷把她叫过来给傅夜沉当女伴的。
<br/>
<br/>        傅夜沉刚巧也需要一个女伴陪同，本来他打算让落以柔出面陪他一起参加，但是落以柔说她来生理期了，肚子不舒服，所以，此事也就作罢了。
<br/>
<br/>        “长得真漂亮，难怪夜沉会忍不住带着你出来炫耀一番了。”苏漫雪得不到响应，转而试探地夸奖，又加以暧昧的暗示，希望傅夜沉能主动介绍一下。
<br/>
<br/>        傅夜沉却只是微微勾了勾唇角，没有应声也没有否认，模棱两可的态度让那位苏漫雪搞不清他的想法。
<br/>
<br/>        话说回来，他傅夜沉不是喜欢童筱颖吗？
<br/>
<br/>        不过，她倒是听说，童筱颖跟傅昼景去了澳洲定居了。
<br/>
<br/>        也对，像傅夜沉这么优秀的男人，总不能在童筱颖那一棵树上吊死吧！
<br/>
<br/>        童筱颖离开的这两年里，苏漫雪也再次试图去追求过傅夜沉。
<br/>
<br/>        可是她追求傅夜沉已经有段时间，却从没有被他接受，此时她的心中充满了危机感和焦虑，忍不住试探性地问：“夜沉，能否借一步说话？”
<br/>
<br/>        她要再次赌一赌，傅夜沉没有明确说出这个女人的身份，搞不好只是带来当挡箭牌的，她也许还有希望。
<br/>
<br/>        毕竟，她还有着一张，跟童筱颖长得一模一样的脸！
<br/>
<br/>        听到这话，李忆莎连忙朝傅夜沉投去哀求的目光，她在这里认识的人只有他，如果他离开的话，她一个人真的不知道如何是好。
<br/>
<br/>        但是，傅夜沉看不懂她的眼神，而是淡淡地对她说道：“我跟苏小姐谈一些事情，你先自己去吃点东西。”
<br/>
<br/>        语毕，他轻轻地拉开了李忆莎搭在自己臂弯上的手。
<br/>
<br/>        傅夜沉先是应付完苏漫雪，随后又被几个商界大佬给绊住。
<br/>
<br/>        他虽然是后起之秀，但是也不会自持清高，不与前辈们往来。
<br/>
<br/>        傅夜沉在这几位大佬们的面前，哪怕他的身价现在已经是他们的好几倍，他还是会以晚辈的姿态，对他们非常毕恭毕敬。
<br/>
<br/>        觥筹交错，总有人陆陆续续地主动来找他傅夜沉碰杯交流。
<br/>
<br/>        李忆莎完全被傅夜沉晾在了一边，更确切地来说，傅夜沉压根就忘了，自己是带着女伴来的这件事情。
<br/>
<br/>        他对她不上心，所以也就对她不在意。
<br/>
<br/>        李忆莎愣愣地站在原地，不知所措地望着傅夜沉随着那几位商界人士一起走远，他的身影很快又被一群女人遮挡住，她连看都看不到他了。
<br/>
<br/>        委屈地走向角落，李忆莎忍不住有点想哭。
<br/>
<br/>        她都已经这么努力了，为什么还是入不了他的心呢？
<br/>
<br/>        李忆莎只觉得胸口闷得慌，想要冲过去将他拉出女人们的包围圈，但是怎么也迈不开步伐。
<br/>
<br/>        他又不是她的谁谁谁，她也不是他的谁谁谁，她又凭什么冲过去？
<br/>
<br/>        就在李忆莎独自坐在角落里的沙发上，感到无助又无聊的时候，抬眸地无意间她发现了一个跟刚刚那个女明星苏漫雪长得一模一样的女孩。
<br/>
<br/>        李忆莎之所以能分清楚苏漫雪和这女孩，是因为这个女孩不同于苏漫雪那般成熟妩媚。
<br/>
<br/>        女孩那玲珑有致的身段搭配一件合身的白色连衣裙，脚上踩着一双精致的米色低跟鞋，将她整个人衬托得十分活泼。
<br/>
<br/>        一头披散及腰的如瀑布般的长发，为她那过分精致的容颜增添了几分可爱，再加上她时不时懊恼地垂眸的神情，让一旁经过的男士们频频侧目。
<br/>
<br/>        暗中关注那个女孩的人很多，她突然说出的一句话，让周围的人神情都愣一愣。
<br/>
<br/>        “夏工，我们还是回包房里去做吧！”
<br/>
<br/>        童筱颖的这句话，完完全全地让周遭的人给误会了。
<br/>
<br/>        大家一个个用异样的目光，投向了她和她的雇主夏凌焰。

<br/>        顿时，童筱颖汗颜了。
<br/>
<br/>        喂喂喂！
<br/>
<br/>        别想歪了啊！

<br/>        她说的是回包房去做建筑模型。
