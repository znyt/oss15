　　“我们只是暂时落脚，待会还有正事要办，不必和他们争长论短！”
<br/>　　叶尘忽然开口道。
<br/>　　纳兰柔然原本正气的不行，听到叶尘这话，顿时便不敢再继续惹事，当即哦了一声，乖乖退到一旁。
<br/>　　不料，这个时候，
<br/>　　那位赫连公子却忽然开口了，
<br/>　　“纳兰小姐是吧？相逢即是有缘，如果小姐不嫌弃的话，可以随我一同去天字号包厢，我请你和你的朋友喝一杯，咱们交个朋友，你看如何？”
<br/>　　这赫连公子表面上彬彬有礼，两眼却在纳兰柔然，那堪称完美的娇躯上，瞄来瞄去，眼眸深处闪过一抹炙热，心中暗道：
<br/>　　‘都说罗刹族女子美貌无双，果然是名不虚传啊！’
<br/>　　纳兰柔然此时还在气头上，直接翻了一个白眼，没好气道：
<br/>　　“谁稀罕和你交朋友！一边待着去！”
<br/>　　“大胆！我家公子请你喝酒，那是看得起你，你竟然敢对我家公子无礼？”
<br/>　　赫连公子的身后，当即窜出一名金甲侍卫，冲着纳兰柔然怒声喝道。
<br/>　　纳兰柔然身后的侍卫，自然也不甘示弱，
<br/>　　“是你大胆！敢对我们公主无礼！”
<br/>　　“一个小小的罗刹城，也敢自称公主？这里可不是你们能撒野的地方！”
<br/>　　那名金甲侍卫说完这话，直接一拳砸了过去。
<br/>　　而那名罗刹城侍卫，慌忙抬起双臂格挡，可是两人的实力差距实在太大，后者几乎没有任何抵抗之力，就被金甲侍卫打飞了出去。
<br/>　　“你找死！”
<br/>　　纳兰柔然看到自己的手下被人殴打，顿时也坐不住了，当即抽出长鞭，便朝那名金甲侍卫冲了过去。
<br/>　　啪啪啪！
<br/>　　一连几记长鞭挥出，一道道清脆的鞭声响起，那些被长鞭击中的桌椅，顿时瞬间破碎。
<br/>　　一眨眼的功夫，整个酒馆已经千疮百孔。
<br/>　　然而，
<br/>　　那金甲侍卫的修为，着实不弱！
<br/>　　尽管这纳兰柔然，也算的是少有的天才，可是毕竟年轻太轻，修为和对方有着一定的差距。
<br/>　　两人交手了数十招后，纳兰柔然不但未能伤到那金甲侍卫分毫，整个人也已经落至下风，开始节节败退。
<br/>　　“阿三！不要伤她性命，本公子要活的！”
<br/>　　“是！公子！”
<br/>　　那金甲侍卫桀桀一笑，忽然身影一阵变幻，速度猛然暴涨，然后以一个诡异刁钻的角度，一掌拍在了纳兰柔然的肩头之上。
<br/>　　蹬蹬蹬！
<br/>　　纳兰柔然顿时一连数步，脸色煞白无比。
<br/>　　“桀桀！小姑娘，再给你最后一次机会，乖乖陪我们家公子喝一杯，否则可休怪我不客气啦！”
<br/>　　金甲侍卫一脸奸笑，再次朝纳兰柔然缓缓逼近。
<br/>　　唰唰唰！
<br/>　　纳兰柔然身后的数名侍卫，当即纷纷冲了上来。
<br/>　　可惜，这些人的实力，比纳兰柔然都大为不如，哪里会是那金甲侍卫的对手？
<br/>　　砰砰砰！
<br/>　　几乎一小会的功夫，几名侍卫已经全部被那金甲侍卫打倒在地，一片哀嚎起来。
<br/>　　周围众人看到这一幕，又纷纷出言嘲讽开来，
<br/>　　“都说罗刹城实力暗弱，果然是不假！”
<br/>　　“要我说，是那位太强！恐怕至少也有凝血境中期的实力吧？”
<br/>　　“看此人的装扮，应该是赫连将军府，大名鼎鼎的金甲卫！”
<br/>　　“我想起来了，此人应该是金甲卫八大统领中的铁阿三，号称金刚铁拳！”
<br/>　　“难怪有如此实力！”
<br/>　　……
<br/>　　听到众人的议论声，铁阿三顿时越发得意，
<br/>　　“纳兰小姐，你的手下，已经全部被我打倒，你可还要反抗吗？”
<br/>　　纳兰柔然俏脸发白，不过猛然看到身后的叶尘，顿时两眼一转，当即快速退到叶尘的身旁，可怜巴巴道：
<br/>　　“叶前辈，救我……”
<br/>　　叶尘原本并不想多生事端，可是这纳兰柔然，毕竟是他的同伴，他也不能坐视不管，不由得摇了摇头，道：
<br/>　　“站到我身后！”
<br/>　　纳兰柔然顿时大喜过望，当即乖乖站到了叶尘的身后，然后还不忘冲着铁阿三扮了一个鬼脸，
<br/>　　“想让我陪你们公子喝酒，你先打赢叶前辈再说吧！”
<br/>　　“叶前辈？”
<br/>　　那金甲侍卫的目光，这才落在了叶尘的身上，打量了片刻之后，顿时冷冷一笑，
<br/>　　“一个毛头小子而已，也敢称前辈？”
<br/>　　“小子，识相的话，就乖乖自己让开，本统领或许可以饶你不死！”
<br/>　　叶尘两眼低垂，从头至尾，连眼皮都没有抬一下，淡淡道：
<br/>　　“给你十息时间，不滚，就死！”
<br/>　　哗！
<br/>　　叶尘这话一出口，顿时满座哗然，
<br/>　　“这小子，真是好大的口气啊！”
<br/>　　“竟然敢和铁统领如此说话！”
<br/>　　“看他的模样，好像也就是一个普通侍卫吧？而且这罗刹城，又能有什么高手？”
<br/>　　……
<br/>　　铁阿三听到叶尘的话，顿时气得肺都要炸了，
<br/>　　“小子！你可真是大言不惭！既然如此，那你就给本统领死去吧！”
<br/>　　一声暴喝之后，铁阿三猛然一掌拍出，
<br/>　　轰！
<br/>　　一股强悍的拳劲，透着赤红色的妖异光芒，朝叶尘狠狠轰了过去。
<br/>　　论威力，比之前强大了一倍都不止！
<br/>　　“这才是铁统领真正的实力吗？”
<br/>　　众人看到这一幕，顿时再次暗暗感叹，
<br/>　　轰！
<br/>　　这一拳，狠狠砸落在叶尘的身上。
<br/>　　在众人看来，叶尘就好像是没来得及躲避了一般。
<br/>　　“铁统领的实力太强了，这小子竟然连躲开的能力都没有！”
<br/>　　“中了如此强大的一拳，肯定必死无疑！”
<br/>　　“太弱了！”
<br/>　　……
<br/>　　众人纷纷摇头，铁阿三的脸上也露出一抹得意之色，
<br/>　　“还以为你有什么本事，原来也不过……什么！”
<br/>　　铁阿三的话，还没有说完，猛然脸色大变，仿佛看到了世间最为恐怖的事情一般。
<br/>　　只见，那个生生挨了他最强一拳的小子，随意拍了拍衣衫，淡淡道：
<br/>　　“就你这点力道，给我挠痒，我都嫌轻！”
<br/>　　说完这话，叶尘直接屈指一弹，
<br/>　　嗖！
<br/>　　一掉凌厉无比的剑气，自他的指尖爆射而出。
<br/>　　“不！”
<br/>　　铁阿三顿时瞳孔一缩，当即大吼了一声，身体迅速暴退。
<br/>　　可惜，那道剑气实在太快了，快到他根本来不及躲闪。
<br/>　　噗！
<br/>　　一道微弱的剑芒在空中闪过，
<br/>　　下一刻，
<br/>　　那铁阿三的脑袋，已经飘飞了起来，
<br/>　　噗通！
<br/>　　片刻后，
<br/>　　无头尸体轰然倒地，
<br/>　　整个酒馆，顿时鸦雀无声。
