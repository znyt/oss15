　　察觉到气氛略有些压抑，木嶋式坐在餐桌首座前，率先拿起了筷子。
<br/>　　“快些吃吧，我们还要去总部集合。”夹起一块鸡肉放入了旧二多福面前的餐盘中，木嶋故作轻松的开口。
<br/>　　作为一名老牌的准特等搜查官，他自然知道这次任务的危险程度，他们班的这些人，也不知道能有几个活着回来。
<br/>　　但他们的工作就是如此，就算明知道可能会死，也必须参与进去。
<br/>　　这些事情，总要有人去做才行。
<br/>　　在一阵压抑的气氛中，众人吃过了晚饭，木嶋式用餐巾擦了擦手，起身提起了自己的银色手提箱。
<br/>　　刘哔等人也站起了身来，跟随木嶋式走了出去。
<br/>　　像他们这种刚刚加入CCG的新人搜查官，虽然没有自己的库克因，但CCG也给他们提供了一些精良的热武器。
<br/>　　本来搜查官的身体素质就优于常人，再加上这一身不次于特种部队的装备，能发挥出的战斗力还是十分可观的。
<br/>　　一路上有些拥堵，木嶋开着一辆小型轿车，行驶到CCG的总部大楼门前。
<br/>　　现在这个时间，大部分搜查官都已集合完毕，会议室宽广的空间内，挤满了密密麻麻的人群。
<br/>　　丸手斋站在台上，似乎在等待着什么，木嶋班几人推门进来的时候，丸手眼睛一亮，冲几人走了过来。
<br/>　　“刘哔是吧，你提供的消息已经得到了证实，我们会在今夜针对那家咖啡厅组织一场进攻，考虑到这则消息是由你提供，所以需要你的跟随。”
<br/>　　礼貌的冲木嶋式打了个招呼，丸手十分客气的对刘哔开口。
<br/>　　而丸手的要求倒也正好符合他一开始制定的计划，能跟随另一组人员行动，这对他而言再适合不过了。
<br/>　　反正青铜树那边，又没有太大的动作可以搞，从始至终，那场战役死去的高阶喰种并不多。
<br/>　　虽说搞得规模挺大，但知道内情的人，都明白那场战役只是一场双方提前密谋好的演习。
<br/>　　“好的，那我现在应该去哪里？”顺从的点了点头，刘哔向丸手斋开口问道。
<br/>　　似乎是十分满意刘哔的服从度，丸手拍了拍他的肩膀，示意其跟上他的脚步。
<br/>　　看了一眼满脸担忧的木嶋式，两人一前一后从会议室中走了出去。
<br/>　　或许是因为大部分的搜查官都聚集在会议室，走廊上有些空旷，平日里四处可见的工作人员也没见到一个。
<br/>　　走过长廊，丸手将他带入了一间办公室内，此时屋里已经等候了一些人，有一些他认识，但大部分都不怎么熟悉。
<br/>　　听到门口处的响动，众人都是将目光投了过来，但大部分人都皱着眉头，没有说话。
<br/>　　只有什造冲他眨了眨眼睛，开心的笑了起来。
<br/>　　“啊，这位就是给我们提供情报的新人了，这次行动，也会跟随你们一起出征。”丸手一只手扶着刘哔的肩膀，向众人介绍到。
<br/>　　被丸手的话语吸引，屋内的搜查官们目光皆是闪动了一下，后方一个矮胖的中年人走了过来，冲他伸出了手。
<br/>　　“现在的新人，可真是厉害呢，这么机密的消息都能搞到，真是让人意外。”中年男子脸上带着笑容，表情有些憨厚。
<br/>　　顿了一下，男子看着刘哔有些苍白的右手，笑道：“我是筱原幸纪，特等搜查官，也是二十区的总负责人。”
<br/>　　只是一个特等吗，这样的实力，应该还不足以对付‘枭’吧。
<br/>　　暗中皱了皱眉头，刘哔也没有显露出什么，只是抽回自己的右手，站在丸手斋的身后。
<br/>　　“好了，这里就没我什么事情了，我要回会议室了，那边的行动也不能耽搁呢，最后，祝大家的行动都能顺利。”丸手挥了挥手，转身退出了办公室。
<br/>　　筱原幸纪也没有多说什么，等到丸手离开之后，转身来到了办公桌前。
<br/>　　木质的桌子上，整齐摆放了一些书信，封面上的字体，标明了这是一封封遗书。
<br/>　　屋内没开窗户，光线有些阴沉，众人都低垂着头颅，攥起的拳头微微有些颤抖，士气稍稍有些低落。
<br/>　　和外面那些家伙不同，他们知道自己的任务是什么，而这次任务所面临的危险，也是无比巨大。
<br/>　　在一名SSS级超高等阶的喰种面前，没有谁敢保证自己不死，就连平日里尊贵无比的特等搜查官，也是如此。
<br/>　　“各位，放心战斗，有我陪在你们身边。”看着心情低落的众人，筱原幸纪轻叹一声，收起了脸上的笑容。
<br/>　　看着筱原幸纪宽厚的身体，众人的心情都稍稍平复了下来，一股无比踏实的感觉从心底涌了上来。
<br/>　　似乎不管前方有再大的风雨，只要这个男人还在，就不会伤及到他们一般。
<br/>　　唯一没有受到场中气氛影响的，也只有不知何时溜到刘哔身边的铃屋什造了，这个家伙，恐怕没有什么事能引起她心中的恐惧。
<br/>　　良久，就在刘哔等的都有些厌烦的时候，众人腰间的联络器却不约而同的响了起来，什造斜靠在一旁的墙壁上，满脸疑惑的拿出了联络器。
<br/>　　几乎所有的人都执行了这一动作，联络器上的信息，只有两个字。
<br/>　　“行动！”筱原幸纪一把抓起了桌子上的手提箱，带头从屋内冲了出去。
<br/>　　……
<br/>　　CCG总部大楼门前的广场上，十余名手持银白色手提箱的搜查官匆匆跑了出来，其中两手空空的刘哔，尤为扎眼。
<br/>　　门外已经聚集了黑压压的一大片人群，几架直升机吊在上风，正在挥舞的螺旋桨发出一阵刺耳的呼啸声。
<br/>　　一名身穿白色军服的男子站在人群前方，像是在等待着什么，随着诸多搜查官一齐出征的，还有一众全副武装的士兵。
<br/>　　看着这副架势，刘哔微微放下了心，他猜想的没错，CCG对于这场战争，给出了足够的重视。
<br/>　　筱原幸纪带领着众人，来到自己的位置站好，他身边的这十几个人，都是二十区最为精锐的搜查官们。
<br/>　　不光是二十区，其他几个区域大致情况也是如此，可以说这些人，汇聚了CCG绝大部分的力量。
<br/>　　静等了片刻，几个空出的区域都被填满，前方的白衣男子统计了一下人数之后，缓缓平举了右手，指向前方。
<br/>　　“讨伐作战，开始！”随着一声令下，广场上立即沸腾了起来。
