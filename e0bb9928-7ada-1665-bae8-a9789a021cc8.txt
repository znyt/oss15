　　擦拭着还有些湿漉的头发，刘哔穿着一条短裤，从洗澡间里走了出来。
<br/>　　客厅里，西尾锦正在看电视，注意到刘哔出来，十分恭敬的站起身来，抱着衣物向洗澡间走去。
<br/>　　“给我准备一套衣服，我一会要出去一趟。”扔掉手里的毛巾，刘哔淡淡的吩咐了一句，打开卧室的大门。
<br/>　　什造还没醒，正侧身躺在床上熟睡着，脸色却不像昨晚那样苍白了。
<br/>　　轻轻叹了口气，刘哔来到床边，看着什造熟睡的测验，不免有些担心。
<br/>　　本来身上就带着伤，昨晚在月山家的时候还进行了那么激烈的战斗，让伤势更加严重了。
<br/>　　“真是的，逞什么强啊，那些家伙，交给我来对付就好了。”
<br/>　　摸了摸什造的头发，刘哔语气有些无奈，虽然从商城里兑换了一些药品，可依旧很难短时间内将什造救好。
<br/>　　还有一个月的时间么……
<br/>　　想到自己马上离开，刘哔有些惆怅的停下了手，看着什造，心中不知道在想些什么。
<br/>　　良久，门外传来了一阵敲门声，收回了自己的思绪，刘哔打开了屋门。
<br/>　　“大人，这是您吩咐准备的衣物，都是刚从外面买来的。”
<br/>　　西尾手中捧着一套还未开封的西服，递到了刘哔面前。
<br/>　　点了点头，刘哔接过衣服，他需要去做一些事情。
<br/>　　……
<br/>　　丸手斋身为CCG的特等搜查官，住所自然也不会在集体宿舍里。
<br/>　　CCG后方的一栋别墅区里，一队队身穿军装的护卫巡逻着，这段时间东京并不安全，所有人都在防备着可能到来的危险。
<br/>　　晚秋，淡黄色的枯叶落了满地，被风一吹，掀起了诸多小型旋涡。
<br/>　　似乎是察觉到什么，此时恰好巡逻到的护卫队长，有些疑惑的停下了脚步。
<br/>　　“队长，有什么指示吗？”身后，几个巡逻队员凑了上来，顺着护卫队长的视线向前看去，看到前方空空如意的空地。
<br/>　　摇了摇头，护卫队长整了一下队伍，继续在别墅区内巡逻着。
<br/>　　他刚才感觉身边好像经过了什么一般，但是转头查看的时候，却没有发现什么。
<br/>　　或许是这阵子太过于紧张原因，看样子是要多注意休息了。
<br/>　　嘀咕了一声，护卫队长领着队伍缓缓离开，这片区域瞬间安静了下来。
<br/>　　过了不久，远处的绿化带中探出一个戴着鸭舌帽的头颅，确定了四周没什么危险之后，从藏身处走了出来。
<br/>　　“叮咚~”
<br/>　　门铃声响起，刘哔站在丸手斋门前，穿着一身从快递小哥身上脱下来的衣服，手中还提着一件包裹。
<br/>　　被帽子遮盖住的脸庞，根本让人看不清他的真实面容。
<br/>　　当然，那位被他选中的倒霉快递员，已经不幸身亡了，他可不想在行动的时候发生什么意外。
<br/>　　“刺……刺……你是谁？”
<br/>　　大门处的对讲机上，传来丸手斋浑厚的声音，似乎是刚睡醒不久，声音中还带着一丝慵懒。
<br/>　　“请问这里是丸手先生家吗？”低头查看了一下包裹，刘哔十分客气的问道。
<br/>　　对讲机中一阵沉默，似乎是在核对什么事情，几分钟之后，丸手的声音再度从中传出：“我并没有买东西啊。”
<br/>　　“可是收件人写的是丸手斋啊，地址也是对的，是不是您家人购买的，或者是其他人给您送的？”做出一副奇怪的样子，刘哔重新核对了一下包裹。
<br/>　　“我问过了，都没买东西啊，或许是有人给我送的，寄件人是……”眼前的别墅大门打开，丸手斋穿着睡衣，满脸疑惑的站在门口，话说到一般，却突然被捂着嘴巴带入了别墅内。
<br/>　　“砰！”
<br/>　　别墅大门重重关上，刘哔捏碎了丸手斋四肢的骨骼，像扔死狗一般将其扔到地上。
<br/>　　一阵头皮发麻的惨叫声响起，四肢被废，丸手根本做不了什么。
<br/>　　而像这样的高端住宅区，别墅内的隔音效果自然是一流的，别说发出惨叫，即使是从屋里用音响放重低音摇滚，从外面也很难听到。
<br/>　　只是，他从来没有像今天这样讨厌过自己家的隔音效果。
<br/>　　将丸手斋限制住，刘哔并没有掉以轻心，谨慎的搜查了一圈，从一间小房间的衣柜中提出一个和丸手斋有八分相似的男童。
<br/>　　确认没有什么问题之后，刘哔拉过一张椅子，坐在了丸手面前。
<br/>　　“这是你的儿子？”看着怀中瑟瑟发抖的男童，刘哔摸着他的脑袋，心中不知道在想什么。
<br/>　　长得还挺像，几乎就和缩小版的丸手斋一样，看起来倒也有些可爱。
<br/>　　小孩子嘛，即使什么都不做，都能很容易的讨人欢心。
<br/>　　“你……你想干……什么，有事……就冲我来，别伤……害我的……孩子。”
<br/>　　强忍着身体上的疼痛，丸手斋语气有些断断续续，他不知道这个家伙是谁，只是声音听起来有些熟悉。
<br/>　　好像……在哪里听到过的样子，但是现在他的脑子很乱，根本找不到一点头绪。
<br/>　　“喔，不伤害你的孩子……可以呀，只要你乖乖听话，你们都能活下去。”摘下头顶上的帽子，刘哔面带微笑的看着丸手斋。
<br/>　　他早已经被通缉，资料也进行过备份，根本没必要隐藏自己的身份。
<br/>　　况且，用自己身份做事的话，多少也是有些方便的。
<br/>　　“你是刘……修罗！”看清了刘哔的面孔，丸手斋有些吃惊：“怪不得……说吧，你想让我做些什么。”
<br/>　　怪不得，听他说话的时候，自己会感觉熟悉，之前讨伐古董咖啡厅的战役，就是他在会议上透露出来的消息。
<br/>　　在修罗出现之后，CCG也进行过几次抓捕计划，但是自修罗从库克利亚逃走以来，都没得到过这个喰种的任何资料。
<br/>　　一直到昨天讨伐月山家的时候，他才留下了一些模糊的信息。
<br/>　　针对这些线索，CCG自然展开了调查，然后抽调相当多的一部分高端战力，在美林大厦对修罗进行埋伏。
<br/>　　但最终还是失败了，非但没有成功抓捕刘哔，甚至连有马贵将，都永远的留在了那里。
