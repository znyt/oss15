夏夕绾拿出了陆寒霆给的黑金卡，这种卡大家都认识，财富的不二象征，私人订制。

<br/>        导购员迅速接下了卡，热情道，“可以可以的，小姐，请这边结账。”
<br/>
<br/>        夏夕绾走到收银台边去结账。
<br/>
<br/>        霍璇已经不可思议的张大了嘴巴，“夏夕绾那个土包子怎么会有这种黑金卡，而且我刚才还在黑金卡上看到了一个…陆字。”
<br/>
<br/>        霍璇对陆寒霆没什么了解，因为陆寒霆跟她不熟。
<br/>
<br/>        不过在海城，一看到这个“陆”字，自然就会想到只手遮天的陆寒霆。

<br/>        夏妍妍一直是娇美端庄的名媛模样，现在她的脸上终于出现了裂缝，霍璇能想到的，她早就想到了，姓陆还能拥有这张黑金卡的，一定是陆寒霆！
<br/>
<br/>        夏夕绾怎么会有陆寒霆的卡？
<br/>
<br/>        夏夕绾结了账，和叶翎大包拎小包，战果累累的出门了，这陆寒霆的名号还真是好用，不过就是一张卡，就能让夏妍妍和霍璇这上蹿下跳演戏的变成了哑巴。
<br/>
<br/>        夏夕绾走到夏妍妍的身边，突然停下了脚步，她勾唇一笑，轻声道，“玩暧昧算什么本事，有本事就转正啊。”
<br/>
<br/>        夏妍妍的脸色冷了下去，看着有些阴森可怕，“你认识陆寒霆？你跟陆寒霆是什么关系？”
<br/>
<br/>        “夏妍妍，你不是很聪明么，自己猜啊，男人将自己的卡交给女人刷，你说我们是什么关系？”
<br/>
<br/>        夏妍妍的双眼里流露出了跟李玉兰如出一辙的怨毒来，她讨厌夏夕绾，讨厌叶翎！
<br/>
<br/>        那时夏夕绾和叶翎并称南绾北翎，走到哪里都是瞩目的焦点，而她夏妍妍在她们身边就像是绿叶，哦不，准确的来说，就像是她们的贴身丫鬟。
<br/>
<br/>        夏夕绾满意的看着夏妍妍再也伪装不下去的脸庞，挑眉道，“是不是后悔了，当时你怎么就蠢到只要十二个亿呢，跟陆寒霆要一张黑金卡多好啊，真可惜，你是不会体会到这种随便刷卡的快感了。”
<br/>
<br/>        夏夕绾啧了一声，带着叶翎离开。
<br/>
<br/>        ……
<br/>
<br/>        叶翎不会回头看都能猜出夏妍妍的脸色有多难看，“绾绾，你太坏了，你又不告诉夏妍妍你和陆寒霆的关系，就让她在那里瞎猜，这挠心挠肺的滋味会将她折磨疯的。”
<br/>
<br/>        夏夕绾就要的这个效果，她太了解夏妍妍了，夏妍妍就像是一个躲在阴暗角落里的偷窥者，她的心理极度的扭曲和变态，特别渴望瞩目和虚荣。
<br/>
<br/>        现在将她拥有的东西一件一件的抢过来，再将她打回原形，夏妍妍肯定会疯的。
<br/>
<br/>        “翎翎，我们现在去哪里，已经买不少东西了。”
<br/>

<br/>
<br/>        夏夕绾一囧，刚才刷卡的时候她也是脑袋一热，现在想一想就觉得自己冲动消费了。

<br/>        收到消费短信的陆先生在想什么喵？

<br/>        叶翎将夏夕绾拉上前，“绾绾，一个会花钱的女人会让男人又爱又恨，陆先生财大气粗怕什么？”

<br/>        夏夕绾无法反驳。
