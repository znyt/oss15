　　“额。”萧逸摸了摸鼻子，略微有些尴尬。
<br/>　　“据我所知，紫炎好像并没有参加裂天剑派的考核。”秦飞扬面露惋惜。
<br/>　　林劲也道，“我在外历练时，在不少猎妖师口中都听过他的名字，风评不错。”
<br/>　　“有些猎妖师遇到危险，又恰好遇到他，他都会顺手帮一把。”
<br/>　　铁牛瓮声道，“我听说，现在有很多人在找他。百武城盛宝商行，暗影楼无忧公子，白家白冰雪等等。”
<br/>　　“不过，既然他没来，森林内可就没人压得住雷战了。”
<br/>　　“嗯。”秦飞扬面露凝重之色，道，“雷战是出了名的狠人，虽不至于心狠手辣，但为人性格暴躁，一言不合就是大打出手。”
<br/>　　“要是遇到他，我们赶紧跑。”
<br/>　　林劲点点头，目光灼灼地看向秦飞扬，道，“遇到雷战，秦飞扬你赶紧变雕驮我们走。”
<br/>　　秦飞扬面露怪异之色，大半个月来，这两人从未停止过说服他变雕。
<br/>　　......
<br/>　　时间一天天过去。
<br/>　　萧逸一行人这些天只进行了零星几场战斗，实在是没找到几个武者。
<br/>　　小型队伍们，都躲了起来；大型团伙，也同样如此。
<br/>　　双方都在等待最后一天，在这之前，谁都不会，也不希望有其它争斗，只想做好完全准备。
<br/>　　大型团伙的首领们，皆是北山榜上有名的天才，一个个声明在外，实力不凡，心思过人。
<br/>　　他们除了想得到前十的奖励外，更希望自己的手下也能通过考核。
<br/>　　这样，一旦他们成为裂天剑派弟子，手下，将会是自己日后的一大助力。
<br/>　　至于小型队伍们，也要在最后一天全力拼搏，以求得到更高的分数。
<br/>　　总之，每个人都有着自己的打算。
<br/>　　.....
<br/>　　十天时间，转瞬便过。
<br/>　　今日，便是考核的最后一天。
<br/>　　森林里的气息，陡然间变得诡异起来。静悄悄的，静谧得可怕。
<br/>　　鸟兽不飞，走兽不鸣，连风，似乎也悄然停顿。
<br/>　　森林最中央，十数个大型团伙，各自占据了一个方向，互相对视着，却又沉默着。
<br/>　　没人敢率先有动作，甚至不敢发出一丝声响，只得发出自身气势，互相做着试探。
<br/>　　数千武者的气势，快速凝聚出一股滔天气息。
<br/>　　便是这股气息，压得方圆十里内一切鸟禽走兽瑟瑟发抖，喘不过气来。
<br/>　　而在这十数大型团伙的最边缘，便是一支支小型队伍，相互聚拢着，怕被那些大型团伙攻击。
<br/>　　但又一个个都面露无比战意，渴望一场酣畅淋漓的战斗。
<br/>　　萧逸一行五人，赫然在其中。
<br/>　　肃杀而沉寂的气氛不断僵持着，渐渐让得在场武者开始坐不住了。
<br/>　　半晌，顾长风率先战了出来。
<br/>　　他一动，立刻让得其余数千武者齐刷刷地盯着他。
<br/>　　“诸位莫急。”顾长风高声道，“如今，森林内就剩下我们，最后之战，自然是要开始的。”
<br/>　　“只是，若是让得某些人浑水摸鱼，那便不好了。”
<br/>　　说着，他的目光看向了边缘处的小型队伍们，冷声道，“在我们火拼前，倒不如先将他们清理干净。”
<br/>　　同一时间，赵不群也战了出来，道，“长风公子说得不错，那些还留在这里的小型队伍，一个个可都分数不少阿。”
<br/>　　若是只有顾长风一人提议，其他团伙的首领必然不会有动作。
<br/>　　但赵不群也这样说，其他团伙便沉思了起来。
<br/>　　“我赞成。”其中一个团伙的首领站了出来。
<br/>　　“便听长风公子和不群少爷的。”
<br/>　　“长风公子和不群少爷所言极是。”
<br/>　　“....”
<br/>　　一个个首领都站了出来，表示同意。
<br/>　　北山榜排名14、15的两大天才同时发话，自然没有人反对。
<br/>　　而小型队伍这边，所有人瞬间脸色难看起来。
<br/>　　“卑鄙。”
<br/>　　“这群小人，联合起来了。”
<br/>　　“跟他们拼了。”
<br/>　　小型队伍中的武者，个个都是心高气傲之辈，露出了愤怒的脸色以及冲天的战意。
<br/>　　顾长风见状，冷笑一声，“哼，一群蝼蚁，还敢反抗，杀了他们。”
<br/>　　“好。”一众首领应答一声，两千多武者同时动了起来。
<br/>　　如今，还在森林内的武者，不足三千。
<br/>　　大型团伙这边占了两千多，小型队伍这边，不足千人，顶多八百左右。
<br/>　　从人数上看，小型队伍明显在下风。
<br/>　　但，那些甘愿加入大型团伙的，本就是靠自身实力无法通过考核，顶多只有先天四重、五重的人。
<br/>　　而小型队伍中，大部分都是实力不错，靠自己就有希望通过考核的。大多是先天六重，先天七重、八重也不在少数。
<br/>　　像之前被萧逸一行人打败的花剑何健、遁地鼠何苍、猎妖师王虎，个个都战力惊人。
<br/>　　他们中，有很多的修为都足以一开始就组建一支大型团伙，只不过不屑这样做罢了。
<br/>　　所以，小型队伍这边，普遍实力更高。
<br/>　　此战，孰胜孰败还不一定呢。
<br/>　　战斗瞬间打响，两方武者也已经互相接触。
<br/>　　数千人的混斗，场面澎湃而混乱。
<br/>　　十数个首领，没有出手，只是自信地看着。在他们看来，自己的手下足以靠人数优势完胜。
<br/>　　另一边，萧逸目光一凛，冷静地注视着，而后沉声道，“大型团伙那边，只有顾长风和赵不群是先天九重。”
<br/>　　“其他首领，皆是北山榜排名19之后的人物，清一色先天八重。”
<br/>　　“顾长风和赵不**给我，其余首领归你们。”
<br/>　　既然敌人数量占优，那边擒敌先擒王好了。
<br/>　　“好，除了顾长风与赵不群外，其他人，老子还不放在眼中。”秦飞扬自信地说着，整了整衣冠，一个帅气的飞跃，直朝其他首领而去。
<br/>　　“这个家伙，总爱出风头，不过好歹是北山榜16，自然不将其他首领放在眼中。”林劲笑笑，也找上了其他首领。
<br/>　　“萧逸，顾长风和赵不群就交给你了，好好教训这两个小人。”
<br/>　　不知何时起，花剑何健和遁地鼠何苍等人都站到了萧逸身后，说了一声，便瞬间出手。
<br/>　　花剑何健，北山榜17。
<br/>　　萧逸点点头，对身旁的铁牛和柳嫣然吩咐道，“嫣然，你战力不强，就站在原地用柳絮干扰敌人就好了。”
<br/>　　“铁牛，你防御力过人，保护好嫣然。”
<br/>　　“好。”铁牛认真地应答一声。
<br/>　　随着一个个实力强劲的武者出手，那些首领也不得不出手。
<br/>　　顾长风和赵不群见状，刚准备帮忙，一道冷傲的人影挡在了他们面前。
<br/>　　“你们的对手，是我。”萧逸冷冷说着，一把紫云精铁剑，横亘在两人面前。
<br/>　　剑在人在，那两人休想参与其他战斗。
