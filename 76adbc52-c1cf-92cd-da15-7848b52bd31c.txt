　　嘉乐今天心情很不错。
<br/>　　因为他那个脾气古怪的师父，马上就要带着货物回来了。
<br/>　　虽然师父在的时候，时常会出手教训他，但他还是很愿意和自己的师父待在一起。
<br/>　　他是个孤儿，是师父将他养大，对他来说，四目道长就像是父亲一般。
<br/>　　当然，最终要的是，师父将货物带回来，就代表着又有一大笔收入进账了。
<br/>　　随着那些拿着烧火棍的外国人闯进来，赶尸这个行当越来越不好干了，有时候一个月都凑不齐一支队伍。
<br/>　　不过还好，只要生活能维持下去，他就已经非常满足了。
<br/>　　至于以后的事情，他暂时也没考虑那么多。
<br/>　　肩上挑着两只水桶，嘉乐哼着不知名的小调，从山林中间穿插着。
<br/>　　家里没水了，他打算打一些回去，要不家里水缸空空的，有要挨师父骂了。
<br/>　　他还打算抓一些鱼，师父最喜欢的就是吃鱼了。
<br/>　　脸上洋溢着快乐的笑容，嘉乐从林中走出，看到了前方立在湖边的一道人影。
<br/>　　一身紧身的粗布麻衣，蓄着长发，身材有点瘦弱，像是一个落魄的书生，脚边有一群乌鸦环绕着。
<br/>　　这男子的面容棱角分明，很是陌生，他敢确定，之前自己绝对都没有见过，否则这么俊俏的一张脸，自己不可能没有印象。
<br/>　　难道是过路的行人，但又有谁会跑到这荒郊野地里面呢？
<br/>　　心中带着疑惑，嘉乐停住了口中的哨声，默默拉远了两人之间的距离。
<br/>　　他只是来打水的，这人是干什么的，和他没有半点关系。
<br/>　　只是有些可惜，没办法给师傅抓鱼吃了。
<br/>　　……
<br/>　　“你是嘉乐吧？”看着远处正在打水的青年，刘哔脸上带着微笑，走了上去。
<br/>　　这个山上总共只有这么几个人，四目道长他见过，老和尚不可能这么年轻，那这个青年的身份自然呼之欲出了。
<br/>　　正在给水桶舀水的动作一顿，嘉乐抬起头来，有些惊讶的看着刘哔。
<br/>　　这个男人，怎么会知道自己的名字，可是他明明不认识对方。
<br/>　　“我是九叔的徒弟，四目师叔和我提起过你。”似乎是看到了嘉乐脸上的疑惑，刘哔笑了笑，开口解释道。
<br/>　　恍然大悟的点了点头，嘉乐把手放在衣服上擦了擦，伸出手来，露出一口洁白的牙齿。
<br/>　　将视线放在嘉乐伸出来的手掌上，刘哔稍一犹豫，伸手碰触了一下，又迅速撤了回来，随即继续说道：“四目师叔在吗？九叔让我给他捎个信。”
<br/>　　“师父出去送货了，还不知道什么时候回来，你有什么事吗，我可以帮你转告给他。”嘉乐脸上笑嘻嘻的说道，却没有将实际情况告诉刘哔。
<br/>　　在没有确定来人身份之前，他觉得还是不要将师父的行踪给透露出来。
<br/>　　毕竟，身为赶尸人，还是有很多仇家的，难保不是哪家仇人捏造身份，打算上山来寻仇。
<br/>　　微微颔了颔首，刘哔笑道：“是九叔要找师叔，说让师叔有时间去找他一趟，既然师叔不再，你就帮我转告一下吧。”
<br/>　　说罢，刘哔向前走了几步，却引起嘉乐的一阵警觉，连忙后退。
<br/>　　看到嘉乐防备的样子，刘哔无奈的摊了摊手，告辞一声，从另一旁离开了。
<br/>　　看着其逐渐离去的身影，嘉乐脸上的笑容凝固了下来，他是一个细心的人，在之前刘哔的语气中，他听出了一些不对劲。
<br/>　　出门在外，哪有直呼自己师父名讳的，再说，这家伙出现的也太诡异了一些。
<br/>　　据师父出门前的交代，这次是要经过任家镇的，如果不出意外的话，自然也会去找九叔，而九叔没理由再叫一个弟子前来送信才对。
<br/>　　最重要的是，之前他刚来的时候，在刘哔脚边看到的那一群乌鸦，那种东西，只有在阴气过重的人身边逗留。
<br/>　　能引来这么多的乌鸦，除了僵尸和鬼怪之外，也只有鬼修才能做到了。
<br/>　　因为修为尚浅，嘉乐看不透刘哔的深浅，但好在，最终也没有发生什么冲突，这件事情，也只有师父回来的之后在做决定了。
<br/>　　心中思索着，嘉乐继续往木桶里舀着水，却突然发现，自己的食指不知道什么时候，被划开了一道细微的口子，一缕猩红的鲜血，从中流淌了出来。
<br/>　　这缕血液，顺着嘉乐的指尖，缓缓流进了木桶里，在清澈的水中，无比明显。
<br/>　　或许是被树枝不小心剐蹭到了吧，嘉乐有些埋怨自己的鲁莽，又将木桶里的水倒掉，重新装了两桶。
<br/>　　虽然只是一缕血，但如果被师父看到了，又免不了一顿臭骂，他可不想找这种不痛快。
<br/>　　……
<br/>　　隐藏在不远处的树后，刘哔静静的观察者湖边，之前他正准备动手的时候，突然又察觉到一股生人的气息，这股气息，还颇为强大。
<br/>　　计算了一下时间，他没办法在杀死嘉乐之后顺利隐匿，故此，也就没有贸然动手。
<br/>　　不过还好，他趁着之前握手的时候，在嘉乐手指上开了一道伤口，如果不出意外，用不了多久嘉乐就会变成丧尸。
<br/>　　杀掉嘉乐，这本来就是刘哔计划中的一部分。
<br/>　　在原著中，就是这个家伙提议将遮挡棺材的伞给撤走，才导致后来雨水冲刷掉了墨斗，里面的僵尸才有机会出来。
<br/>　　如果现在他就将嘉乐击杀的话，后面的剧情自然也就无法展开，而他的实验，也就能成功看到结果了。
<br/>　　但是距离剧情开始还有一段时间，嘉乐暂时也不会变成丧尸，现在刘哔也只能静静等待了。
<br/>　　果不其然，在他刚刚从湖边离开的时候，一旁的山路上就出现了一个光头，是那个筑基期的老和尚。
<br/>　　老和尚似乎也是来打水，慢悠悠的来到湖边，和嘉乐交谈了一会，两人拎着水桶返了回去。
<br/>　　因为距离有些远，刘哔并没有听到两人在说什么，不过现在他倒是可以顺着两人的脚步，找到四目道长居住的具体方位。
<br/>　　虽然这个山并不算很大，但是要让自己寻找的话，也是个十分麻烦的事情。
