　　萧逸足足远遁了半个时辰，才在妖兽森林某处停下来。
<br/>　　“噗。”刚从高空中落下，他猛地吐出一口腥血。
<br/>　　慕容克之前那一拳，显然让他受了重伤。
<br/>　　“半步破玄，果然厉害。”萧逸咬了咬牙，连忙原地打坐疗伤。
<br/>　　偌大个妖兽森林，任何人在其中，都如同沧海一粟。
<br/>　　任慕容克手段再高，也不可能在其中凭空找到人。
<br/>　　再者，他有四火傀儡球防身，也不必怕慕容克。
<br/>　　……
<br/>　　一日后，伤势无碍，他才在森林内饶了一段路，进入别的大城。
<br/>　　再通过别的大城，来到疾风猎妖队的总部。
<br/>　　疾风猎妖队，乃是东荒十八城最有名的猎妖队。
<br/>　　麾下足有十数支分队，自然有总部。
<br/>　　萧逸刚到来，门口两个守门的猎妖师已经认出了他。
<br/>　　“易霄大师，这边请。”两人恭敬地说道。
<br/>　　显然，宁浩料到他会来，提前吩咐了守门的猎妖师。
<br/>　　“嗯。”萧逸点点头，跟着两人来到内堂。
<br/>　　内堂中，宁浩以及他的几个副队长都在。
<br/>　　“易霄大师，来了。”宁浩热情地招呼道。
<br/>　　“请上座，来人，上茶。”
<br/>　　“不必了。”萧逸摇摇头，道，“昨日，为何助我？”
<br/>　　“助你？那是救你一命好吧，你显然不是慕容克的对手。”
<br/>　　宁浩还未说话，那几个副队长已然面露敌意地看着萧逸，不满地说着。
<br/>　　“放肆，何时轮到你们这样与易霄大师说话。”宁浩呵斥一声。
<br/>　　“救我？”萧逸嘴角微微上扬，淡淡道，“随你们怎么说吧。”
<br/>　　“我来，只是不想欠你们人情。”萧逸说着，随手拿出几个玉瓶，道，“这里有些巨力丹、隐气丹和驱兽丹。”
<br/>　　“都是些猎妖师在外必备的丹药。”
<br/>　　萧逸放下玉瓶，便要离开。
<br/>　　“易霄大师，慢。”宁浩连忙推辞道，“昨日相助，并非图易霄大师的回报，这些丹药，还是收回去吧。”
<br/>　　旁边一副队长不屑道，“队长，让他走吧。不过是些寻常丹药，还真把自己当回事了。”
<br/>　　“巨力丹什么的，我们疾风猎妖队多的是。”
<br/>　　疾风猎妖队，其实已经可以算是一个大势力了。
<br/>　　麾下逾千猎妖师，自然是身家丰厚，而且也有大量如巨力丹、驱兽丹等猎妖师在外必备的丹药。
<br/>　　“高石，再敢放半个屁，立刻给我滚出去。”宁浩冷冷地剐了眼那副队长，满脸冷意，显然是动了真怒。
<br/>　　萧逸没有理会宁浩，而是看向那高石，淡淡道，“哦，这些丹药你们多的是？”
<br/>　　说着，萧逸打开了玉瓶的盖子，霎那间，一阵浓郁的丹香传来。
<br/>　　“好浓郁的丹香。”宁浩一惊，连忙看了一眼，惊道，“四品级别的巨力丹，而且还是完美级别的，服之可马上生效。”
<br/>　　“易霄大师，果然手段过人。”宁浩惊讶道。
<br/>　　“四品级别？还是完美丹药？”高石等副队长也是一惊。
<br/>　　他们所谓的有大把这些丹药，几乎都是二品和三品的。
<br/>　　四品的巨力丹，价格高昂，也就只有几个副队长用得起，而且还是劣质的。
<br/>　　萧逸没有理会他们的惊讶，淡淡道，“丹药我便留下了，告辞。”
<br/>　　“易霄大师，慢。”宁浩连忙道。
<br/>　　“宁浩队长，还有何事？”萧逸皱了皱眉头。
<br/>　　“易霄大师不要误会。”宁浩拱了拱手，道，“之前你救了内人一命，宁某还未回报。”
<br/>　　“不必了。”萧逸摇摇头，“那是任务，猎妖殿也已经把赏金给我了。”
<br/>　　宁浩摇摇头，认真道，“对于易霄大师你来说，那是任务。可对于我宁浩来说，那是救了我妻子一命。”
<br/>　　“这些丹药，易霄大师便收回去吧。昨日相助，就当是我对您的回报。”
<br/>　　“队长。”高石急道，“这些丹药，可不是普通货色阿，完美级别，能卖不少钱呢。他让我们损失这么大…”
<br/>　　“闭嘴。”宁浩冷声打断道，“出去。”
<br/>　　“我…”高石脸上一阵委屈。
<br/>　　萧逸皱了皱眉头，问道，“宁浩队长，易某让你们损失了什么吗？”
<br/>　　“没什么。”宁浩摆摆手，不以为意道，“不过是慕容克那小人，觉得我们昨日是成心与他作对。所以盛宝商行断了与我们的生意来往罢了。”
<br/>　　“还不滚出去？”宁浩说罢，冷冷地看了高石一眼。
<br/>　　高石脸上的委屈之色更浓。
<br/>　　但显然，宁浩在这里极有威信，高石还是乖乖地挪动了脚步。
<br/>　　“生意？”这时，萧逸淡淡地问了一句。
<br/>　　高石差不多走到门口了，听到萧逸的话后，愤愤地转过身，道，“我们猎妖师，天天猎杀妖兽，自然是靠贩卖妖兽的东西为生。”
<br/>　　“疾风猎妖队，逾千号兄弟，可就靠…”
<br/>　　高石话还没说完，已经被宁浩愤怒地一脚踹了出去。
<br/>　　宁浩面露愧疚之色，道，“易霄大师，我这兄弟是个愣头青，快人快语，但他没有恶意的，还请不要放在心上。”
<br/>　　“没事。”萧逸摇摇头，算是明白过来了，为何这几个副队长对他充满敌意。
<br/>　　猎妖师这个职业，充满了危险。
<br/>　　在妖兽森林内，拿命拼，每日都在刀口下讨生活。
<br/>　　而盛宝商行，作为最大的商行，自然是大部分猎妖师的交易对象。
<br/>　　若无法交易，猎妖师们，得来的妖兽内丹、精血、皮毛、血肉等等，根本就无处贩卖。
<br/>　　也就是说，这简直断了他们的生计。
<br/>　　当然，也可想而知，盛宝商行有着何等的力量。
<br/>　　连疾风猎妖队这样的大势力，慕容克不过是一句话的事，就能让他们陷入如此麻烦的地步。
<br/>　　“妖兽内丹和精血，盛宝商行不与你们交易，易某一并给你们收购了。”萧逸忽然说道。
<br/>　　妖兽，最值钱的还是内丹和精血，至于皮毛和血肉，并不怎么值钱。
<br/>　　宁浩一愣，拒绝道，“易霄大师不必如此，就算盛宝商行不与我们交易。但，东荒十八城，还是有许多本土商行的。”
<br/>　　“有不少商行，还与我交情不错，虽然有些麻烦，但要找到买家并不难。”
<br/>　　显然，宁浩误会了。
<br/>　　“宁浩队长误会了。”萧逸认真地说道，“易某是真的需要妖兽内丹和精血，有多少要多少。”
<br/>　　“真的需要，还是有多少要多少？”宁浩狐疑地看着萧逸。
<br/>　　半晌，他还是摇摇头，傲然道，“易霄大师的好意，在下心领了。盛宝商行虽然势力极大，但还做不到一手遮天。”
<br/>　　“这点麻烦，我们自己能解决，就不劳烦易霄大师你了。”
<br/>　　萧逸苦笑一声，这宁浩怎么就不信他呢。
