　　萧逸瞬间打了个冷颤。
<br/>　　人之所以为人，而不是行尸走肉，便是因为人有思想，有记忆。
<br/>　　萧逸脑海中的记忆，便是他知晓自己是萧逸的一份凭证。
<br/>　　举个例子，如果一个人失忆了，人还是那个人，可这个人还会按照原本的自己去‘活着’吗？
<br/>　　如果自己真的将这位斩星前辈的一生经历都经历一遍。
<br/>　　如此悠长的岁月经历。
<br/>　　对比自己区区20来年的经历，就算算上上一辈子活过的岁月，也不过寥寥数十年。
<br/>　　二者根本不是一个层次。
<br/>　　那么到时候，会是哪一份记忆当主导地位？
<br/>　　以前，他也吸收过一些残损记忆。
<br/>　　如冰尊者的记忆，在风絮秘境内的那些灵识的记忆。
<br/>　　但那些记忆终究只是出现于脑海中。
<br/>　　对比自己亲自去经历一番这漫长岁月，自己亲自去走一遭这一段人生之路，这可根本是两回事。
<br/>　　到时候，萧逸会认为自己是萧逸，还是这位斩星前辈？
<br/>　　萧逸皱着眉，迟疑着。
<br/>　　剑灵的声音，再次响起。
<br/>　　“小子，我便直言告诉你。”
<br/>　　“这位斩星前辈，是这数万剑碑主人中，最古老的一位。”
<br/>　　“除却剑帝，没有任何一位剑修能比他更古老。”
<br/>　　萧逸闻言，脸色变得更加难看。
<br/>　　他当然知晓这里的剑碑主人，生前都活了极长的岁月。
<br/>　　云渊前辈二人曾说过，以他们的修为，都能随便活个万年。
<br/>　　这里的剑碑主人，随便一人生前的修为境界，都比云渊长老二人更强，寿命自然也更长。
<br/>　　而且，这万剑碑林，也不知存在了多少岁月了。
<br/>　　怕是上古时期就已存在。
<br/>　　“咕。”萧逸不禁咽了口口水。
<br/>　　“怕了？”剑灵阴测测的声音，再次响起。
<br/>　　萧逸皱了皱眉，道，“这个剑帝碑空间，只会存在半月时间。”
<br/>　　“半月时间，如何能真正走完这些前辈的一生悠长岁月和经历？”
<br/>　　“笨蛋。”剑灵轻蔑一笑。
<br/>　　“你看看你两边的剑气。”
<br/>　　萧逸看了眼。
<br/>　　这条剑气之路，脚下、左右两边，还有上方，尽皆被滔天剑气所覆盖。
<br/>　　剑气，隐隐有惊人威能。
<br/>　　剑灵继续道，“这条剑气之路，在剑帝碑空间的力量之下。”
<br/>　　“参悟者走在其中，可改变岁月流逝。”
<br/>　　“特别是你参悟的过程中，若你参悟的速度越快，那么时间的流逝就会越慢。”
<br/>　　“嘶。”萧逸倒吸一口凉气。
<br/>　　“剑帝碑的力量，可以操控时间空间？”
<br/>　　“有何不能。”剑灵霎时傲然道，“剑帝可是剑道一帝。”
<br/>　　“虽未成武神，也没有武神那般凌驾天地的实力。”
<br/>　　“但操纵天地阴阳，打破天地桎梏，却是轻而易举。”
<br/>　　“这点区区操控时间流逝，算得了什么。”
<br/>　　“嗯？”萧逸忽然发现，剑灵的声音，有些莫名。
<br/>　　不，准确来说，是剑灵的反应，有些奇怪。
<br/>　　剑灵，这时也发现了萧逸的轻咦，霎时收敛了语气。
<br/>　　半晌，剑灵继续道，“你在这剑气之路上，看似只过了半个月；但在参悟之中，则可能已过万载岁月。”
<br/>　　“我发现你知道的挺多。”萧逸双眼一眯。
<br/>　　剑灵语气一滞，半晌，桀桀一笑，“老夫活的岁月，同样很久。”
<br/>　　萧逸耸耸肩，“就不劳烦你这老怪物提醒了。”
<br/>　　萧逸看了眼另一边，丁秋月正闭目缓慢走着她的那条剑气之路。
<br/>　　“一个小丫头尚且敢走，我萧逸，难道怕了？”
<br/>　　说罢，萧逸再次闭上了眼睛，开始参悟。
<br/>　　耳边，传来剑灵的一声呢喃，“固守剑心，勿忘初心。”
<br/>　　冷焱剑，已被萧逸收回乾坤戒中。
<br/>　　萧逸的参悟之路，再次开始。
<br/>　　渐渐的，萧逸开始明白这条剑气之路的意义了。
<br/>　　与其说剑气之路，不如说是斩星前辈自己走过的剑道之路。
<br/>　　萧逸每多感悟一分，脚步便会自动前进一步。
<br/>　　也不知过了多久。
<br/>　　萧逸渐渐感悟到斩星前辈天元境修为时的层次。
<br/>　　而这期间，斩星前辈所经历过的战斗，一些难忘之事，萧逸尽皆亲身感受过一遍。
<br/>　　包括斩星前辈这个阶段所得到的功法、武技，还有修习过程等等。
<br/>　　时间，渐渐过去。
<br/>　　萧逸来到了斩星前辈地极境修为的层次。
<br/>　　真正的一飞冲天，从这里开始。
<br/>　　斩星剑道的一丝感悟，也是这时起被斩星前辈悟得。
<br/>　　在感悟状态中，萧逸的身份，便是‘斩星前辈’，故也同样参悟起这一丝剑道来。
<br/>　　……
<br/>　　时间飞逝。
<br/>　　不知过去了多久。
<br/>　　萧逸忽然睁开了眼睛。
<br/>　　“呼。”萧逸深呼吸一口气，转过头，看了眼身后。
<br/>　　自己在这条剑气之路上，已经走了不短的距离了。
<br/>　　萧逸闭目感知了一番，“才过去了一天时间。”
<br/>　　不错，他才在这条剑气之路上，走了一天。
<br/>　　但在参悟状态中，却已走了30年之久。
<br/>　　参悟状态中，‘斩星前辈’已30岁，便也是在这个岁数，他踏入了绝世强者的层次。
<br/>　　“接下来，便是绝世强者层次的经历了。”
<br/>　　事实上，虽然是走一遍斩星前辈的经历，但却并非是一生事无巨细都经历。
<br/>　　真正经历的，只是所有战斗场面、参悟场面，以及一些难忘的事情。
<br/>　　至于其他的寻常生活和事情，则无需经历。
<br/>　　所以说，斩星前辈这一生的经历中，萧逸所走过的岁月，缩水大半都不止。
<br/>　　当然了，即便缩水大半，那同样是一段极其漫长的岁月。
<br/>　　时间再次飞跃。
<br/>　　萧逸在参悟状态中，已过千年。
<br/>　　这期间经历的战斗、感悟等等画面，早已过数万之数。
<br/>　　这个时期的斩星前辈，修为滔天，早已远远跨过了绝世强者。
<br/>　　但之后的境界到底是什么，萧逸却不得而知。
<br/>　　只知，现今出现的战斗画面，感悟画面，变得异常奇怪。
<br/>　　萧逸闭目参悟中。
<br/>　　周遭，是一片辽阔的战场。
<br/>　　战场之内，此时充斥着惊人血腥味。
<br/>　　周遭，尸体遍地，怕是布下数十万之数。
<br/>　　有人类的，有…妖兽…
<br/>　　“妖兽？”萧逸皱了皱眉。
<br/>　　这些妖兽，他所能认出的，寥寥无几。
<br/>　　但其气息，却让他心悸。
<br/>　　心头，蓦地涌现一股热血。
<br/>　　“战个痛快。”
