　　因为叶少阳这次去是找人的，而且还有毛小方陪着，翠云比较放心，也不想当他的累赘，于是比较爽快地答应下来，把叶少阳的金条拿出来，给叶少阳一根，让他带着路上用，其余的自己继续放起来，帮他藏好。然后给他收拾了行李，带了一些干粮、自己亲手纳的两双平底鞋，还有几件衣服等等，一并给他装起来，好生嘱咐了一番。
<br/>　　过了一会，陈三也上楼来，跟叶少阳结钱和告别……这一路上赶尸，现在赚钱了，当然要分一份给他，更不用说对付那些土匪和人形煞，叶少阳等于是救了陈三一命。
<br/>　　那几块金条失而复得之后，叶少阳现在颇有点资产，自然不会再要赶尸的钱，再说自己当初拿了张道长五十块大洋，帮他赶尸也算是还情，钱肯定是不要了。
<br/>　　叶少阳推脱了半天，看陈三过意不去，只好象征性地拿了一点。陈三表示这边事情办完，自己要回义庄去了，就此告别。
<br/>　　“叶天师，我看的出，你是人中之龙，我只是个普通人，不求结交，也谈不上报恩了，不过将来你叶天师但凡有用得着的地方，只管去义庄找我，我这条命，随时为你准备着！”
<br/>　　分别在即，陈三也很有点动情，抱着拳头对叶少阳说道。
<br/>　　叶少阳也很感动，拍了拍他的肩膀，说了些“青山绿水”之类的话，然后跟翠云一起送陈三出门，等了没一会，毛小方也到了，牵了两匹马过来，叶少阳跟翠云依依不舍道别，然后跟毛小方一人牵着一匹马，朝东北方离去。
<br/>　　出城之后，一路上都算太平，两人又骑着马，速度还算挺快的，也比较悠闲，只是叶少阳每每想到自己跟传说中的毛小方结伴同行，内心就忍不住有一种十分特别的感觉。想着将来回到自己的时代，见到朋友们之后，一定要将这段经历跟大伙面前吹吹牛比。
<br/>　　自己一定是可以回去的。
<br/>　　对这一点，叶少阳深信不疑。
<br/>　　两人一路上闲聊各种，讨论道法，情谊在不知不觉之间产生。在叶少阳看来，这个毛小方跟电视剧里演的还真的挺像的，不太爱说话，但是说起话来也很诙谐。性格耿直，做事情很认真，只是没有电视里演的那样抠门，路上住宿吃饭之类的，也主动掏钱。
<br/>　　不过叶少阳自己现在是土豪，也就没让他请客。
<br/>　　几天之后，两人来到了鹰潭，龙虎山脚下。
<br/>　　龙虎山矗立在信江边上，山峦叠起，风景秀美，是道家七十二福地之首，风水还在茅山之上，包括位于附近的三清山和武夷山，是一个绝佳的风水盘，龙虎山就在这个风水盘中最好的位置。
<br/>　　张道陵曾在这里修行得道，创立正一道宗，龙虎山于是成为道教的发源地，被尊称为“道都”。传闻这一带附近山上曾经有一共六十四座道观，但是很多都毁在了时间的长河中，只剩下天师宫一座，坐落在龙虎山的主峰天门峰的山顶。
<br/>　　一片祥云之中，天师宫依稀在目。
<br/>　　叶少阳对这座宫殿并不陌生，他小到大也来过好几回，仔细观察的眼前的景致，跟一百年后好像也差不多，只不过树木更茂盛一点，最主要的是，这里还没有成为景区，前山没有那些供人上香和游玩而修建的景区，寥寥的山路上，走着几个道士。
<br/>　　毛小方还是第一次来到龙虎山，对眼前的一切赞不绝口。“好一块灵气充盈的宝地，不愧是道教第一福地！”
<br/>　　两人一路上山，来到山门前，叶少阳对守山的两位弟子通报了姓名，那两弟子听到天道宗还好说，听说叶少阳是茅山弟子，有些吃惊，好奇地打量了他几眼，行礼说道：“二位道友，是要挂单还是拜谒掌教？”
<br/>　　“不挂单，也不拜谒掌教，我是来找一个人，道渊真人。”
<br/>　　“道渊？”两个弟子顿时就愣住了，上下打量他，“这位道友，你找道渊做什么？”
<br/>　　“我……是他一个朋友，正好路过这里，想要跟他见面叙叙旧，仅此而已。”叶少阳撒了个谎。
<br/>　　一位道童说道：“不瞒你说，道渊因为犯禁，被罚面壁思过，这件事你可知？”
<br/>　　叶少阳点头说道：“听说了，所以来看看他，面壁思过是不能见客？”
<br/>　　道童道：“那也没有这个规矩。只是道渊不许出山，只是……实话说吧，他虽是天师后人，但因为犯禁，门中弟子都懒得跟他接触，再说路远，我们也没兴趣去通报，你们若想找他，直接去便是，他在左边那座山峰上，你们伤到山顶就能找到他了。”
<br/>　　叶少阳谢过，带着毛小方一起过去，穿过山门，也不上主峰，直接奔着道童说的山峰去了。
<br/>　　“这位道渊真人，到底犯了什么错，被罚面壁思过？”毛小方忍不住小声问道。
<br/>　　“我是真不知道，只有见面再问问他了。”
<br/>　　一路走去，风景很不错。毛小方在欣赏着风景，感受仙家名山大川的美好景致，叶少阳却没有这个心情，一路上都在想，见到道渊之后要怎么说才能让他相信自己，想来想去，还是实话实说的好，因为所有一切都建立在让道渊相信自己的基础上。
<br/>　　山顶上，有一座岩石，如同天外飞来，屹立在山巅，石上镌刻着“清风”二字，岩石下面有一方泉眼，被砌成了井口的形状，一道山泉，顺着山势流下去，周围绿树环绕，虽然没有什么壮观的风景，不过看上去也自有一种怡然的趣味，是个清修的所在。
<br/>　　两人上到山顶之后，都有一种心神为之宁静的感觉。
<br/>　　“果然是面壁思过的地方啊。”毛小方指着岩石说道，“这里有石壁，坐在岩石下，在宁静中一览群山，的确可以想清楚很多事情。”
<br/>　　再往上走，来到悬崖边上，才看到在悬崖的另一边，有一座四面通风的小亭子，半边都悬在悬崖外边，也不知道当时是怎么修出来的。
