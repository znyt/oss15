　　“中品圣器？”
<br/>　　周遭观赛席上，爆发出一声声惊呼。
<br/>　　一道道灼热的目光，纷纷聚集在萧逸手中剑上。
<br/>　　那把散发着冷白剑芒的锋锐之剑，显然是一把强大的中品圣器。
<br/>　　中品圣器，那是连等闲武道皇者都无法拥有的重宝。
<br/>　　更别说现在这是一把剑类圣器，而且观其气息，明显比一般的中品圣器要强得多。
<br/>　　比武台上，萧逸淡漠握剑，并未理会周遭惊呼。
<br/>　　淡漠的目光，看向比武台边缘的三位裁判。
<br/>　　三位裁判点了点头，闪身而来。
<br/>　　看了眼已跌落比武台的石惊涛，高声道，“此战，黑云学教，萧逸胜。”
<br/>　　萧逸微微点头，收回了剑，转身离去。
<br/>　　他之所以拔剑，是因为这石惊涛修的是山石一道。
<br/>　　山石一道，本就极擅防御。
<br/>　　五痴学宫，位列五大学宫之一，论排名，也仅次于天藏学宫。
<br/>　　其内弟子，个个天骄。
<br/>　　石惊涛，又是五痴学宫力痴弟子中，仅次于力痴首席的强者。
<br/>　　一身修为，达到圣王境八重。
<br/>　　也就是说，此人不仅力大无穷，而且防御力惊人。
<br/>　　若他不使剑或者武技，与此人慢慢缠战，还是比较费劲的事。
<br/>　　毕竟，除却一身手段，他的真正修为只有圣王境六重。
<br/>　　另外，费劲归费劲，可与此人战斗，若说能有收获的话，倒也没有。
<br/>　　故萧逸便不浪费时间了，直接拔剑。
<br/>　　而他手中趁手的剑，又只有两把，一是暴雪剑，二是冷焱剑。
<br/>　　暴雪剑，不过是极品元器，连亚圣器都比不上。
<br/>　　对现在萧逸的实力而言，暴雪剑已几乎无用，便也只能拿出冷焱剑。
<br/>　　而冷焱剑之前吸收了大量血气之力后，强了不少。
<br/>　　故萧逸仅仅是一道剑气，便将这石惊涛轰出了比武台。
<br/>　　根据规则，对手落台，直接落败，三位裁判便也直接宣布了胜负。
<br/>　　萧逸转身离去。
<br/>　　身后，却忽然爆发出一声惊呼。
<br/>　　“我想起来了，这小子不是数天前在五灵峰首上一剑重伤了金虎师兄的家伙？”
<br/>　　当日五灵峰首上，毕竟还有不少学院队伍未曾感到天藏学宫。
<br/>　　而即便那时在场的弟子，恐怕也大多只是将心思和目光放到金辰学宫弟子，还有楚柔等人身上。
<br/>　　压根儿没有多少人理会萧逸和青麟二人。
<br/>　　即便最后萧逸忽然重伤了金虎，引起注意。
<br/>　　可那些注意，也仅仅是放到了萧逸手中之剑上罢了。
<br/>　　自然的，直到现在，才渐渐有观赛弟子认出了萧逸。
<br/>　　这时，又是一声惊呼响起。
<br/>　　“我也想起来了，此子不正是两年前天藏学宫的弃徒？”
<br/>　　“当日，他也同样是手持此剑，轻松败了通天峰座下亲传八弟子王越。”
<br/>　　“这家伙，别的不行，运气倒是不错，得了把中品圣器。”
<br/>　　“呵，有中品圣器在手，倒是实力惊人。”
<br/>　　一道道议论声，在观赛席上响起。
<br/>　　萧逸充耳不闻，自顾走回黑云学教席位。
<br/>　　比武台下，石惊涛堪堪从跌落失败中反应过来。
<br/>　　冰冷的目光，直视萧逸。
<br/>　　“黑云学教，靠着中品圣器之威败我，算什么本事？”石惊涛愤怒吼着。
<br/>　　萧逸并未理会，已然回到席位上，淡淡坐下，假寐休息。
<br/>　　周遭观赛席，仍旧议论纷纷，不过更多的，是关于萧逸手中的中品圣器。
<br/>　　但这些议论，很快便随着三位裁判的高喝声而消散。
<br/>　　“57号，五痴学宫，令狐忘，上台。”
<br/>　　“58号，千川学宫，唐千流，上台。”
<br/>　　嗖…嗖…
<br/>　　两道身影，瞬间跃上比武台。
<br/>　　“是令狐忘师兄。”周遭观赛席上，陡然爆发出惊呼。
<br/>　　而且，这种惊呼，比之前的任何一道都要猛烈得多。
<br/>　　一道道带着崇拜的目光，几乎悉数集中在此刻比武台上的一道淡然身影中。
<br/>　　令狐忘，五痴学宫弟子，剑痴弟子首席，亦是五痴首席。
<br/>　　这时，黑云学教席位上，一直脸色淡然的莫悠，陡然双眼一眯。
<br/>　　“剑痴令狐忘。”莫悠自语了一声，眼中一道战意闪过。
<br/>　　比武台上，令狐忘并未理会周遭惊呼，而是转过头，直视莫悠。
<br/>　　“忘忧剑，莫悠。”令狐忘说了一声，眼中同样一道战意闪过。
<br/>　　“听说你挑了西部几乎所有顶尖天骄，连金尘都败于你手中。”
<br/>　　“后面的比拼，若有机会的话，我期待与你一战。”
<br/>　　“可以。”莫悠点了点头。
<br/>　　令狐忘的目光，从莫悠身上移过，看向了一旁正在假寐的萧逸。
<br/>　　“你的修为尚弱，还须勤加修炼才是。”
<br/>　　“你既是剑修，应当知晓剑即是命。”
<br/>　　“你的剑，很强，希望你不要让它蒙羞。”
<br/>　　萧逸假寐着，不语。
<br/>　　一旁青麟倒是撇撇嘴，“先管好你自己吧，希望你只是个剑痴，别是个傻子。”
<br/>　　“嗯？”令狐忘看向青麟，眼中一道锐利的目光，直视青麟。
<br/>　　“嘶。”青麟的目光与之对视，竟是眼眸顿时一阵刺痛。
<br/>　　一旁萧逸，霎时睁开了眼睛，冷漠的目光，直视令狐忘。
<br/>　　冷漠的目光注视下，令狐忘脸色微变，不得不收回了直视青麟的目光，转而与萧逸对视。
<br/>　　两道目光，在空气中碰撞，一道锐利，一道冷漠，竟是互不相让。
<br/>　　萧逸淡漠的话语，缓缓响起，“我想，在你胜我之前，还没那个资格教我如何待剑。”
<br/>　　“起码，一个圣王境巅峰，还不够格。”
<br/>　　不错，令狐忘乃是圣王境巅峰修为，距离圣皇境，仅差一步。
<br/>　　“很好。”令狐忘轻淡一笑，收回了目光。
<br/>　　比武台上的战斗，不多时便打响。
<br/>　　黑云学教席位上，萧逸看了青麟一眼，“没事吧？”
<br/>　　“没。”青麟摇摇头，眨了眨有些刺痛的眼睛。
<br/>　　“这家伙，很强。”
<br/>　　一旁莫悠说道，“两位师弟不要介意，这令狐忘是个剑痴，爱剑如命，这才多说了几句罢了。”
<br/>　　“这等人物，还不至于有恶意。”
<br/>　　“嗯。”青麟点点头，“不过，也正是个剑痴，萧逸师弟你若是之后的比拼上遇到这家伙，小心点才是。”
<br/>　　“呵。”萧逸淡淡一笑，不置可否。
