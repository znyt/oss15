夏夕绾想了一下，“我也不知道我要找什么书，用什么药，我还在思考阶段，所以双双不用你帮忙了，我在这里泡几天好好考虑一下，如果需要你帮忙我一定不会客气的，你快点去照顾你的花花草草吧。”
<br/>
<br/>        双双就觉得夏夕绾有点神秘兮兮的，果然是学霸，套路都跟别人不一样，枢密这个藏书阁跟殿堂一样庞大，她每次走到这里都想睡觉。
<br/>
<br/>        这时双双眼尖，突然指着夏夕绾的脖子，“绾绾，你脖子里是什么痕迹啊，谁咬你的咬这么狠？”
<br/>
<br/>        夏夕绾为了遮身上的伤今天穿了高领，将自己遮的很严实，但是无奈双双眼太尖了，看到了。
<br/>
<br/>        夏夕绾迅速用手遮住了自己的衣领，否认道，“双双，你不要乱说，我只是昨晚被…蚊子咬了！”
<br/>
<br/>        双双将夏夕绾拉到了一个无人的角落里，小声道，“绾绾，听说你嫁给了幽兰苑那个鬼夫，你实话告诉我，那个鬼夫是不是心理变态，什么蚊子咬的，这明明就是人咬的，牙齿都咬进你的肉里去了，这是要吸血啊。”
<br/>
<br/>        夏夕绾勾了一下红唇，将手里的医书放在了书架上，“什么吸血啊，又不是吸血鬼，双双你不要吓人了！”
<br/>
<br/>        “绾绾，我跟你说认真的，你有没有在听啊，像这些人都不是正常人，他们心里有病的，而且是精神方面的疾病，根本就治不好的！”
<br/>
<br/>        “我以前有一个堂姐夫，堂姐夫人很好，对我堂姐也百般体贴和呵护，但是他就有一个毛病，喜欢家暴。”
<br/>
<br/>        “那个堂姐夫发病的时候，将我堂姐绑在床上虐待，我堂姐越痛苦，他就越开心，他还喜欢咬人，嗜血，一闻到血腥味就亢奋，变得疯狂又恐怖。”
<br/>
<br/>        “我堂姐是很爱那个堂姐夫的，本来想咬咬牙过下去的，但是我堂姐怀孕了，被堂姐夫给打没了，而且就算生下了孩子，让孩子在家暴的阴影里长大么？”
<br/>
<br/>        “绾绾，我说这么多的意思就是，你现在还年轻，又这么优秀，璀璨似锦的人生刚开始，你赶紧想办法跟幽兰苑那个鬼夫离婚，抽身而出，外面优秀的男人太多了，以后你会遇到比他更好的。”
<br/>
<br/>        夏夕绾纤长的羽捷垂落了下来，几秒后拉住了双双的小手，“双双，谢谢你跟我说这么多的知心话，我知道我自己想要什么，所以不用替我担心。”
<br/>
<br/>        “那就好，绾绾你有什么困难就跟我说哈，我看你脸色挺白的，我现在就去厨房那里让他们给你中午炖个汤补补。”
<br/>
<br/>        “双双，你跟厨房的人很熟啊？”
<br/>
<br/>        双双讪讪一笑，转身跑了，“吃货嘛~”
<br/>
<br/>        双双走了，夏夕绾贴着墙壁缓缓坐在了松软的地毯上，双目放空了一会儿，她拿出了自己的手机，陆先生应该醒了吧…
<br/>
<br/>        ……
<br/>
<br/>        幽兰苑。
<br/>
<br/>        床上的陆寒霆缓缓睁开了眼，此时绚烂的晨曦已经透过层层窗幔镀了进来，他第一次起这么晚。
<br/>
<br/>        眸里染着惺忪的睡意，他翻了一个身去搂身边的女孩，想再睡一会儿。
<br/>
<br/>        完全不想起。
<br/>
<br/>        但是，身边空空如也。
