        以前私宅有安防系统保护着，没有真人保镖，但是现在有了，并且，她童筱颖还被禁足了。
<br/>
<br/>        她只能在私宅的范围内活动，只要想离开私宅的院门，她就会被保镖拿枪指着逼回屋内。
<br/>
<br/>        不过，他连华生和刘婶倒是可以自由进出。
<br/>
<br/>        由此可见，那些保镖是针对她童筱颖的。
<br/>
<br/>        “你离开私宅做什么？等着被那个冒险者集团的人抓回去审讯啊？”连华生嗤之以鼻。
<br/>
<br/>        童筱颖终于默不作声了。
<br/>
<br/>        连华生瞥了她一眼，转开了话题：“我明天给你带几盒叶酸回来，怀孕初期，吃叶酸对你和肚子里的那个未知数都好。反正，哪怕没怀孕也能吃。”
<br/>
<br/>        “谢谢。”童筱颖谢道。
<br/>
<br/>        连华生忍不住多嘴了一句，吐槽起来：“就目前这种情况，你待在私宅里是最安全的。毕竟，两边的人都在找你，阿沉把你‘藏’起来，已经挺不容易了。”
<br/>
<br/>        “两边的人都在找我？”童筱颖顿时感到诧异。
<br/>
<br/>        连华生一个回神，才意识到自己说漏了嘴。
<br/>
<br/>        “你的话是什么意思？”童筱颖凝眉追问。
<br/>
<br/>        连华生连忙敷衍道：“就是你跟那个十爷扯上了关系，我上头怀疑你，暗道上的那个集团也在查你。”
<br/>
<br/>        “你撒谎。”童筱颖淡定道。
<br/>
<br/>        连华生怔了一下，一时间没接上话。
<br/>
<br/>        童筱颖接着说道：“我是不是，也跟你们明暗两道上，需要找的那个‘药’有关？还有，我外婆的那本‘蛊札’，你们没找到，所以，都觉得，‘蛊札’在我身上，只是我不记得了。”
<br/>
<br/>        “我说，你这个女人，是不是太敏感了点？”连华生心虚地啧啧嘴。
<br/>
<br/>        童筱颖垂下眼帘：“你和大哥都有事瞒着我，所以，我只能在这胡思乱想。”
<br/>
<br/>        “我们的事，你知道得越少，你就越安全。懂吗？”连华生咧嘴一笑。
<br/>
<br/>        童筱颖不再追根究底了。
<br/>
<br/>        三天后，她仍旧没有见到傅夜沉，就连他的手机也处于关机状态。
<br/>
<br/>        他每次都是这样，总会无缘无故地联系不上人。
<br/>
<br/>        因为生理期没来，童筱颖怀疑自己有可能是怀上了，于是拜托连华生给自己买验孕棒回来。
<br/>
<br/>        其实，买三支就行了，结果连华生带回来了三十支，而且还是三十中不同牌子的。
<br/>
<br/>        童筱颖面对连华生的这种“夸张”行为，无言以对。
<br/>
<br/>        不过，当她试了三支，发现自己都没有怀孕后，第二天早上起来，又将剩余的那十七支全用了，测试结果仍旧是现实她没怀孕。
<br/>
<br/>        连华生忙完事情回私宅后，记起这件事来，就像是他要当准爸爸了一样，笑不拢嘴地追着童筱颖问道：“喂喂喂！你怀上了没有？”
<br/>
<br/>        童筱颖不想说话，直接将那一袋子三十支验孕棒全给了连华生。
<br/>
<br/>        连华生一本正经地将每一支的结果看完后，面容惊悚：“你、居然、没怀上？阿沉不至于这么弱吧！”
<br/>
<br/>        “你怎么就不怀疑是我有问题？”童筱颖没好气地说，她的例假到现在还没来了。
<br/>
<br/>        连华生见童筱颖一脸懈气地坐在沙发上，于是凑过去，在她的身旁坐下，反问道：“你真的很想给阿沉生孩子啊？”
<br/>
<br/>        “你这不是废话吗？”童筱颖心情不好地白了连华生一眼。
<br/>
<br/>        虽然，那个时候她想怀孩子是为了逼大哥去救季雅珊。
<br/>
<br/>        但是，大哥将她想救季雅珊的希望全部掐灭后，她选择了认命。
<br/>
<br/>        毕竟，生米煮成熟饭，她以为肚子里即将会有个宝宝扎根，与其纠结于雅珊的那个不可能的结局，倒不如好好照顾自己肚子里的孩子。
<br/>
<br/>        所以，这段日子，她才会一门心思地全扎在了自己的肚子上。
<br/>
<br/>        结果，空欢喜一场。
<br/>
<br/>        “要不，我给你安排一下。你去医院做个检查。若是你没问题，那就真的是阿沉有问题了。这种事情，发现得越早越好，及时治疗，就不会耽误你们俩的最佳怀孕时期。”作为医生的连华生，给出了忠恳的建议。
<br/>
<br/>        童筱颖也不反对，强颜欢笑地应道：“好！那辛苦连医生，帮我做安排。”
<br/>
<br/>        连华生点了点头。
<br/>
<br/>        只是现在，童筱颖心里有点点的不安。
<br/>
<br/>        连华生说帮她做安排，还真就做了安排，第二天，他便将她带出了私宅去了医院。
<br/>
<br/>        离开私宅后，童筱颖全程听从连华生的安排。
<br/>
<br/>        在医院里做各项检查，折腾了快一整天，终于到了出结果的时候了。
<br/>
<br/>        童筱颖却将连华生赶出了医生办公室。
<br/>
<br/>        连华生觉得自己站在妇科医生的诊室里确实不太妥，那么多女人用异样的目光盯着他，他也很尴尬，索性依了童筱颖，自己主动离开，去门外候着。
