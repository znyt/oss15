　　正所谓人有失手，马有失蹄，这种事情和明不明白没有绝对关系存在。
<br/>　　所以在王子木看来称心显然并不专业，充其量也就是个急于求成，幸进的小官而已。
<br/>　　李承乾之所以会宠信他估计也是看在他的长相上，而不是看在他的能力上。当然，也有可能是这家伙活儿好，把皇位上那个年轻的皇帝伺候的舒服了也说不定。
<br/>　　……
<br/>　　而此时的咸阳城其他地方也都乱成了一团，独孤玉凤留下的消息实在是太过劲爆，让那些所有关心这件事情的人全都坐不住了。
<br/>　　李毅恒自然也是其中之一，在得到消息之后他的第一个反应是兴奋，第二个反应便是觉得这件事情里透着诡异。
<br/>　　独孤玉凤完全没有理由拿了东西之后不走，反而再次现身，难道她就那么有信心一定能从大唐情报科那些人的手中逃走？
<br/>　　黄浦的脸上一直带着面罩，似乎那面罩已经长到了他的脸上一样，从来没有拿下来过。
<br/>　　他见到李毅恒一脸纠结的样子，便在一边说道：“皇子殿下，难道是觉得这件事情里面有别的门道？”
<br/>　　“难道没有么？独孤玉凤突然现身，你就不觉得奇怪？”李毅恒扭头看了他一眼，眉头微微皱着，显然是对黄浦那刺耳的声音有些受不了。
<br/>　　但那黄浦似乎毫无所觉，依旧用他那刺耳的声音继续说道：“我倒觉得没什么，独孤玉凤这个女人之所以这样做不过就是想要找一个合作伙伴而已，而她之所以会现身，那不过就是向众人展示一下她的手段，表示她有能力躲开众人的耳目罢了。”
<br/>　　“事情怎么可能这么容易？如果真是这样那她为什么不在那里再多待一会儿？还有，万一这件事情里面有诈怎么办？”李毅恒露出担心的表情，也顾得黄浦的声音是如何刺耳了。
<br/>　　“总要冒一些风险的，现在正是李承乾刚刚做了皇帝不久，下面人心还没有全完被他收服，如果这个时候不趁机动起来，再等下去只怕我们的生存空间只会越来越小。”
<br/>　　“可是这么多年我们都等了，难道还差再等一段时间？”李毅恒急声说道。
<br/>　　“皇子殿下，不要告诉我你怕了。”黄浦阴恻恻的说道。
<br/>　　原本他的声音就有些难听刺耳，再配合上他的语气，简直让人听的毛骨悚然。
<br/>　　不过李毅恒这一次却没有停下，而是执着的说道：“这不是怕不怕的问题，而是谨慎，为了些不知道是什么东西的情报，为了一个不知所谓的秘密，到底值不值得我们冒着暴露自己一切的风险，你想过没有？”
<br/>　　“我想的很清楚！”黄浦一声大吼，把脸凑到了李毅恒的面前，双眼死死盯着他的眼睛：“你之所以会一再的找借口，不过就是怕了而已，怕李承乾会像他那个老子一样把你杀死，对不对？对不对？！”
<br/>　　“我，我没有。”李毅恒面对有些疯狂的黄浦，连退了四五步，这才停了下来摇头喃喃说道：“我没有，我也想为父亲报仇，每天都想，每时都想……。”
<br/>　　“不，你是怕了，你不敢，你这个懦夫，当初老子们就不应该找你出来。”黄浦歇斯底里的低吼道：“你根本就不配做太子的儿子，你不配！”
<br/>　　李毅恒被黄浦吼的一句话也说不出来，牙齿咬的‘咯咯’作响，一双拳头紧紧的握着，毫无疑问，如果他现在手里有把刀，他一定会杀了眼前这个对自己不敬的家伙。
<br/>　　黄浦吼过之后，冷冷的看着李毅恒：“你觉得我是看重那份情报？觉得我是看重那些秘密？你认为我不知道这是一个局？告诉你，老子都知道。”
<br/>　　“可那又如何？独孤玉凤掌握大唐情报科这么多年，她的手上会有多少机密的东西你想过没有？如果能把她抓住，我们还需要其他的资料么？还需要么？”
<br/>　　“大唐情报科的手里到底有什么？大唐官员到底有多少丑事掌握在情报科的手里，这些事情你都知道么？只要抓住了独孤玉凤，问出了这些东西，你觉得皇宫里的那些人会有多少倒向我们！你这个贪生怕死的家伙到底想过没有？想过没有！”
<br/>　　不得不说，黄浦这家伙的智力的确非同小可，竟然早已经看破了李承乾布下的局。
<br/>　　可是就算这样，他还是想要去试一下，就像他刚刚说的，单凭独孤玉凤的这个身份，其实就完全值得他们出手，至于说那些资料，老实说，不管是真是假，其实意义都不是很大。
<br/>　　李毅恒虽然被骂的狗血淋头，但也在瞬间醒悟了过来，暗骂了一声当局者迷，自己竟然被事情的表象所迷惑了，没有看清这其中的关键是什么。
<br/>　　如此一来，刚刚被骂了一顿似乎也就变的可以接受，李毅恒原本有些狰狞的脸色也恢复了正常，向着黄浦深施一礼：“黄叔，刚刚是小侄错了，希望黄叔不要放在心上。”
<br/>　　“罢了！”黄浦摆了摆手：“我们这些人陪了你这么多年，就是希望看到你长大成人，看到你继承你父亲的一切，但如果你自己都不知道努力，那还有什么好说的。”
<br/>　　李毅恒抿着嘴没有说什么，他被骂也不是一次两人次了，小的时候甚至还被打过，不过他知道，这是因为父亲的臣子关心则乱，恨自己不争气而已。
<br/>　　可就算是这样，他还是无法原谅这些人对自己的不敬。
<br/>　　随着时间越来越长，李毅恒越来越容不得别人对自己不敬，那些以前打过自己，骂过自己的人，必须要死，而且要让他们死的很惨，就算是这些人当初是为了自己好也不行。
<br/>　　另外还有一点就是，那些人是唯一知道自己身份的人，单凭这一点，他们就该死，李毅恒可不想将来事成之后，这些人反咬一口说自己只是一个替身，若是那样的话，这么多年的努力岂不是都白白浪费了。
<br/>　　所以，知道他身世的人必须要死，对不起他的人也要死，反正将来事成之后，他总不会缺手下，这些人死了才是他们最好的归宿。
